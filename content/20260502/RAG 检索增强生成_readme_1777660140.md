# SoloAI RAG 检索增强生成

[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/rag/main.yml?branch=main&label=build)](https://github.com/SoloAI/rag/actions)
[![License](https://img.shields.io/github/license/SoloAI/rag)](./LICENSE)

面向企业级知识问答、文档搜索与 AI 应用集成的开源 **RAG（Retrieval-Augmented Generation，检索增强生成）** 项目。  
SoloAI RAG 通过将向量检索、语义召回、上下文压缩与大语言模型生成能力结合，帮助开发者快速构建高准确率、可追溯、可扩展的智能问答系统。

## Overview

**SoloAI RAG 检索增强生成** 是一个专注于知识库问答、企业文档检索、私有数据增强生成的开源框架。  
它支持从多源文档导入、文本切分、向量化、索引构建到检索问答的完整流程，适用于 AI 搜索、客服机器人、内部知识助手、研究助手与行业垂直大模型应用。

## Features

- **多源知识接入**：支持 Markdown、PDF、TXT、HTML、Word 等常见文档格式导入
- **向量检索与语义召回**：支持嵌入模型、向量数据库与混合检索策略
- **上下文增强生成**：基于高相关片段构建提示词，提升 LLM 回答准确性与可解释性
- **可插拔架构设计**：易于替换 Embedding 模型、LLM、向量存储与重排组件
- **面向生产环境**：支持配置化部署、批量索引、日志监控与 API 集成
- **适用于企业场景**：覆盖私有知识库问答、智能客服、文档助手、行业检索系统等业务需求

## Installation

### Requirements

- Python 3.10+
- pip 或 poetry
- 可选：Docker
- 可选：OpenAI / 通义千问 / 本地 LLM API Key
- 可选：FAISS、Chroma、Milvus、Weaviate 等向量数据库

### Install with pip

```bash
git clone https://github.com/SoloAI/rag.git
cd rag
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -U pip
pip install -r requirements.txt

```

## Quick Start

以下示例展示如何导入本地文档、构建索引并完成一次 RAG 问答。

```python
from soloai_rag import RAGPipeline

rag = RAGPipeline(
    data_dir="./data",
    index_dir="./storage/index",
    embedding_model="text-embedding-3-large",
    llm_model="gpt-4o-mini",
    vector_store="faiss",
)

# 1. 构建或更新知识库索引
rag.index()

# 2. 发起检索增强问答
response = rag.query("请总结产品部署流程，并列出关键前置条件。")

print(response.answer)
print(response.sources)
```

## Usage Examples

### 1. Load documents and build embeddings

```python
from soloai_rag import DocumentLoader, IndexBuilder

loader = DocumentLoader("./data")
documents = loader.load()

builder = IndexBuilder(
    embedding_model="text-embedding-3-large",
    vector_store="faiss",
    index_dir="./storage/index"
)

builder.build(documents)
```

### 2. Retrieve top-k relevant chunks

```python
from soloai_rag import Retriever

retriever = Retriever(index_dir="./storage/index", top_k=5)
results = retriever.search("RAG 系统如何降低大模型幻觉？")

for item in results:
    print(item.score, item.text[:120], item.metadata)
```

### 3. Generate answer with retrieved context

```python
from soloai_rag import Generator

generator = Generator(llm_model="gpt-4o-mini")

answer = generator.generate(
    query="RAG 在企业知识库中的优势是什么？",
    contexts=[
        "RAG 可以将私有数据注入生成过程，降低模型脱离事实的概率。",
        "通过引用检索片段，系统可以提升答案可追溯性与可信度。"
    ]
)

print(answer)
```

### 4. Run API service

```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

  }'
