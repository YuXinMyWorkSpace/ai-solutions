# SoloAI 大模型微调数据服务

[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/llm-finetuning-data-service/ci.yml?branch=main)](https://github.com/SoloAI/llm-finetuning-data-service/actions)

专业级 **大模型微调数据服务**，面向企业与开发者提供高质量的数据清洗、标注、结构化、质检与交付能力，帮助团队高效构建适用于 SFT、Instruction Tuning、RAG、Alignment 等场景的数据资产。  
由 **SoloAI** 开源，聚焦可扩展、可审计、可集成的数据处理工作流，适用于多行业大模型训练与优化项目。

## Features

- **多源数据接入**：支持文本、问答、文档、对话、知识库等多种数据来源的统一处理
- **微调数据清洗与标准化**：自动完成去重、脱敏、格式统一、脏数据过滤与字段映射
- **高质量标注与质检流程**：支持人工标注、规则校验、抽样审核与多轮质量控制
- **适配主流训练格式**：兼容 JSONL、Alpaca、ShareGPT、ChatML 等常见大模型微调数据格式
- **可扩展工作流**：支持自定义处理管道、校验规则与导出模板，便于集成内部系统
- **面向企业交付**：支持行业语料构建、指令数据集设计、私有化流程与批量交付

## Quick Start

以下示例展示如何将原始问答数据清洗并转换为适用于大模型监督微调的 JSONL 格式。

### Example input

```json
[
  {
    "question": "什么是向量数据库？",
    "answer": "向量数据库用于存储和检索高维向量，常用于语义搜索与 RAG。"
  },
  {
    "question": "请解释监督微调",
    "answer": "监督微调是使用标注数据对预训练模型进行任务适配的过程。"
  }
]
```

### Run the pipeline

```bash
python scripts/prepare_dataset.py \
  --input data/raw/qa.json \
  --output data/processed/train.jsonl \
  --format alpaca
```

### Example output

```jsonl
{"instruction":"什么是向量数据库？","input":"","output":"向量数据库用于存储和检索高维向量，常用于语义搜索与 RAG。"}
{"instruction":"请解释监督微调","input":"","output":"监督微调是使用标注数据对预训练模型进行任务适配的过程。"}
```

## Usage

### 1. Validate dataset quality

使用内置校验器检查缺失字段、重复样本、长度异常与格式错误。

```bash
python scripts/validate_dataset.py \
  --input data/processed/train.jsonl \
  --schema sharegpt
```

### 2. Export to different fine-tuning formats

将统一中间格式导出到不同训练框架所需的格式。

```bash
python scripts/export_dataset.py \
  --input data/processed/train.jsonl \
  --output data/exports/train_sharegpt.jsonl \
  --format sharegpt
```

### 3. Python API example

```python
from src.pipelines import DatasetPipeline

pipeline = DatasetPipeline(
    deduplicate=True,
    normalize_text=True,
    remove_pii=True,
    target_format="chatml"
)

pipeline.run(
    input_path="data/raw/customer_support.json",
    output_path="data/processed/customer_support_chatml.jsonl"
)
```

### 4. Custom rule-based cleaning

```python
from src.cleaners import TextCleaner

cleaner = TextCleaner(
    strip_html=True,
    normalize_whitespace=True,
    remove_emails=False,
    remove_phone_numbers=True
)

sample = {
    "instruction": "请总结以下内容",
    "input": "<p>这是   一段     带有HTML的文本。</p>",
    "output": "这是摘要。"
}

cleaned = cleaner.clean(sample)
print(cleaned)
```
