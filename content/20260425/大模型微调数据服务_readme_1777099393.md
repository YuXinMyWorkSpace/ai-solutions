## 项目简介

**SoloAI 大模型微调数据服务**旨在为 AI 模型训练提供标准化、工程化的数据解决方案，覆盖数据采集、清洗、标注、质检、格式转换与交付全流程。项目适用于通用大模型、垂直行业模型、Agent 系统及多轮对话场景的数据准备需求。

---

## 核心特性

- **全流程数据服务**：支持数据采集、去重、清洗、脱敏、分类、标注、质检与交付。
- **多种训练范式支持**：覆盖 SFT、Instruction Tuning、Preference Data、DPO/RLHF、对话数据与问答对构建。
- **标准化数据格式**：兼容 JSONL、CSV、Parquet 及主流大模型训练格式。
- **高质量质检机制**：支持一致性校验、规则审查、人工抽检与自动评估。
- **行业定制化能力**：可面向金融、医疗、教育、电商、制造、政务等场景提供领域数据方案。
- **可扩展交付体系**：支持小规模 PoC 到大规模批量交付，满足企业级数据生产需求。

以下示例展示了一个典型的微调数据样本结构，适用于指令微调（SFT）场景：

```json
{
  "id": "sample-0001",
  "system": "你是一名专业的行业助手。",
  "instruction": "请总结以下客服对话中的用户诉求。",
  "input": "用户：我想退款，但是订单已经发货。客服：您好，可以为您申请售后。",
  "output": "用户希望对已发货订单申请退款或售后处理。",
  "domain": "ecommerce",
  "language": "zh",
  "quality_score": 0.98
}
```

将数据保存为 `train.jsonl`：


## 使用示例

### 1. 指令微调数据格式

适用于通用问答、行业助手、知识抽取等场景：

```json
{
  "instruction": "根据合同条款提取付款节点",
  "input": "合同约定：签约后支付30%，验收后支付70%。",
  "output": "付款节点：签约后支付30%；验收后支付70%。"
}
```

### 2. 多轮对话数据格式

适用于聊天机器人、Agent、客服助手：

```json
{
  "messages": [
    { "role": "system", "content": "你是企业知识库问答助手。" },
    { "role": "user", "content": "如何申请发票？" },
    { "role": "assistant", "content": "请提供订单号，我将为你查询开票流程。" }
  ]
}
```

### 3. 偏好数据 / DPO 数据格式

适用于对齐训练、偏好优化、响应质量提升：

```json
{
  "prompt": "请用专业但简洁的方式解释什么是增值税专用发票。",
  "chosen": "增值税专用发票是用于记录销售方销项税额和购买方进项税额的重要凭证，通常用于企业间交易及税务抵扣。",
  "rejected": "就是一种发票，企业会用到。"
}
```

### 4. Python 读取与校验示例

```python
import json

def validate_record(record):
    required_fields = ["instruction", "output"]
    return all(field in record and record[field] for field in required_fields)

with open("train.jsonl", "r", encoding="utf-8") as f:
    for line_no, line in enumerate(f, 1):
        data = json.loads(line)
        if not validate_record(data):
            print(f"Invalid record at line {line_no}")
```

### 5. 数据转换示例

将原始 CSV 转换为 JSONL：

```python
import csv
import json

input_file = "raw.csv"
output_file = "train.jsonl"

with open(input_file, "r", encoding="utf-8") as csvfile, \
     open(output_file, "w", encoding="utf-8") as jsonlfile:
    reader = csv.DictReader(csvfile)
    for row in reader:
        record = {
            "instruction": row.get("instruction", ""),
            "input": row.get("input", ""),
            "output": row.get("output", "")
        }
        jsonlfile.write(json.dumps(record, ensure_ascii=False) + "\n")
```
