## 快速开始

以下示例展示了一个典型的微调数据样本生成流程：将原始问答数据转换为适用于指令微调的数据格式。

### 输入示例

```json
{
  "question": "什么是向量数据库？",
  "answer": "向量数据库是一类用于存储、索引和检索高维向量数据的数据库系统。"
}
```

### 转换为 SFT JSONL

```python
import json

raw_data = [
    {
        "question": "什么是向量数据库？",
        "answer": "向量数据库是一类用于存储、索引和检索高维向量数据的数据库系统。"
    },
    {
        "question": "RAG 的核心作用是什么？",
        "answer": "RAG 通过检索外部知识来增强大模型回答的准确性和时效性。"
    }
]

with open("sft_dataset.jsonl", "w", encoding="utf-8") as f:
    for item in raw_data:
        record = {
            "instruction": item["question"],
            "input": "",
            "output": item["answer"]
        }
        f.write(json.dumps(record, ensure_ascii=False) + "\n")

print("SFT 数据集已生成：sft_dataset.jsonl")
```

### 输出示例

```json
{"instruction":"什么是向量数据库？","input":"","output":"向量数据库是一类用于存储、索引和检索高维向量数据的数据库系统。"}
{"instruction":"RAG 的核心作用是什么？","input":"","output":"RAG 通过检索外部知识来增强大模型回答的准确性和时效性。"}
```

## 4. 数据清洗脚本示例

下面示例演示如何对原始文本做简单清洗，包括去空值、去多余空格和长度过滤。

```python
def clean_text(text: str) -> str:
    if not text:
        return ""
    text = text.strip()
    text = " ".join(text.split())
    return text

samples = [
    "  大模型 微调 数据服务   ",
    "",
    "RAG   数据构建   流程"
]

cleaned = [clean_text(x) for x in samples if clean_text(x)]

print(cleaned)
# ['大模型 微调 数据服务', 'RAG 数据构建 流程']
```
