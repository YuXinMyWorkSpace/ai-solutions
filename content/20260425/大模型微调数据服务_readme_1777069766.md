## 项目简介

**SoloAI 大模型微调数据服务** 是一个聚焦于 **LLM 微调数据生产与交付** 的开源项目说明模板与服务能力展示页，适用于需要构建高质量指令数据、问答数据、对话数据、分类数据、偏好优化数据（如 SFT / DPO / RLHF 前置数据） 的团队。

该项目旨在帮助企业快速理解并接入专业的数据服务流程，包括数据规范设计、领域语料构建、人工与模型协同标注、质量评估以及可直接用于训练的标准化数据交付。

- **多类型微调数据支持**  
  支持 SFT、Instruction Tuning、Multi-turn Dialogue、Classification、Extraction、Summarization、Preference Data 等多种数据形态。

## 使用示例

以下示例演示如何组织、校验与导出适用于大模型微调的数据。

### 1. Python：读取并校验 JSONL 微调数据

```python
import json

required_fields = {"instruction", "input", "output"}

def validate_jsonl(file_path: str):
    valid_count = 0
    invalid_count = 0

    with open(file_path, "r", encoding="utf-8") as f:
        for line_no, line in enumerate(f, start=1):
            try:
                item = json.loads(line)
                if required_fields.issubset(item.keys()):
                    valid_count += 1
                else:
                    invalid_count += 1
                    print(f"[Line {line_no}] Missing fields: {required_fields - set(item.keys())}")
            except json.JSONDecodeError:
                invalid_count += 1
                print(f"[Line {line_no}] Invalid JSON")

    print(f"Valid: {valid_count}, Invalid: {invalid_count}")

validate_jsonl("train.jsonl")
```

### 2. Python：将对话数据转换为训练格式

```python
import json

source_data = [
    {
        "messages": [
            {"role": "user", "content": "请解释什么是向量数据库？"},
            {"role": "assistant", "content": "向量数据库是用于存储和检索高维向量数据的数据库系统，常用于语义搜索与 RAG 场景。"}
        ]
    }
]

with open("sharegpt_format.jsonl", "w", encoding="utf-8") as f:
    for item in source_data:
        f.write(json.dumps(item, ensure_ascii=False) + "\n")
```
