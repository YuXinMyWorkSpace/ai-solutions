# SoloAI RAG 检索增强生成

[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/rag-retrieval-generation/ci.yml?branch=main)](https://github.com/SoloAI/rag-retrieval-generation/actions)

面向生产环境的 **RAG（Retrieval-Augmented Generation，检索增强生成）** 开源框架，帮助开发者快速构建基于私有知识库的智能问答、企业搜索、文档助手与 AI Copilot 系统。  
由 **SoloAI** 打造，项目专注于高质量检索、可扩展知识接入、多模型兼容与工程化部署，适用于企业级 AI 应用开发。

---

## Overview

**SoloAI RAG 检索增强生成** 是一个用于构建检索增强生成系统的开源项目，支持从文档解析、向量化、索引构建到检索、重排、上下文注入和大模型生成的完整工作流。  
该项目适合用于：

- 企业知识库问答
- 文档检索与摘要
- AI 客服与内部助手
- 合规文档查询
- 多源数据增强生成应用

---

## Features

- **多源知识接入**：支持 Markdown、PDF、TXT、HTML、Office 文档等常见知识源接入
- **高质量语义检索**：支持向量检索、关键词检索与混合检索策略
- **Rerank 重排能力**：集成检索结果重排机制，提升上下文相关性与回答准确率
- **多模型兼容**：可对接主流 Embedding 模型与 LLM 服务，便于灵活替换
- **模块化架构设计**：易于扩展文档切分、索引、召回、生成与评估链路
- **面向生产部署**：支持本地开发、Docker 部署与 API 服务化集成

## Installation

### Requirements

- Python 3.10+
- pip / poetry
- 可选：Docker
- 可选：OpenAI-compatible API 或本地大模型服务
- 可选：向量数据库（如 FAISS、Milvus、Qdrant、Chroma）

### Install from source

```bash
git clone https://github.com/SoloAI/rag-retrieval-generation.git
cd rag-retrieval-generation
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -U pip
pip install -r requirements.txt
```

### Environment variables

创建 `.env` 文件：

```bash
OPENAI_API_KEY=your_api_key
OPENAI_BASE_URL=https://api.openai.com/v1
EMBEDDING_MODEL=text-embedding-3-large
CHAT_MODEL=gpt-4o-mini

VECTOR_DB=faiss
DATA_DIR=./data
INDEX_DIR=./storage/index
```

## Quick Start

下面示例展示如何将本地文档构建为知识索引，并执行一次 RAG 问答。

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline(
    data_dir="./data",
    index_dir="./storage/index",
    embedding_model="text-embedding-3-large",
    chat_model="gpt-4o-mini",
)

# 1. 构建索引
pipeline.ingest()

# 2. 执行问答
response = pipeline.query("请总结知识库中关于 RAG 检索优化 的最佳实践。")

print(response.answer)
print(response.sources)
```

运行：

```bash
python examples/quick_start.py
```

## Usage Examples

### 1. Build index from documents

```python
from soloai_rag import DocumentIngestor

ingestor = DocumentIngestor(
    data_dir="./data",
    index_dir="./storage/index",
    chunk_size=800,
    chunk_overlap=120
)

ingestor.load()
ingestor.split()
ingestor.embed()
ingestor.index()
```

### 2. Retrieve top-k context

```python
from soloai_rag import Retriever

retriever = Retriever(index_dir="./storage/index", top_k=5)
results = retriever.search("什么是混合检索？")

for item in results:
    print(item.score, item.metadata["source"])
    print(item.text[:200])
```

### 3. Ask grounded questions with citations

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline(index_dir="./storage/index")
response = pipeline.query(
    question="如何提高 RAG 系统的召回率与准确率？",
    return_citations=True
)

print("Answer:", response.answer)
print("Citations:", response.sources)
```

### 4. Run API service

```bash
uvicorn api.main:app --host 0.0.0.0 --port 8000
```

示例请求：

```bash
curl -X POST "http://localhost:8000/query" \
  -H "Content-Type: application/json" \
  -d '{
    "question": "请解释 RAG 中的 rerank 的作用",
    "top_k": 5
  }'
```
