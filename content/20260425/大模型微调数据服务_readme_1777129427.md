## 项目简介

**SoloAI 大模型微调数据服务**是一个面向大模型训练与微调场景的数据工程项目，聚焦于**指令数据构建、清洗去重、质量评估、格式转换、数据增强与交付标准化**。  
本项目适用于企业知识库问答、行业垂直模型、SFT/RLHF 前置数据准备、多轮对话、分类抽取、Agent 工具调用等场景。

### 适用场景

- 大语言模型（LLM）监督微调数据集构建
- 行业领域问答数据生产与清洗
- 企业私有知识库转训练数据
- 多轮对话与指令跟随样本生成
- 文本分类、信息抽取、摘要、改写等任务数据制作
- AI 应用上线前的数据标准化与质量控制

---

## 核心功能

- **多源数据接入**：支持 CSV、JSON、JSONL、Excel、数据库导出及自定义原始文本输入。
- **高质量数据清洗**：提供去重、脱敏、异常过滤、格式统一、长度控制与噪声样本剔除能力。
- **微调格式标准化**：支持 Alpaca、ShareGPT、OpenAI Messages、SFT 指令对等常见训练格式转换。
- **数据质量评估**：支持规则校验、字段完整性检查、标签一致性验证与抽样质检流程。
- **可扩展处理流水线**：模块化设计，便于按行业、任务类型、交付标准进行定制。
- **面向企业交付**：适配金融、医疗、电商、教育、制造、政务等行业大模型数据服务需求。

## 快速开始

以下示例展示如何将原始问答数据转换为适合大模型监督微调（SFT）的 JSONL 数据格式。

### 输入示例 `data/raw_sample.json`

```json
[
  {
    "instruction": "解释什么是向量数据库",
    "input": "",
    "output": "向量数据库是一类专门用于存储、索引和检索向量嵌入的数据系统，常用于语义搜索、RAG 和推荐系统。"
  },
  {
    "instruction": "将以下文本总结为一句话",
    "input": "大模型微调通常需要高质量、结构化且符合目标任务分布的数据。",
    "output": "大模型微调依赖高质量、结构化的数据支持。"
  }
]
```

### 执行处理命令

```bash
python cli.py \
  --input data/raw_sample.json \
  --output data/sft_dataset.jsonl \
  --format alpaca \
  --deduplicate \
  --validate
```

### 输出示例 `data/sft_dataset.jsonl`

```jsonl
{"instruction":"解释什么是向量数据库","input":"","output":"向量数据库是一类专门用于存储、索引和检索向量嵌入的数据系统，常用于语义搜索、RAG 和推荐系统。"}
{"instruction":"将以下文本总结为一句话","input":"大模型微调通常需要高质量、结构化且符合目标任务分布的数据。","output":"大模型微调依赖高质量、结构化的数据支持。"}
```

## 使用示例

### 1. 转换为 ShareGPT 对话格式

```bash
python cli.py \
  --input data/customer_service.json \
  --output data/sharegpt.jsonl \
  --format sharegpt
```

示例输出：

```jsonl
{"conversations":[{"from":"human","value":"什么是大模型微调？"},{"from":"gpt","value":"大模型微调是基于特定任务数据对预训练模型进行进一步训练，以提升其在目标场景中的表现。"}]}
```

---

### 2. 数据清洗与去重

```bash
python cli.py \
  --input data/raw.jsonl \
  --output data/cleaned.jsonl \
  --deduplicate \
  --remove-empty \
  --normalize-text \
  --min-length 5 \
  --max-length 2000
```

适用于：

- 重复样本清理
- 空字段过滤
- 文本标准化
- 长短文本约束控制

---

### 3. Python API 调用示例

```python
from soloai.pipeline import DataPipeline
from soloai.validators import SchemaValidator
from soloai.exporters import JsonlExporter

pipeline = DataPipeline(
    input_path="data/raw.json",
    output_path="data/output.jsonl",
    target_format="alpaca"
)

pipeline.load()
pipeline.clean(deduplicate=True, remove_empty=True, normalize_text=True)
pipeline.validate(SchemaValidator())
pipeline.export(JsonlExporter())

print("Dataset generated successfully.")
```

---

### 4. 企业定制数据处理流程示例

```python
from soloai.pipeline import DataPipeline

pipeline = DataPipeline(
    input_path="data/industry_corpus.xlsx",
    output_path="deliverables/medical_sft.jsonl",
    target_format="openai_messages"
)

pipeline.load()
pipeline.clean(
    deduplicate=True,
    remove_empty=True,
    mask_sensitive=True,
    normalize_text=True
)
pipeline.transform(task_type="qa")
pipeline.sample_review(sample_size=200)
pipeline.export()

print("Enterprise delivery package is ready.")
```
