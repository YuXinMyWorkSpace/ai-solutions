## Installation

### Requirements

- Python 3.10+
- `pip` or `poetry`
- Docker (optional, recommended for production)
- NVIDIA GPU + CUDA (optional, for accelerated inference)

### Option 1: Install from source

```bash
git clone https://github.com/SoloAI/llm-deployment-optimization.git
cd llm-deployment-optimization
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -U pip
pip install -r requirements.txt
```

### Option 2: Install with Poetry

```bash
git clone https://github.com/SoloAI/llm-deployment-optimization.git
cd llm-deployment-optimization
poetry install
poetry shell
```

### Option 3: Run with Docker

```bash
docker build -t soloai/llm-deployment-optimization:latest .
docker run --rm -p 8000:8000 soloai/llm-deployment-optimization:latest
```

## Quick Start

Start a local optimized inference server:

```bash
python -m soloai_llm_opt.serve \
  --model meta-llama/Llama-3-8B-Instruct \
  --host 0.0.0.0 \
  --port 8000 \
  --dtype float16 \
  --max-batch-size 16 \
  --enable-quantization
```

Test the API:

```bash
curl -X POST "http://localhost:8000/generate" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Explain how LLM deployment optimization reduces production cost.",
    "max_tokens": 128,
    "temperature": 0.2
  }'
```

Example response:

```json
{
  "text": "LLM deployment optimization reduces production cost by improving hardware utilization, lowering memory overhead, increasing request throughput, and reducing latency per token..."
}
```

## Usage Examples

### 1. Launch with a YAML configuration

```yaml
# config/deploy.yaml
model: meta-llama/Llama-3-8B-Instruct
server:
  host: 0.0.0.0
  port: 8000
runtime:
  dtype: float16
  quantization: true
  max_batch_size: 32
  tensor_parallel_size: 1
observability:
  metrics: true
  logging: info
```

Run:

```bash
python -m soloai_llm_opt.serve --config config/deploy.yaml
```

### 2. Python API usage

```python
from soloai_llm_opt import OptimizedLLMServer

server = OptimizedLLMServer(
    model="meta-llama/Llama-3-8B-Instruct",
    dtype="float16",
    quantization=True,
    max_batch_size=16,
)

server.start(host="0.0.0.0", port=8000)
```

### 3. Benchmark deployment performance

```bash
python -m soloai_llm_opt.benchmark \
  --endpoint http://localhost:8000/generate \
  --concurrency 32 \
  --requests 500 \
  --input-tokens 256 \
  --output-tokens 128
```

Example output:

```text
Average latency: 420 ms
P95 latency: 690 ms
Throughput: 78.4 req/s
GPU memory usage: 14.2 GB
```

### 4. Docker Compose for production-style deployment

```yaml
version: "3.9"

services:
  llm-server:
    build: .
    ports:
      - "8000:8000"
    environment:
      MODEL_NAME: meta-llama/Llama-3-8B-Instruct
      DTYPE: float16
      ENABLE_QUANTIZATION: "true"
    restart: unless-stopped
```

Start:

```bash
docker compose up -d
```
