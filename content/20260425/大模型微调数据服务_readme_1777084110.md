# SoloAI · 大模型微调数据服务

[![Build Status](https://img.shields.io/badge/build-passing-brightgreen.svg)](#)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](#license)

专业的大模型微调数据服务项目，面向企业与开发者提供高质量、可扩展、可审计的数据处理能力，覆盖从原始数据清洗、标注、指令构造到训练集交付的完整流程。  
本项目聚焦 **LLM Fine-tuning Data Service** 场景，帮助团队高效构建适用于监督微调（SFT）、偏好优化、评测与行业定制的大模型训练数据资产。

---

## 概述

**SoloAI 大模型微调数据服务** 致力于为各行业提供标准化、工程化的数据服务能力，支持对话数据、知识问答、分类抽取、RAG增强数据、偏好数据与多轮指令数据的构建与交付。项目强调数据质量、可追溯性、隐私安全与交付效率，适用于金融、医疗、教育、电商、政企、客服等多类 AI 应用场景。

---

## Features

- **多类型训练数据构建**  
  支持 SFT 指令数据、QA 数据、分类/抽取数据、总结改写数据、偏好对比数据等多种数据形态。

- **数据清洗与标准化处理**  
  提供去重、脱敏、格式归一、噪声过滤、异常检测等能力，提升微调数据质量。

- **高质量标注与质检流程**  
  通过规则校验、抽样复核、多轮质检与一致性检查，保障数据准确性与可用性。

- **面向行业场景定制交付**  
  支持客服、法律、医疗、教育、金融、电商等领域数据方案设计与定制化生产。

- **支持主流模型训练格式**  
  可输出适配 OpenAI-style JSONL、ShareGPT、Alpaca、对话式多轮结构等常见微调格式。

- **可扩展的数据服务流程**  
  支持批量处理、任务配置化、流程自动化，便于团队集成到现有 AI/ML 工程体系中。

## Installation

> 如果你希望将本项目作为本地数据处理工具链或服务模板进行二次开发，可参考以下安装方式。

### Requirements

- Python 3.10+
- pip 23+
- Git

### Clone Repository

```bash
git clone https://github.com/SoloAI/llm-finetune-data-service.git
cd llm-finetune-data-service
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Optional: Create Virtual Environment

```bash
python -m venv .venv
source .venv/bin/activate   # Linux / macOS
# .venv\Scripts\activate    # Windows
pip install -r requirements.txt
```

## Quick Start

下面示例展示如何将原始问答数据转换为适合大模型监督微调的数据格式。

### Example Input

```json
[
  {
    "instruction": "介绍一下大模型微调数据服务的价值",
    "input": "",
    "output": "大模型微调数据服务能够帮助企业将非结构化业务知识转化为可训练数据，提升模型在垂直场景中的效果。"

```

### Convert to JSONL

```bash
python scripts/prepare_dataset.py \
  --input data/raw/sample.json \
  --output data/processed/sft_dataset.jsonl \
  --format openai
```

### Example Output

```json

```

## Usage Examples

### 1. 数据清洗

对原始文本执行去重、清洗与脱敏：

```bash
python scripts/clean_data.py \
  --input data/raw/source.jsonl \
  --output data/clean/cleaned.jsonl \
  --dedup \
  --normalize \
  --desensitize
```

### 2. 构造监督微调数据

将业务文本转为 instruction-following 格式：

```bash
python scripts/build_sft.py \
  --input data/clean/cleaned.jsonl \
  --output data/processed/sft.jsonl \
  --template alpaca
```

### 3. 构造偏好优化数据

生成适用于偏好学习或对比训练的数据结构：

```bash
python scripts/build_preference.py \
  --input data/clean/cleaned.jsonl \
  --output data/processed/preference.jsonl
```

示例格式：

```json
{
  "prompt": "请总结以下客服对话的核心问题",
  "chosen": "用户主要反馈订单延迟发货，并要求确认补偿方案。",
  "rejected": "这是一次普通对话，没有重点。"
}
```

### 4. Python API Example

```python
from soloai_data.pipeline import DataPipeline

pipeline = DataPipeline(
    dedup=True,
    normalize=True,
    desensitize=True,
    output_format="openai"
)

pipeline.run(
    input_path="data/raw/source.jsonl",
    output_path="data/processed/train.jsonl"
)
```

### 5. 行业定制交付示例

- **智能客服**：FAQ、意图识别、工单总结、多轮对话构造
