# SoloAI RAG 检索增强生成
[![Build Status](https://img.shields.io/github/actions/workflow/status/soloai/rag/main.yml?branch=main)](https://github.com/soloai/rag/actions)
[![License](https://img.shields.io/github/license/soloai/rag)](./LICENSE)

**SoloAI RAG 检索增强生成** is an open-source framework for building high-accuracy, production-ready **Retrieval-Augmented Generation (RAG)** systems. It helps developers combine large language models with document retrieval, knowledge indexing, and context-aware generation for enterprise AI applications.

---

## Overview

RAG（检索增强生成）通过将**向量检索、知识库索引、重排序、提示工程和大语言模型生成**结合起来，显著提升回答的准确性、可解释性与实时性。  
This project is designed for teams who need a scalable, modular, and developer-friendly foundation for **AI knowledge assistants, enterprise search, customer support copilots, and domain-specific Q&A systems**.

## Features

- **Modular RAG pipeline**: ingestion, chunking, embedding, retrieval, reranking, and generation
- **Multiple knowledge source support**: Markdown, PDF, TXT, HTML, and structured documents
- **Vector database integration**: easy connection to FAISS, Chroma, Milvus, or other embedding stores
- **LLM-ready architecture**: works with OpenAI-compatible APIs and local/open-source models
- **Configurable retrieval strategies**: top-k search, hybrid retrieval, metadata filtering, and reranking
- **Production-friendly design**: simple CLI/API usage, environment-based config, and extensible components

## Installation

### Requirements

- Python 3.10+
- `pip` or `poetry`
- Optional: Docker for containerized deployment

### Install with pip

```bash
git clone https://github.com/soloai/rag.git
cd rag
python -m venv .venv
source .venv/bin/activate   # Linux / macOS
# .venv\Scripts\activate    # Windows

pip install -U pip
pip install -r requirements.txt
```

### Install in editable mode

```bash
pip install -e .
```

### Environment Configuration

Create a `.env` file in the project root:

```env
OPENAI_API_KEY=your_api_key
EMBEDDING_MODEL=text-embedding-3-large
CHAT_MODEL=gpt-4o-mini
VECTOR_DB=faiss
DATA_DIR=./data
INDEX_DIR=./storage/index
```

## Usage

## 1. Python API Example

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline(
    collection="knowledge_base",
    embedding_model="text-embedding-3-large",
    chat_model="gpt-4o-mini"
)

response = pipeline.ask("请总结企业知识库 RAG 系统的核心组成。")
print(response.answer)
print(response.sources)
```

## 2. Document Indexing Example

```python
from soloai_rag import DocumentIndexer

indexer = DocumentIndexer(collection="product_docs")
indexer.add_path("./documents")
indexer.build()
```

## 3. Retrieval + Generation Example

```python
from soloai_rag import Retriever, Generator

retriever = Retriever(collection="product_docs", top_k=5)
docs = retriever.search("如何优化向量检索召回率？")

generator = Generator(model="gpt-4o-mini")
answer = generator.generate(
    question="如何优化向量检索召回率？",
    context=docs
)

print(answer)
```

## 4. Example Configuration

```yaml
project:
  name: soloai-rag

retrieval:
  top_k: 5
  rerank: true
  hybrid_search: true

models:
  embedding: text-embedding-3-large
  chat: gpt-4o-mini

storage:
  vector_db: faiss
  index_dir: ./storage/index
```
