## 📖 Overview
A production-grade toolkit for optimizing Large Language Model (LLM) deployment across heterogeneous compute environments. SoloAI/llm-deploy-opt reduces inference latency by up to 3× and cuts GPU memory overhead by 60% through hardware-aware quantization, dynamic continuous batching, and kernel-level memory management.

## ✨ Key Features
- **🔹 Advanced Quantization Pipeline**: Seamless INT4/INT8/FP8 calibration with GPTQ/AWQ support and automatic fallback to FP16 for unsupported layers.
- **🔹 Dynamic Continuous Batching**: Request-level scheduling with PagedAttention to maximize throughput while minimizing tail latency.
- **🔹 Intelligent KV Cache Management**: Sliding window caching, prefix reuse, and automatic eviction policies for long-context workloads.
- **🔹 Hardware-Agnostic Acceleration**: Unified backend supporting NVIDIA CUDA, AMD ROCm, Intel Xeon/ARC, and Apple Silicon MPS.
- **🔹 Plug-and-Play Ecosystem Integration**: Drop-in compatibility with Hugging Face Transformers, vLLM, and TensorRT-LLM model weights.
- **🔹 Observability & Auto-Scaling**: Built-in Prometheus metrics, request tracing, and Kubernetes HPA/VPA integration hooks.

## 📦 Installation

### Prerequisites
- Python 3.9+
- CUDA 11.8+ / ROCm 5.7+ / MPS (macOS 13+)
- `pip` or `conda`

### Quick Install
```bash
pip install solo-llm-opt
```

### From Source (Recommended for Custom Kernels)
```bash
git clone https://github.com/SoloAI/llm-deploy-opt.git
cd llm-deploy-opt
pip install -e ".[all]"
```

> 💡 **Tip**: For NVIDIA GPU acceleration, ensure `torch` and `triton` are installed with matching CUDA versions. See `docs/hardware-setup.md` for detailed driver requirements.

## ⚡ Quick Start

```python
from solo_opt import LLMEngine, QuantizationConfig

# Initialize optimized inference engine
engine = LLMEngine(
    model_id="meta-llama/Llama-3-8B-Instruct",
    quantization=QuantizationConfig(bits=4, method="awq"),
    max_batch_size=64,
    device="auto"  # Automatically selects best available backend
)

# Run synchronous generation
response = engine.generate(
    prompt="Explain the difference between attention and self-attention.",
    max_tokens=256,
    temperature=0.7
)

print(response.text)
```

## 🛠️ Usage Examples

### 1. Multi-GPU Tensor Parallelism & FP8 KV Cache
```python
engine = LLMEngine(
    model_id="Qwen/Qwen2.5-72B",
    tensor_parallel_size=4,
    kv_cache_dtype="fp8_e4m3",
    enable_paged_attention=True,
    gpu_memory_utilization=0.92,
    max_seq_len=8192
)
```

### 2. Launch Production REST API Server
```bash
solo-opt serve \
  --model meta-llama/Llama-3-8B \
  --port 8000 \
  --workers 4 \
  --max-concurrent-requests 128 \
  --enable-metrics
```

**Client Request:**
```bash
curl -X POST http://localhost:8000/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "model": "llama-3-8b",
    "messages": [{"role": "user", "content": "Summarize the latest advances in LLM routing."}],
    "max_tokens": 512
  }'
```

### 3. Custom Scheduler & Benchmark Hook
```python
from solo_opt.benchmark import InferenceProfiler

profiler = InferenceProfiler(engine)
stats = profiler.run(
    dataset="openai_humaneval",
    concurrency_levels=[1, 8, 32],
    measure=["throughput", "p95_latency", "gpu_mem_peak"]
)
print(stats.summary_table())
```
