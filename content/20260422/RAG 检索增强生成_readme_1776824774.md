## 项目简介

**SoloAI RAG** 是一个专注于 **RAG 检索增强生成** 场景的开源项目，支持从非结构化文档摄取、文本切分、向量化索引、语义检索、检索结果重排，到基于上下文的答案生成的完整工作流。  
它适用于企业知识库问答、智能客服、文档助手、研发知识检索、内部搜索增强等 AI 应用场景。

---

## 核心特性

- **端到端 RAG Pipeline**：覆盖文档导入、切分、Embedding、向量检索、Re-ranking、Prompt 构造与生成。
- **多数据源支持**：可扩展接入 Markdown、PDF、HTML、TXT、数据库及对象存储中的知识文档。
- **高性能向量检索**：支持主流向量数据库与相似度搜索策略，适配大规模知识库场景。
- **可插拔模型架构**：灵活接入 Embedding 模型、重排模型与各类 LLM（本地部署或云端 API）。
- **面向生产的可观测性**：便于记录检索链路、召回结果、生成上下文与响应延迟，支持优化与调试。
- **开发者友好 API**：提供简洁的 Python 接口与清晰模块边界，便于二次开发与企业集成。

## 安装

### 环境要求

- Python 3.10+
- pip / poetry
- 可选：Docker
- 可选：向量数据库（如 FAISS、Milvus、Qdrant、Weaviate）

### 使用 pip 安装

```bash
git clone https://github.com/SoloAI/rag.git
cd rag
pip install -U pip
pip install -e .
```

### 安装可选依赖

```bash
# PDF 解析支持
pip install -e ".[pdf]"

# 向量数据库支持
pip install -e ".[vectordb]"

# 全量开发依赖
pip install -e ".[dev]"
```

### 环境变量配置

创建 `.env` 文件：

```env
OPENAI_API_KEY=your_api_key
EMBEDDING_MODEL=text-embedding-3-large
CHAT_MODEL=gpt-4o-mini
VECTOR_STORE=faiss
```

## 快速开始

以下示例展示如何构建一个最小可用的 **RAG 检索增强生成** 流程。

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline(
    embedding_model="text-embedding-3-large",
    chat_model="gpt-4o-mini",
    vector_store="faiss"
)

# 1. 建立索引
pipeline.ingest(
    source="./docs",
    file_types=[".md", ".txt", ".pdf"],
    chunk_size=500,
    chunk_overlap=50
)

# 2. 发起问答
response = pipeline.query(
    question="什么是 RAG 检索增强生成？",
    top_k=5,
    rerank=True
)

print(response.answer)
print(response.sources)
```

运行：

```bash
python examples/quickstart.py
```

## 使用示例

### 1. 从本地文档构建知识库

```python
from soloai_rag import KnowledgeBase

kb = KnowledgeBase(vector_store="faiss")
kb.add_documents("./knowledge", recursive=True)
kb.build_index()
```

### 2. 执行语义检索

```python
results = kb.search(
    query="如何提升检索召回率？",
    top_k=10
)

for item in results:
    print(item.score, item.metadata["source"])
```

### 3. 基于检索上下文生成答案

```python
from soloai_rag import Retriever, Generator

retriever = Retriever(vector_store=kb.vector_store)
generator = Generator(chat_model="gpt-4o-mini")

contexts = retriever.retrieve("RAG 中 rerank 的作用是什么？", top_k=5)

answer = generator.generate(
    query="RAG 中 rerank 的作用是什么？",
    contexts=contexts
)

print(answer)
```

### 4. 自定义 RAG 配置

```python
from soloai_rag import RAGPipeline, RAGConfig

config = RAGConfig(
    chunk_size=800,
    chunk_overlap=100,
    top_k=8,
    rerank_model="bge-reranker-base",
    similarity_threshold=0.72
)

pipeline = RAGPipeline.from_config(config)
```
