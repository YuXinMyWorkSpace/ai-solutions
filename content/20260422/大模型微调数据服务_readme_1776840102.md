## 📖 Overview
FTDataKit is a high-performance, scalable data processing pipeline engineered specifically for **large language model (LLM) fine-tuning data services**. It automates raw data ingestion, quality filtering, format normalization, and provenance tracking, enabling researchers and ML engineers to rapidly produce production-ready instruction-tuning datasets.

## ✨ Key Features
- **🔍 Intelligent Data Cleaning & Deduplication**: MinHash/SimHash-based near-duplicate removal with configurable similarity thresholds.
- **📊 Automated Quality Scoring**: Multi-dimensional filtering (instruction clarity, response coherence, toxicity, length, and perplexity-based ranking).
- **🔄 Schema-Agnostic Format Conversion**: Seamless transformation to industry-standard fine-tuning formats (`ChatML`, `ShareGPT`, `Alpaca`, `OpenAI JSONL`).
- **⚡ Distributed Processing Engine**: Native Ray & Dask backends for horizontal scaling across multi-node clusters or cloud environments.
- **📦 Hugging Face & MLOps Integration**: Direct export to `datasets` library, MLflow tracking, and versioned artifact storage (S3/GCS compatible).

## ⚡ Quick Start

Prepare a clean instruction-tuning dataset in under 10 lines:

```python
from soloai.ftdatakit import Pipeline, DataLoader, Filter, Formatter

# 1. Initialize pipeline
pipeline = Pipeline(name="llama3-instruct-v1")

# 2. Load raw data (supports JSON, JSONL, Parquet, HF Datasets)
raw_data = DataLoader.from_json("raw_conversations.json")

# 3. Apply quality filters & deduplication
clean_data = (
    raw_data
    .pipe(Filter.remove_duplicates(method="minhash", threshold=0.85))
    .pipe(Filter.quality_score(min_length=50, max_length=4096))
    .pipe(Formatter.to_sharegpt())
)

# 4. Export to Hugging Face compatible format
clean_data.export("output/llama3_dataset", format="parquet", push_to_hub=False)
print(f"✅ Processed {len(clean_data)} high-quality samples.")
```

## 📖 Usage Examples

### 1️⃣ Custom Quality Filtering with LLM-Assisted Scoring
```python
from soloai.ftdatakit import Filter, LLMJudge

# Use a lightweight LLM to score instruction-response alignment
judge = LLMJudge(model="Qwen2.5-7B-Instruct", batch_size=16)

scored_data = data.pipe(
    Filter.apply_custom_scorer(
        scorer=judge.score_alignment,
        threshold=0.75,
        fallback="strict"
    )
)
```

### 2️⃣ Distributed Processing on Cluster
```python
from soloai.ftdatakit import Pipeline
import ray

# Initialize Ray cluster (local or remote)
ray.init(address="auto")

pipeline = Pipeline(backend="ray", num_workers=32)
pipeline.load("s3://data-bucket/raw/").pipe(
    Filter.remove_duplicates(),
    Filter.quality_score(),
    Formatter.to_chatml()
).export("s3://data-bucket/processed/")
```

### 3️⃣ Multi-Format Export & Dataset Versioning
```python
from soloai.ftdatakit import Exporter

exporter = Exporter(dataset=clean_data, version="v2.1.0")
exporter.to_jsonl("outputs/data.jsonl")
exporter.to_hf("SoloAI/llama3-instruct-v2")
exporter.generate_report("outputs/quality_metrics.md")
```
