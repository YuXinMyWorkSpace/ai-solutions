# 🤖 SoloAI: LLM Fine-tuning Data Service (大模型微调数据服务)
[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/llm-finetune-data/ci.yml?branch=main&label=build)](https://github.com/SoloAI/llm-finetune-data/actions)
[![License](https://img.shields.io/github/license/SoloAI/llm-finetune-data)](LICENSE)

A high-performance, enterprise-ready pipeline for generating, curating, and formatting instruction-tuning datasets optimized for Large Language Model (LLM) fine-tuning. Engineered to accelerate domain-specific model adaptation with automated quality control, multi-format export, and scalable distributed processing.

## ✨ Key Features
- **Automated SFT Data Generation**: LLM-driven synthesis and rule-based augmentation for high-quality instruction-response pairs.
- **Intelligent Quality Filtering**: Semantic deduplication, heuristic validation, toxicity screening, and perplexity-based scoring.
- **Multi-Format & Framework Compatibility**: Native export to JSONL, Parquet, Alpaca, ShareGPT, and Hugging Face `datasets` format.
- **Domain-Specific Template Engine**: Pre-configured pipelines for finance, healthcare, legal, code, and multilingual instruction tuning.
- **Distributed & GPU-Accelerated**: Built on Ray and PyTorch for horizontal scaling, enabling terabyte-scale dataset processing.
- **Built-in Evaluation Suite**: Automated metrics tracking (instruction-following accuracy, factual consistency, bias detection) pre-export.

## 📦 Installation
Requires Python 3.9+. Install via pip:
```bash
pip install soloai-llm-data
```
For GPU acceleration and distributed cluster support:
```bash
pip install soloai-llm-data[gpu,ray]
```

## 🚀 Quick Start
Initialize a fine-tuning pipeline and generate a validated dataset in under 5 lines of code:
```python
from soloai.data import FineTunePipeline

# Configure pipeline for financial domain SFT
pipeline = FineTunePipeline(
    domain="finance",
    output_format="jsonl",
    quality_threshold=0.85,
    dedup_method="minhash_lsh"
)

# Process raw corpus and export validated dataset
dataset = pipeline.generate_and_validate(raw_data_path="data/raw/")
dataset.save("output/finance_sft_dataset.jsonl")
```

## 📖 Usage Examples

### Custom Data Filtering & Augmentation
```python
from soloai.filters import SemanticDeduplicator, ToxicityFilter
from soloai.augment import BackTranslationAugmenter

filters = [
    SemanticDeduplicator(threshold=0.92, embedding_model="sentence-transformers/all-MiniLM-L6-v2"),
    ToxicityFilter(max_toxicity_score=0.15)
]
augmenter = BackTranslationAugmenter(languages=["zh", "en"])

processed_data = pipeline.apply_filters(raw_data, filters)
augmented_data = pipeline.apply_augmentation(processed_data, augmenter)
```

### CLI Batch Processing
```bash
soloai-cli process \
    --input ./raw_corpus/ \
    --output ./sft_datasets/ \
    --domain legal \
    --format hf \
    --workers 8 \
    --gpu-memory 0.8
```
