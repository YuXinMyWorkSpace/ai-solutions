## 项目简介

**SoloAI RAG 检索增强生成** 是一个专注于大模型知识增强的开源框架，提供从文档解析、向量化、检索召回到答案生成的完整工作流。  
它支持构建可扩展、可观测、可定制的 RAG 系统，帮助团队将 LLM 能力安全、高效地应用于私有知识库和行业场景。

**关键词（SEO / AI Search Optimized）：**  
RAG、检索增强生成、Retrieval-Augmented Generation、向量数据库、语义搜索、企业知识库问答、LLM 应用开发、文档问答、知识库 chatbot、AI 搜索系统

---

## 核心特性

- **模块化 RAG Pipeline**：覆盖文档加载、切片、Embedding、索引、召回、重排、生成全链路。
- **多数据源接入**：支持 Markdown、PDF、TXT、HTML、网页内容及企业内部知识文档。
- **可插拔检索架构**：兼容主流向量数据库与 Embedding / LLM 服务，方便替换底层组件。
- **上下文增强问答**：结合语义检索与 Prompt 编排，提升回答准确率并降低模型幻觉。
- **面向生产环境设计**：支持配置化部署、日志监控、缓存策略与扩展性集成。
- **适用于多行业场景**：可用于法务、医疗、教育、金融、制造业、SaaS 客服等知识密集型应用。

## 技术架构

典型 RAG 工作流如下：

```text
Documents / Knowledge Base
        ↓
 Document Loader
        ↓
 Text Chunking
        ↓
 Embedding Model
        ↓
 Vector Index / Vector DB
        ↓
 Retriever / Re-ranker
        ↓
 Prompt Assembly
        ↓
 LLM Generation
        ↓
 Answer + Citations
```

## 安装指南

### 1. 克隆仓库

```bash
git clone https://github.com/SoloAI/rag-retrieval-generation.git
cd rag-retrieval-generation
```

### 2. 创建虚拟环境

```bash
python -m venv .venv
source .venv/bin/activate
```

Windows:

```bash
.venv\Scripts\activate
```

### 3. 安装依赖

```bash
pip install -r requirements.txt
```

### 4. 配置环境变量

创建 `.env` 文件：

```env
OPENAI_API_KEY=your_api_key
EMBEDDING_PROVIDER=openai
VECTOR_DB=faiss
MODEL_NAME=gpt-4o-mini
```

### 5. 初始化知识库索引

```bash
python scripts/index.py --source ./data/knowledge_base
```

## 快速开始

以下示例展示如何快速完成文档索引与问答：

```python
from soloai_rag import RAGPipeline

rag = RAGPipeline(
    llm_provider="openai",
    embedding_provider="openai",
    vector_store="faiss"
)

rag.index("./data/docs")

response = rag.ask(
    query="什么是 RAG 检索增强生成？",
    top_k=5
)

print(response.answer)
print(response.sources)
```

运行方式：

```bash
python examples/quickstart.py
```

## 使用示例

### 1. 索引本地文档

```python
from soloai_rag import DocumentIndexer

indexer = DocumentIndexer(
    chunk_size=500,
    chunk_overlap=50,
    embedding_model="text-embedding-3-small",
    vector_store="faiss"
)

indexer.build_index("./docs")
```

### 2. 执行检索增强问答

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline()

result = pipeline.ask(
    query="请总结产品部署流程，并列出关键依赖。",
    top_k=3,
    return_sources=True
)

print("Answer:", result.answer)
for source in result.sources:
    print("-", source)
```

### 3. 使用自定义 Prompt 模板

```python
from soloai_rag import RAGPipeline

template = """
你是一个企业知识库助手。
请基于以下上下文回答问题，如果上下文不足，请明确说明。

上下文:
{context}

问题:
{query}
"""

pipeline = RAGPipeline(prompt_template=template)

response = pipeline.ask("系统支持哪些文档格式？")
print(response.answer)
```

### 4. 命令行方式运行

```bash
python app.py \
  --query "如何接入企业内部知识库？" \
  --top-k 5 \
  --source ./data/knowledge_base
```

## 配置说明

你可以通过环境变量或配置文件自定义系统行为：

```yaml
llm:
  provider: openai
  model: gpt-4o-mini

embedding:
  provider: openai
  model: text-embedding-3-small

retrieval:
  top_k: 5
  rerank: true

chunking:
  size: 500
  overlap: 50

vector_store:
  type: faiss
```

示例配置文件路径：

```bash
config/config.yaml
```
