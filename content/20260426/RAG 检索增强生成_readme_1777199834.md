# SoloAI RAG 检索增强生成
[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/rag/ci.yml?branch=main)](https://github.com/SoloAI/rag/actions)
[![License](https://img.shields.io/github/license/SoloAI/rag)](./LICENSE)

**SoloAI RAG 检索增强生成** is an open-source framework for building high-quality **Retrieval-Augmented Generation (RAG)** systems. It helps developers connect large language models with private or domain-specific knowledge bases to improve factual accuracy, reduce hallucinations, and deliver production-ready AI search and question-answering experiences.

## Overview

RAG（检索增强生成）combines semantic retrieval with large language model generation to ground responses in trusted data sources. This project by **SoloAI** provides a modular, extensible foundation for document ingestion, embedding, indexing, retrieval, reranking, and answer generation across enterprise and vertical AI applications.

## Features

- **Modular RAG Pipeline**: Document loading, chunking, embedding, vector indexing, retrieval, reranking, and generation
- **Multi-Source Knowledge Ingestion**: Support for Markdown, PDF, TXT, HTML, and structured business content
- **Pluggable Vector Backends**: Easily integrate FAISS, Milvus, Chroma, Pinecone, or other vector databases
- **LLM and Embedding Provider Agnostic**: Works with OpenAI-compatible APIs and self-hosted models
- **Production-Oriented Retrieval**: Hybrid search, metadata filtering, top-k retrieval, and context optimization
- **Developer-Friendly API**: Clean Python interface, CLI workflows, and configurable pipeline components

## Quick Start

Below is a minimal example to index local documents and query the knowledge base.

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline(
    embedding_model="text-embedding-3-large",
    llm_model="gpt-4o-mini",
    vector_store="faiss"
)

pipeline.ingest("./data")

response = pipeline.query(
    "什么是 RAG 检索增强生成？请结合知识库内容进行解释。",
    top_k=5
)

print(response.answer)
print(response.sources)
```

## Usage

### 1. Ingest documents

```python
from soloai_rag import DocumentIngestor

ingestor = DocumentIngestor(
    chunk_size=800,
    chunk_overlap=100
)

ingestor.load("./docs")
ingestor.process()
ingestor.save("./artifacts")
```

### 2. Build a vector index

```python
from soloai_rag import VectorIndexer

indexer = VectorIndexer(
    backend="faiss",
    embedding_model="text-embedding-3-large"
)

indexer.build_from_path("./artifacts")
indexer.persist("./index")
```

### 3. Retrieve relevant context

```python
from soloai_rag import Retriever

retriever = Retriever.load("./index")

results = retriever.search(
    query="企业知识库问答系统如何提升召回效果？",
    top_k=5,
    filters={"lang": "zh"}
)

for item in results:
    print(item.score, item.metadata["source"])
```

### 4. Generate grounded answers

```python
from soloai_rag import Generator

generator = Generator(model="gpt-4o-mini")

answer = generator.generate(
    query="请总结 RAG 系统在企业场景中的价值。",
    contexts=[
        "RAG 可以通过检索企业内部知识降低幻觉。",
        "结合向量检索与重排序可提升回答质量。"
    ]
)

print(answer)
```

### 5. Example CLI workflow

```bash
python -m soloai_rag.ingest --input ./docs --output ./artifacts
python -m soloai_rag.index --input ./artifacts --index ./index
python -m soloai_rag.query --index ./index --question "RAG 如何减少幻觉？"
```
