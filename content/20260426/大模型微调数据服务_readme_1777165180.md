## 📖 Overview
SoloAI FineTuneData is an enterprise-grade pipeline for generating, validating, and managing high-quality instruction-tuning and alignment datasets for large language models. It automates data cleaning, quality scoring, and multi-format export to accelerate LLM fine-tuning workflows.

## ✨ Key Features
- **Automated Data Synthesis & Augmentation**: Generate domain-specific instruction-response pairs with configurable templates, controlled randomness, and synthetic preference generation.
- **Intelligent Quality Filtering**: Multi-stage validation pipeline featuring semantic deduplication, toxicity screening, and heuristic quality scoring.
- **Multi-Format Export Engine**: Seamless conversion to JSONL, Parquet, and Hugging Face `datasets` formats with strict schema validation.
- **Framework-Native Integration**: Drop-in compatibility with Hugging Face Transformers, Axolotl, vLLM, Unsloth, and DeepSpeed training loops.
- **Scalable Distributed Processing**: Built on Ray/Polars for parallelized processing of multi-terabyte corpora without memory bottlenecks.
- **Audit-Ready Data Lineage**: Full version tracking, provenance logging, and reproducibility guarantees for enterprise compliance and research.

## 🚀 Quick Start
Initialize a data pipeline and process raw text into fine-tuning-ready instruction format:
```python
from soloai.finetune import DataPipeline, QualityFilter

# Configure pipeline with quality thresholds
pipeline = DataPipeline(
    input_path="raw_corpus.json",
    output_path="finetune_ready.jsonl",
    min_quality_score=0.85,
    dedup_strategy="minhash_lsh"
)

# Execute cleaning, formatting, and validation
stats = pipeline.run()
print(f"Processed {stats['total']} records. Kept {stats['passed']} after filtering.")
```

## 🛠️ Usage Examples
### Domain-Specific Template Application
```python
from soloai.finetune import TemplateEngine, Exporter

# Load pre-configured domain templates (e.g., legal, medical, code)
engine = TemplateEngine.load_domain("financial_qa")
processed_data = engine.transform("raw_financial_reports.json")

# Export to Hugging Face Hub
exporter = Exporter(format="hf_datasets")
dataset = exporter.convert(processed_data, split="train")
dataset.push_to_hub("your-org/fin-qna-v2")
```

### Integration with Axolotl / Transformers
```yaml
# axolotl_config.yaml
base_model: meta-llama/Llama-3-8B-Instruct
datasets:
  - path: ./finetune_ready.jsonl
    type: chatml
    field: messages
```
