## 📖 Overview
SoloRAG is a production-ready, modular Retrieval-Augmented Generation (RAG) framework engineered to seamlessly integrate domain-specific knowledge bases with large language models. It delivers deterministic, context-aware outputs through optimized semantic retrieval, intelligent chunking, and enterprise-grade observability.

## ✨ Key Features
- **Hybrid Retrieval Pipeline**: Combines dense vector search with BM25 lexical matching and cross-encoder re-ranking for maximum recall & precision.
- **Multi-Vector Store Support**: Native adapters for Milvus, Qdrant, Chroma, Weaviate, and PostgreSQL/pgvector.
- **Adaptive Context Management**: Dynamic window sizing, token-aware chunking, and metadata filtering to prevent hallucination and context overflow.
- **Pluggable LLM Routing**: Unified interface for OpenAI, Anthropic, vLLM, Ollama, and Azure OpenAI with automatic fallback & load balancing.
- **Built-in Evaluation & Tracing**: Integrated RAGAS metrics, LangSmith/OpenTelemetry tracing, and automated regression testing for production monitoring.

## 🚀 Quick Start

```python
from solorag import RAGPipeline, EmbeddingModel, VectorStore

# 1. Initialize core components
embedder = EmbeddingModel("text-embedding-3-small")
vector_db = VectorStore("chroma", collection_name="company_knowledge")
pipeline = RAGPipeline(
    embedder=embedder,
    vector_store=vector_db,
    llm="gpt-4o-mini",
    top_k=3
)

# 2. Ingest documents (supports PDF, Markdown, TXT, DOCX)
pipeline.ingest(["docs/architecture.pdf", "docs/api_reference.md"])

# 3. Query with grounded generation
response = pipeline.query("How do I configure SSO authentication for the admin dashboard?")
print(f"✅ Answer: {response.answer}")
print(f"📚 Sources: {[doc.metadata['source'] for doc in response.sources]}")
```

## 🔧 Usage Examples

### Advanced Configuration & Streaming
```python
from solorag.config import RetrievalConfig, ChunkConfig
from solorag.retrievers import HybridRetriever
from solorag.generators import StreamingLLM

# Configure hybrid search with cross-encoder reranking
retrieval_cfg = RetrievalConfig(
    top_k=8,
    rerank_top_k=3,
    reranker_model="cross-encoder/ms-marco-MiniLM-L-6-v2",
    enable_hybrid_search=True
)

# Initialize streaming generator
llm = StreamingLLM(model="llama-3-70b-instruct", temperature=0.2)
retriever = HybridRetriever(vector_store=vector_db, config=retrieval_cfg)

# Stream response with real-time token generation
prompt = "Explain the deployment architecture for high-availability mode."
for token in llm.stream(prompt, retriever=retriever):
    print(token, end="", flush=True)
```

### Custom Chunking & Metadata Filtering
```python
chunk_cfg = ChunkConfig(
    strategy="semantic",
    chunk_size=512,
    chunk_overlap=64,
    separator="\n\n"
)

# Filter by document type and department
response = pipeline.query(
    "What is the data retention policy?",
    metadata_filter={"doc_type": "compliance", "department": "legal"},
    chunk_config=chunk_cfg
)
```
