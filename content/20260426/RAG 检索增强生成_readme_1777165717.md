# SoloRAG: Enterprise-Grade Retrieval-Augmented Generation Framework
[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/SoloRAG/ci.yml?branch=main&label=build&style=flat-square)](https://github.com/SoloAI/SoloRAG/actions)
[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg?style=flat-square)](https://opensource.org/licenses/Apache-2.0)

**SoloRAG** is a high-performance, modular Retrieval-Augmented Generation (RAG) framework engineered by SoloAI to bridge proprietary knowledge bases with large language models (LLMs). It delivers accurate, context-aware responses with optimized vector retrieval, hybrid search, and production-grade observability for enterprise AI deployments.

## ✨ Key Features
- **🔍 Hybrid Retrieval Engine:** Combines dense vector similarity, BM25 sparse search, and metadata filtering for maximum recall & precision.
- **🧩 Adaptive Chunking & Semantic Routing:** Context-aware document splitting with dynamic routing to optimize LLM context window utilization.
- **🌐 Multi-Backend Compatibility:** Plug-and-play support for major vector databases (Qdrant, Milvus, Weaviate, Pinecone) and LLM providers (OpenAI, Anthropic, vLLM, Ollama).
- **📊 Production Observability:** Built-in OpenTelemetry tracing, latency monitoring, and automated RAG evaluation metrics (faithfulness, context relevance, hallucination rate).
- **⚡ High-Throughput Pipeline:** Async execution, intelligent KV-caching, and batch processing for enterprise-scale query workloads.

## 🚀 Quick Start
Initialize a complete RAG pipeline in under 10 lines of code:
```python
from sorag import RAGPipeline, VectorStore, LLMProvider

# 1. Configure components
vector_db = VectorStore(provider="qdrant", collection="enterprise_knowledge")
llm = LLMProvider(model="openai/gpt-4o", api_key="YOUR_API_KEY")

# 2. Initialize pipeline
pipeline = RAGPipeline(
    retriever=vector_db,
    generator=llm,
    chunk_size=512,
    top_k=5
)

# 3. Ingest documents
pipeline.ingest(["docs/technical_manual.pdf", "docs/faq.md"])

# 4. Query
response = pipeline.query("How do I configure the authentication module?")
print(response.answer)
```

## 🛠️ Usage Examples

### Hybrid Search with Metadata Filtering
```python
from sorag.retrieval import HybridRetriever

retriever = HybridRetriever(
    dense_weight=0.7,
    sparse_weight=0.3,
    metadata_filter={"department": "engineering", "version": ">=2.0"}
)
results = retriever.search("API rate limits", top_k=3)
```

### Streaming Responses & Custom System Prompts
```python
response = pipeline.query(
    "Explain the deployment architecture",
    system_prompt="You are a senior cloud architect. Provide concise, step-by-step explanations.",
    stream=True
)

for chunk in response.stream():
    print(chunk.text, end="", flush=True)
```

### Evaluation & Benchmarking
```python
from sorag.eval import RAGEvaluator

evaluator = RAGEvaluator(dataset="qa_benchmark.json")
report = evaluator.run(pipeline, metrics=["faithfulness", "context_precision", "answer_relevance"])
report.save("rag_eval_report.json")
```
