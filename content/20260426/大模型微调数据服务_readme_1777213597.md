面向企业与研究团队的**大模型微调数据服务**开源项目，专注于提供高质量、可扩展、可审计的数据处理流程，用于支持 LLM/多模态模型的指令微调、偏好对齐与行业数据构建。  
该项目帮助团队标准化完成数据采集、清洗、标注、质检、格式转换与交付，加速大模型训练与落地。

## 项目简介

**SoloAI 大模型微调数据服务** 致力于为大语言模型（LLM）微调提供完整的数据工程能力，包括训练数据集构建、SFT 数据整理、DPO/偏好数据处理、行业知识数据清洗与多格式导出。  
项目适用于金融、医疗、教育、客服、法律、电商、工业等场景，帮助团队以更低成本、更高质量地完成 AI 数据交付。

---

## 核心特性

- **多场景微调数据支持**：支持 SFT、Instruction Tuning、DPO、RLHF 前置数据、分类、抽取、问答、对话等任务类型。
- **标准化数据处理流水线**：覆盖数据采集、去重、脱敏、清洗、切分、标注、质检、格式转换与版本管理。
- **多格式兼容与导出**：支持 JSON、JSONL、CSV、Parquet 等常见训练数据格式，兼容主流微调框架。
- **可审计与可扩展**：支持质量评估、规则校验、样本追踪与批次管理，便于企业级交付与合规管理。
- **行业定制化能力**：支持按垂直领域构建术语体系、标签规范、标注策略与验收标准。
- **面向生产的工程实践**：便于接入现有 MLOps / DataOps 流程，用于批量化、自动化数据服务。

## 快速开始

以下示例演示如何将原始问答数据转换为适用于大模型监督微调（SFT）的 JSONL 格式。

### 1) 准备原始数据

```json
[
  {
    "instruction": "解释什么是向量数据库",
    "input": "",
    "output": "向量数据库是一类专门用于存储、索引和检索高维向量的数据系统。"
  },
  {
    "instruction": "总结下面文本的核心观点",
    "input": "大模型在企业场景中的价值主要体现在自动化、知识增强和交互效率提升。",
    "output": "核心观点是：大模型能够提升企业自动化能力、增强知识利用效率并改善交互体验。"
  }
]
```

### 2) 执行数据处理命令

```bash
python -m soloai.pipeline \
  --input ./data/raw/sample.json \
  --output ./data/processed/sft.jsonl \
  --task sft \
  --clean \
  --deduplicate \
  --validate
```

### 3) 输出示例

```json
{"messages":[{"role":"user","content":"解释什么是向量数据库"},{"role":"assistant","content":"向量数据库是一类专门用于存储、索引和检索高维向量的数据系统。"}]}
{"messages":[{"role":"user","content":"总结下面文本的核心观点\n\n大模型在企业场景中的价值主要体现在自动化、知识增强和交互效率提升。"},{"role":"assistant","content":"核心观点是：大模型能够提升企业自动化能力、增强知识利用效率并改善交互体验。"}]}
```

## 使用示例

## 1. 构建 SFT 数据集

```bash
python -m soloai.pipeline \
  --input ./data/raw/customer_service.csv \
  --output ./data/processed/customer_service_sft.jsonl \
  --task sft \
  --schema instruction,input,output \
  --clean \
  --quality-report
```

适用于：
- 客服问答微调
- 企业知识库助手
- 行业专家对话模型

---

## 2. 构建偏好对齐 / DPO 数据

```bash
python -m soloai.pipeline \
  --input ./data/raw/preference_pairs.jsonl \
  --output ./data/processed/dpo_pairs.jsonl \
  --task dpo \
  --validate \
  --normalize
```

示例输入：

```json
{
  "prompt": "请写一个专业的项目介绍",
  "chosen": "这是结构清晰、专业严谨的版本。",
  "rejected": "这是较弱、表达不完整的版本。"
}
```

---

## 3. 数据清洗与脱敏

```bash
python -m soloai.pipeline \
  --input ./data/raw/domain_data.jsonl \
  --output ./data/processed/domain_data_clean.jsonl \
  --task clean \
  --remove-pii \
  --deduplicate \
  --language zh
```

适用于：
- 医疗文本脱敏
- 金融合规数据清洗
- 法务文档预处理

---

## 4. Python API 调用示例

```python
from soloai.pipeline import DataPipeline

pipeline = DataPipeline(
    task="sft",
    clean=True,
    deduplicate=True,
    validate=True
)

pipeline.run(
    input_path="./data/raw/train.json",
    output_path="./data/processed/train_sft.jsonl"
)
```
