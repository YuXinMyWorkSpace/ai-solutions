## Installation

### Requirements

- Python 3.9+
- Linux/macOS recommended
- CUDA-compatible GPU optional but recommended for high-throughput inference
- Docker optional for containerized deployment

### Install from source

```bash
git clone https://github.com/SoloAI/llm-deployment-optimization.git
cd llm-deployment-optimization
python -m venv .venv
source .venv/bin/activate
pip install -U pip
pip install -r requirements.txt
```

### Optional: install with development dependencies

```bash
pip install -r requirements-dev.txt
```

### Optional: Docker setup

```bash
docker build -t soloai-llm-opt .
docker run --rm -it -p 8080:8080 soloai-llm-opt
```

## Quick Start

Below is a minimal example showing how to start an optimized LLM inference service.

```bash
python -m soloai_llm_opt.serve \
  --model /models/your-llm \
  --host 0.0.0.0 \
  --port 8080 \
  --dtype float16 \
  --max-batch-size 16 \
  --num-workers 2
```

Test the API:

```bash
curl -X POST "http://127.0.0.1:8080/generate" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Explain how to optimize LLM inference latency.",
    "max_tokens": 128,
    "temperature": 0.7
  }'
```

Example response:

```json
{
  "text": "LLM inference latency can be optimized through quantization, batching, efficient scheduling, KV cache tuning, and hardware-aware deployment strategies."
}
```

## Usage Examples

### 1. Run with quantization enabled

Use quantization to reduce memory footprint and improve deployment efficiency.

```bash
python -m soloai_llm_opt.serve \
  --model /models/your-llm \
  --quantization int8 \
  --max-batch-size 32 \
  --port 8080
```

### 2. Enable metrics and monitoring

Expose performance metrics for observability and production tuning.

```bash
python -m soloai_llm_opt.serve \
  --model /models/your-llm \
  --enable-metrics \
  --metrics-port 9090
```

### 3. Python integration example

```python
from soloai_llm_opt.client import LLMClient

client = LLMClient(base_url="http://127.0.0.1:8080")

response = client.generate(
    prompt="Give me 3 strategies for reducing LLM serving cost.",
    max_tokens=100,
    temperature=0.5
)

print(response["text"])
```

### 4. Example production configuration

```yaml
server:
  host: 0.0.0.0
  port: 8080
  workers: 4

model:
  path: /models/your-llm
  dtype: float16
  quantization: int8

inference:
  max_batch_size: 32
  max_input_tokens: 4096
  max_output_tokens: 512
  timeout_ms: 30000

monitoring:
  enable_metrics: true
  log_level: info
```

Start with config:

```bash
python -m soloai_llm_opt.serve --config config.yaml
```
