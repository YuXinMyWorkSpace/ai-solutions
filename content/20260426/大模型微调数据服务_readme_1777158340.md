## 📖 Overview
A production-ready data pipeline and service framework engineered to generate, clean, validate, and format high-quality instruction-tuning datasets for Large Language Models (LLMs). SoloAI streamlines the entire SFT/RLHF data lifecycle, enabling developers and enterprises to accelerate model alignment with enterprise-grade reliability, scalability, and domain adaptability.

## ✨ Key Features
- 🔹 **Automated Data Synthesis & Augmentation**: LLM-powered generation of diverse, high-signal instruction-response pairs with controlled temperature and diversity sampling.
- 🔹 **Multi-Format Export Engine**: Native support for ShareGPT, Alpaca, JSONL, Parquet, and Hugging Face `datasets` format with zero-config conversion.
- 🔹 **Intelligent Quality Control**: Semantic deduplication (MinHash/SimHash), toxicity filtering, length normalization, and LLM-as-a-Judge scoring for data purity.
- 🔹 **Domain-Specific Template Engine**: Pre-built schemas for Finance, Healthcare, Legal, Code, and Customer Service with customizable prompt routing.
- 🔹 **Seamless Ecosystem Integration**: Direct compatibility with `unsloth`, `trl`, `axolotl`, `vLLM`, and Hugging Face Hub for one-click fine-tuning handoff.
- 🔹 **Privacy-First & On-Premise Ready**: Fully offline execution, local LLM routing, and GDPR/CCPA-compliant data handling pipelines.

## 🚀 Quick Start
```python
from soloai import DataPipeline, DatasetConfig

# 1. Configure pipeline
config = DatasetConfig(
    domain="finance",
    output_format="sharegpt",
    quality_threshold=0.85,
    dedup_method="semantic",
    max_tokens=2048
)

# 2. Initialize & run
pipeline = DataPipeline(config)
dataset = pipeline.generate_and_validate(
    seed_data="seed_instructions.jsonl",
    target_size=5000
)

# 3. Export
dataset.export("output/finance_sft_v1.jsonl")
print(f"✅ Generated {len(dataset)} validated samples.")
```

## 🛠️ Usage Examples

### 📊 CLI Batch Processing
```bash
soloai generate \
  --input raw_corpus.csv \
  --domain healthcare \
  --format alpaca \
  --quality 0.9 \
  --output healthcare_sft.jsonl \
  --workers 8
```

### 🔍 Custom Quality Filtering & Scoring
```python
from soloai.quality import QualityScorer

scorer = QualityScorer(model="meta-llama/Llama-3-8B-Instruct")
scores = scorer.evaluate(dataset)
filtered = dataset.filter(lambda x: x["score"] > 0.88)
filtered.save("output/filtered_high_quality.parquet")
```

### 🔄 Hugging Face Hub Integration
```python
from datasets import Dataset
from soloai.export import to_hf_dataset

hf_ds = to_hf_dataset("output/finance_sft_v1.jsonl")
hf_ds.push_to_hub("SoloAI/finance-instruct-v1")
```
