> **大模型微调数据服务 (LLM Fine-Tuning Data Service)**  
A production-grade, open-source pipeline for generating, cleaning, formatting, and versioning high-quality instruction-tuning datasets for Large Language Models. Engineered for enterprise scalability and seamless integration with modern training frameworks (Hugging Face, PEFT, vLLM, Unsloth, Axolotl).

---

## ✨ Key Features
- **Automated Data Curation & Deduplication**: MinHash/SimHash + semantic similarity filtering to remove near-duplicates and low-signal samples.
- **Multi-Format Conversion Engine**: Native support for JSONL, CSV, Parquet, and Arrow → Hugging Face `datasets` format with schema validation.
- **Quality Scoring & Filtering**: Heuristic rules, perplexity-based ranking, and optional LLM-as-a-judge evaluation for instruction-response alignment.
- **Distributed Processing Backend**: Ray/Dask integration for billion-scale dataset processing with automatic sharding and memory optimization.
- **Framework-Agnostic Export**: One-click export to LoRA/QLoRA, Alpaca, ShareGPT, or OpenAI fine-tuning formats with built-in tokenization caching.

## ⚡ Quick Start

```python
from soloai.data import Pipeline, Filter, Formatter

# Initialize pipeline
pipeline = Pipeline(
    backend="ray",
    workers=16,
    cache_dir="./.cache"
)

# Load raw data
pipeline.load("data/raw_instructions.jsonl")

# Apply cleaning & quality filters
pipeline.apply(Filter.deduplicate(method="minhash", threshold=0.85))
pipeline.apply(Filter.length_filter(min_tokens=40, max_tokens=4096))
pipeline.apply(Filter.quality_score(model="meta-llama/Llama-3-8B-Instruct"))

# Format & export
pipeline.apply(Formatter.to_alpaca())
pipeline.export("hf_dataset", format="parquet", output_dir="./output/train")

print("✅ Dataset ready for fine-tuning.")
```

## 🛠 Usage Examples

### 1. YAML Configuration for Reproducible Pipelines
```yaml
# pipeline_config.yaml
input: "data/medical_qa.jsonl"
filters:
  - type: dedup
    method: minhash
    threshold: 0.9
  - type: toxicity
    threshold: 0.15
  - type: instruction_following
    model: "microsoft/Phi-3-mini-4k-instruct"
    threshold: 0.8
formatter: "sharegpt"
output:
  path: "data/cleaned/medical_train.parquet"
  split: {"train": 0.85, "val": 0.1, "test": 0.05}
```

Run via CLI:
```bash
soloai-data process --config pipeline_config.yaml --backend ray --workers 32
```

### 2. Hugging Face Integration
```python
from datasets import load_from_disk
from transformers import AutoTokenizer

ds = load_from_disk("./output/train")
tokenizer = AutoTokenizer.from_pretrained("Qwen/Qwen2.5-7B-Instruct")

def tokenize_fn(batch):
    return tokenizer(batch["text"], truncation=True, max_length=4096)

tokenized_ds = ds.map(tokenize_fn, batched=True, remove_columns=["text"])
tokenized_ds.save_to_disk("./output/tokenized_train")
```
