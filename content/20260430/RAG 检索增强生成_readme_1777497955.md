## 📖 Overview
A production-grade, modular Retrieval-Augmented Generation (RAG) framework engineered by **SoloAI**. It unifies advanced semantic retrieval, intelligent document chunking, and LLM orchestration into a single, highly optimized pipeline for building accurate, hallucination-resistant, and enterprise-ready AI applications.

## ✨ Key Features
- 🔍 **Hybrid Retrieval Engine**: Combines dense vector search (embedding-based) with sparse lexical search (BM25) for optimal recall and precision across diverse query types.
- 🧩 **Adaptive Context Chunking**: Implements semantic-aware splitting, sliding windows, and metadata preservation to maximize LLM context utilization without token waste.
- 🔌 **Pluggable Architecture**: Drop-in adapters for major vector databases (Chroma, Weaviate, Milvus, Qdrant) and LLM providers (OpenAI, Anthropic, vLLM, Ollama).
- 📊 **Built-in RAG Evaluation**: Integrated metrics pipeline (Faithfulness, Answer Relevance, Context Precision) aligned with industry benchmarks like RAGAS.
- 🛡️ **Enterprise Observability**: Structured logging, traceable retrieval paths, latency profiling, and cost tracking for production deployment.

## 🚀 Quick Start
Initialize a pipeline, ingest documents, and query in under 10 lines of code.

```python
from soloai_rag import RAGPipeline, VectorStore, LLMAdapter, ChunkingStrategy

# 1. Configure core components
rag = RAGPipeline(
    vector_store=VectorStore(provider="chromadb", collection="knowledge_base"),
    llm=LLMAdapter(provider="openai", model="gpt-4o-mini"),
    chunking=ChunkingStrategy(method="semantic", max_tokens=512, overlap=100)
)

# 2. Ingest & index documents
rag.ingest(["./docs/product_manual.pdf", "./data/faq.md"])

# 3. Query with grounded generation
result = rag.query("How do I configure OAuth2 for the API gateway?")
print(f"Answer: {result.answer}")
print(f"Sources: {[doc.metadata['source'] for doc in result.retrieved_docs]}")
```

## 🛠️ Usage Examples

### 🔹 Advanced Hybrid Retrieval & Re-ranking
```python
from soloai_rag.retrievers import HybridRetriever, CrossEncoderReranker

retriever = HybridRetriever(
    dense_weight=0.7,
    sparse_weight=0.3,
    top_k=10,
    reranker=CrossEncoderReranker(model="cross-encoder/ms-marco-MiniLM-L-6-v2", top_k=3)
)

# Attach to pipeline
rag.set_retriever(retriever)
response = rag.query("Compare PostgreSQL vs MongoDB for time-series workloads")
```

### 🔹 Production Evaluation Pipeline
```python
from soloai_rag.eval import RAGEvaluator, DatasetLoader

# Load ground-truth QA pairs
dataset = DatasetLoader.from_csv("evaluation/qa_test_set.csv")

# Run evaluation
evaluator = RAGEvaluator(pipeline=rag, metrics=["faithfulness", "answer_relevance"])
report = evaluator.run(dataset)

print(report.summary())
report.export_html("rag_eval_report.html")
```
