## ✨ Features
- **Dynamic KV-Cache & Memory Pooling**: Intelligent VRAM management reduces fragmentation and supports 2–4× longer context windows.
- **Multi-Backend Orchestration**: Unified API for `vLLM`, `TensorRT-LLM`, and `TGI` with automatic hardware-aware routing.
- **Advanced Quantization Pipeline**: Seamless INT4/INT8/AWQ/GPTQ conversion with built-in calibration and accuracy preservation checks.
- **Adaptive Batching & Speculative Decoding**: Real-time request scheduling and draft-model acceleration for sub-100ms latency.
- **Production Telemetry & Auto-Scaling**: OpenTelemetry-native metrics, A/B testing hooks, and Kubernetes HPA integration.
- **Zero-Downtime Rollouts**: Hot-swappable model weights with graceful connection draining and stateful session migration.

**Prerequisites**: Python 3.9+, CUDA 11.8+ (for GPU acceleration), and `pip` 21.3+.

```bash

pip install soloai-llm-opt

# Full GPU + quantization backend support
pip install "soloai-llm-opt[gpu,quant,telemetry]"

# Verify installation
python -c "import soloai; print(soloai.__version__)"
```

## 🚀 Quick Start
Deploy an optimized LLM endpoint in under 10 lines of code:

```python
from soloai import LLMOptimizer, ModelConfig

config = ModelConfig(
    model_id="meta-llama/Llama-3-8B-Instruct",
    backend="vllm",
    quantization="awq_int4",
    max_seq_len=4096,
    tensor_parallel_size=1
)

optimizer = LLMOptimizer(config)
optimizer.deploy(host="0.0.0.0", port=8000)
print("✅ Optimized LLM endpoint running at http://localhost:8000/v1")
```

## 💻 Usage Examples

### 1. Batch Inference with Adaptive Scheduling
```python
from soloai import LLMOptimizer

optimizer = LLMOptimizer.from_pretrained("mistralai/Mistral-7B-v0.1", backend="trt-llm")

responses = optimizer.infer(
    prompts=[
        "Summarize the latest advancements in LLM inference optimization.",
        "Translate the following technical spec to Spanish: ..."
    ],
    max_new_tokens=512,
    temperature=0.6,
    adaptive_batching=True,
    speculative_decoding=True
)

for i, res in enumerate(responses, 1):
    print(f"[{i}] {res.text.strip()}")
```

### 2. Performance Benchmarking & Telemetry Export
```python
metrics = optimizer.run_benchmark(
    dataset="mmlu_subset.json",
    batch_sizes=[1, 4, 16],
    warmup_steps=3
)

print(f"Peak Throughput: {metrics['max_tps']:.2f} tokens/sec")
print(f"VRAM Utilization: {metrics['peak_vram_gb']:.1f} GB")

# Export for Prometheus/Grafana
optimizer.export_telemetry(format="prometheus", output="metrics.yaml")
```
