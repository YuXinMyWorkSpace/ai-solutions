# 🚀 SoloAI/llm-deploy-optimizer
![Build Status](https://img.shields.io/badge/build-passing-brightgreen) ![License](https://img.shields.io/badge/license-MIT-blue.svg) ![Python](https://img.shields.io/badge/python-3.9%2B-blue) ![CUDA](https://img.shields.io/badge/CUDA-11.8%2B-green)

A production-grade toolkit for optimizing Large Language Model (LLM) inference and deployment. Engineered to minimize latency, maximize GPU memory efficiency, and streamline horizontal scaling across heterogeneous compute environments.

## ✨ Key Features
- **Dynamic Batching & Scheduling:** Adaptive request routing that maximizes throughput under variable traffic loads while maintaining strict SLA guarantees.
- **Advanced KV Cache Management:** Paged attention with memory-efficient eviction policies, enabling 4× longer context windows without OOM failures.
- **Hardware-Aware Quantization:** Seamless INT8/INT4/AWQ/GPTQ integration with automatic precision calibration and fallback mechanisms.
- **Multi-GPU & Distributed Inference:** Optimized tensor/pipeline parallelism with zero-overhead RPC communication and NCCL-aware topology mapping.
- **Real-Time Observability:** Built-in Prometheus/Grafana exporters tracking latency, tokens/sec, cache hit rates, and VRAM utilization.
- **Framework-Agnostic Backend:** Drop-in compatibility with Hugging Face `transformers`, `vLLM`, `TensorRT-LLM`, and custom ONNX pipelines.

## 📦 Installation
Requires Python 3.9+ and CUDA 11.8+ (for GPU acceleration).

```bash
# Standard CPU/Compatibility Build
pip install soloai-llm-deploy-opt

# GPU-Accelerated Build (Recommended)
pip install soloai-llm-deploy-opt[gpu]

# Development Build
git clone https://github.com/SoloAI/llm-deploy-optimizer.git
cd llm-deploy-optimizer
pip install -e ".[dev]"
```

## ⚡ Quick Start
Initialize the optimizer and deploy a model in under 10 lines:

```python
from soloai.llm_deploy import Optimizer, ModelConfig

config = ModelConfig(
    model_id="meta-llama/Llama-3-8B-Instruct",
    quantization="int8",
    max_batch_size=64,
    kv_cache_ratio=0.8,
    device="cuda"
)

engine = Optimizer(config)
engine.start()

# Synchronous inference
response = engine.generate("Explain quantum computing in simple terms.", max_tokens=256)
print(response.text)
```

## 🛠️ Usage Examples

### Multi-GPU Distributed Deployment
```python
from soloai.llm_deploy import DistributedConfig

dist_config = DistributedConfig(
    num_gpus=4,
    parallel_mode="tensor",
    nccl_timeout=30,
    gpu_ids=[0, 1, 2, 3]
)
engine = Optimizer(config, distributed=dist_config)
engine.start()
```

### Real-Time Metrics Export
Enable Prometheus scraping on port `9090` for Grafana dashboards:
```python
engine.enable_metrics(port=9090, namespace="llm_inference", scrape_interval="10s")
```

### Custom Request Scheduler with Preemption
```python
from soloai.llm_deploy import PriorityScheduler

scheduler = PriorityScheduler(
    max_queue_depth=2000,
    preemption_enabled=True,
    preemption_threshold_ms=500
)
engine.set_scheduler(scheduler)
```
