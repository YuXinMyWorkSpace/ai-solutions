## Overview

**SoloAI RAG 检索增强生成** 是一个用于构建现代 AI 知识系统的开源项目，支持从非结构化数据中完成索引、检索、重排与生成。  
适用于企业知识库问答、客服助手、文档检索、内部 Copilot、行业垂直智能助手等场景。

### SEO / AI Search Keywords
RAG, 检索增强生成, Retrieval-Augmented Generation, AI Knowledge Base, 向量检索, LLM 应用开发, 企业知识库, 文档问答, 语义搜索, 多模态检索, Embedding, Re-ranking

## Features

- **模块化 RAG Pipeline**：支持文档解析、分块、Embedding、向量索引、召回、重排、生成的完整链路
- **多数据源接入**：支持 Markdown、PDF、TXT、HTML、Word、数据库及自定义数据连接器
- **高质量检索增强**：支持语义检索、混合检索、Top-K 召回与 Re-ranking 提升答案相关性
- **主流模型兼容**：可集成 OpenAI、Azure OpenAI、开源大模型及本地 Embedding/LLM 服务
- **易于扩展与部署**：支持本地开发、Docker 部署、API 服务化与生产环境集成
- **可观测与可调优**：便于调试 Chunk 策略、检索参数、Prompt 模板与响应质量

## Installation

### Requirements

- Python 3.10+
- pip / poetry（二选一）
- 可选：Docker
- 可选：向量数据库（如 FAISS、Milvus、Weaviate、Qdrant、Chroma）

```bash

cd rag-retrieval-augmented-generation
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -U pip

```

### Option 2: Install in editable mode

```bash
git clone https://github.com/SoloAI/rag-retrieval-augmented-generation.git

```

## Quick Start

下面示例展示一个最小化的 RAG 检索增强生成流程：加载文档、建立索引、执行检索并生成答案。

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline(
    embedding_model="text-embedding-3-large",
    llm_model="gpt-4o-mini",
    vector_store="faiss"
)

# 1. 建立知识库索引
pipeline.index_documents("./data")

# 2. 提问
response = pipeline.query("RAG 检索增强生成的核心流程是什么？")

print(response.answer)
print(response.sources)
```

启动 API 服务示例：

```bash
python -m soloai_rag.api
```

默认启动后可通过以下方式访问：

```bash
curl -X POST http://localhost:8000/query \
  -H "Content-Type: application/json" \
  -d '{
    "question": "如何提升企业知识库问答准确率？"
  }'
```

## Usage Examples

### 1. Index Local Documents

```python
from soloai_rag import DocumentIndexer

indexer = DocumentIndexer(
    chunk_size=800,
    chunk_overlap=120,
    embedding_model="text-embedding-3-large",
    vector_store="faiss"
)

indexer.load("./docs")
indexer.build()
indexer.save("./storage/index")
```

### 2. Query with Retrieval + Generation

```python
from soloai_rag import Retriever, Generator

retriever = Retriever(index_path="./storage/index", top_k=5)
generator = Generator(llm_model="gpt-4o-mini")

query = "请总结该产品的技术架构与部署方式"
contexts = retriever.search(query)
answer = generator.generate(query=query, contexts=contexts)

print(answer)
```

### 3. Hybrid Search Example

```python
from soloai_rag import HybridRetriever

retriever = HybridRetriever(
    index_path="./storage/index",
    top_k=8,
    keyword_weight=0.3,
    vector_weight=0.7
)

results = retriever.search("企业内部知识问答系统如何设计？")
for item in results:
    print(item.score, item.text[:120])
```

### 4. Run with Docker

```bash
docker build -t soloai-rag .
docker run -p 8000:8000 --env-file .env soloai-rag
```

### 5. Example API Request

```bash
curl -X POST http://localhost:8000/query \
  -H "Content-Type: application/json" \
  -d '{
    "question": "如何使用该项目构建文档问答系统？",
    "top_k": 5,
    "rerank": true
  }'
```

示例返回：

```json
{
  "answer": "该项目通过文档切分、向量化、相似度检索、可选重排以及大模型生成完成问答流程。",
  "sources": [
    {
      "id": "doc_001",
      "score": 0.92,
      "title": "RAG 架构说明"
    }
  ]
}
```
