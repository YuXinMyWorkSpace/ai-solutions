# SoloAI RAG 检索增强生成
[![Build Status](https://img.shields.io/badge/build-passing-brightgreen.svg)](./)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](./LICENSE)

**SoloAI RAG 检索增强生成** 是一个面向生产环境的开源框架，用于构建高质量、可扩展、可观测的 **RAG（Retrieval-Augmented Generation，检索增强生成）** 应用。  
它帮助开发者将企业知识库、向量检索、重排序与大语言模型推理高效整合，提升问答准确率、可解释性与落地效率。

---

## Features

- **模块化 RAG Pipeline**  
  支持文档加载、切分、Embedding、向量检索、重排序、上下文拼接与生成的完整链路。

- **多数据源接入**  
  支持本地文件、Markdown、PDF、HTML、数据库及自定义知识源扩展。

- **高质量检索与重排序**  
  集成向量检索、混合检索与 reranker，提升召回效果和答案相关性。

- **面向生产环境设计**  
  支持配置化部署、日志监控、可观测性、缓存机制与 API 服务化。

- **易于集成主流 LLM / Vector DB**  
  可对接 OpenAI、Ollama、通义、GLM 等模型，以及 FAISS、Milvus、Weaviate、Chroma 等向量数据库。

- **适用于企业知识问答场景**  
  可用于智能客服、内部知识库问答、SOP 检索、文档助手与行业 Copilot 系统。

## Installation

### Requirements

- Python 3.10+
- pip / uv / poetry 任一包管理工具
- 推荐使用虚拟环境

### Install from source

```bash
git clone https://github.com/SoloAI/rag-retrieval-augmented-generation.git
cd rag-retrieval-augmented-generation
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -U pip
pip install -r requirements.txt
```

### Optional dependencies

如需启用特定向量数据库、PDF 解析或本地模型支持，可安装扩展依赖：

```bash
pip install faiss-cpu
pip install pymupdf
pip install sentence-transformers
```

### Environment variables

创建 `.env` 文件：

```env
OPENAI_API_KEY=your_api_key
EMBEDDING_MODEL=text-embedding-3-large
CHAT_MODEL=gpt-4o-mini
VECTOR_STORE=faiss
```

## Quick Start

以下示例演示一个最小可用的 **RAG 检索增强生成** 流程：加载文档、建立索引、执行检索并生成答案。

```python
from soloai_rag import RAGPipeline

rag = RAGPipeline(
    embedding_model="text-embedding-3-large",
    chat_model="gpt-4o-mini",
    vector_store="faiss"
)

rag.ingest("./data/knowledge_base/")
answer = rag.ask("RAG 检索增强生成的核心优势是什么？")

print(answer.text)
print(answer.sources)
```

运行：

```bash
python examples/quickstart.py
```

## Usage Examples

### 1. Build a knowledge base from local documents

```python
from soloai_rag import DocumentLoader, IndexBuilder

loader = DocumentLoader()
docs = loader.load("./docs")

builder = IndexBuilder(
    embedding_model="text-embedding-3-large",
    vector_store="faiss"
)
builder.build(docs)
builder.save("./storage/index")
```

### 2. Query with retrieval + reranking

```python
from soloai_rag import Retriever, Reranker, Generator

retriever = Retriever.load("./storage/index")
reranker = Reranker(model="bge-reranker-base")
generator = Generator(chat_model="gpt-4o-mini")

query = "如何提升企业知识库问答准确率？"
candidates = retriever.search(query, top_k=8)
top_docs = reranker.rerank(query, candidates, top_k=3)
result = generator.generate(query, context_docs=top_docs)

print(result.text)
```

### 3. Start an API service

```bash
python -m soloai_rag.api --host 0.0.0.0 --port 8000
```

示例请求：

```bash
curl -X POST "http://127.0.0.1:8000/v1/ask" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "请总结 RAG 系统在企业中的典型应用场景",
    "top_k": 5
  }'
```

示例响应：

```json
{
  "answer": "RAG 系统常用于企业知识问答、客服辅助、培训资料检索、合规审查与文档助手场景。",
  "sources": [
    {
      "id": "doc_001",
      "title": "RAG 企业应用白皮书",
      "score": 0.92
    }
  ]
}
```

### 4. Example configuration

```yaml
app:
  name: soloai-rag
  env: production

models:
  embedding: text-embedding-3-large
  chat: gpt-4o-mini
  reranker: bge-reranker-base

retrieval:
  top_k: 10
  final_k: 4
  chunk_size: 800
  chunk_overlap: 120

vector_store:
  provider: faiss
  path: ./storage/index
```
