## Installation

### Prerequisites

- Python 3.10+
- Docker 24+ _(optional but recommended)_
- NVIDIA GPU + CUDA _(optional for accelerated inference)_
- Git

### Clone the repository

```bash
git clone https://github.com/SoloAI/llm-deployment-optimization.git
cd llm-deployment-optimization
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Optional: install development dependencies

```bash
pip install -r requirements-dev.txt
```

### Optional: run with Docker

```bash
docker build -t soloai-llm-opt .
docker run --rm -p 8080:8080 soloai-llm-opt
```

## Quick Start

Below is a minimal example for launching an optimized LLM inference service:

```bash
python serve.py \
  --model ./models/your-llm \
  --host 0.0.0.0 \
  --port 8080 \
  --batch-size 8 \
  --max-concurrency 64 \
  --dtype float16
```

Test the service:

```bash
curl -X POST "http://localhost:8080/generate" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Explain the benefits of LLM inference optimization.",
    "max_tokens": 128,
    "temperature": 0.7
  }'
```

Example response:

```json
{
  "output": "LLM inference optimization improves latency, throughput, hardware efficiency, and serving cost..."
}
```

## Usage Examples

### 1. Start a tuned inference service

```bash
python serve.py \
  --model ./models/llama \
  --quantization int8 \
  --batch-size 16 \
  --max-input-length 4096 \
  --max-output-length 512 \
  --gpu-memory-utilization 0.9
```

### 2. Use a configuration file

Create a config file:

```yaml
model: ./models/llama
host: 0.0.0.0
port: 8080
quantization: int4
batch_size: 16
max_concurrency: 128
max_input_length: 8192
max_output_length: 1024
gpu_memory_utilization: 0.92
enable_metrics: true
```

Run with config:

```bash
python serve.py --config config/deployment.yaml
```

### 3. Python API example

```python
from client import LLMClient

client = LLMClient(base_url="http://localhost:8080")

response = client.generate(
    prompt="Summarize best practices for LLM deployment optimization.",
    max_tokens=200,
    temperature=0.5,
)

print(response["output"])
```

### 4. Kubernetes deployment example

```bash
kubectl apply -f deploy/k8s/
kubectl get pods
kubectl get svc
```

### 5. Performance benchmarking

```bash
python benchmark.py \
  --endpoint http://localhost:8080/generate \
  --requests 1000 \
  --concurrency 50 \
  --input-tokens 512 \
  --output-tokens 128
```
