## 项目简介

**SoloAI 大模型微调数据服务** 是一个面向大语言模型（LLM）训练与微调场景的数据工程项目，提供从原始数据采集、格式标准化、质量控制到训练集导出的完整能力。  
该项目适用于指令微调（SFT）、偏好对齐、行业知识问答、客服语料、RAG 数据预处理等多种 AI 数据服务场景。

---

## 核心特性

- **多源数据接入**：支持 CSV、JSON、JSONL、Excel、数据库导出及 API 数据汇聚
- **数据清洗与标准化**：内置去重、脱敏、分段、字段映射、格式校验等处理能力
- **面向微调的数据结构输出**：支持 SFT、QA、ChatML、指令-回答对等主流训练格式
- **质量评估与审核流程**：支持规则校验、人工抽检、标签一致性检查与数据评分
- **可扩展的数据处理流水线**：便于按行业场景扩展医疗、金融、电商、教育等领域模板
- **工程化交付**：支持版本化管理、批处理、自动导出、可追踪数据交付记录


```bash
  --format sft
```

## 使用示例

### 1. Python API：加载并清洗原始数据

```python
from soloai.pipeline import DataPipeline
from soloai.transforms import Deduplicate, NormalizeFields, PrivacyMask

pipeline = DataPipeline([
    NormalizeFields(field_map={"question": "input", "answer": "output"}),
    Deduplicate(key_fields=["input"]),
    PrivacyMask(fields=["output"])
])

dataset = pipeline.load("data/raw/source.json")
cleaned = pipeline.run(dataset)
pipeline.save(cleaned, "data/processed/cleaned.jsonl")
```

### 2. 导出为对话微调格式

```python
from soloai.exporters import ChatMLExporter

exporter = ChatMLExporter(
    input_field="input",
    output_field="output"
)

exporter.export(
    source_path="data/processed/cleaned.jsonl",
    target_path="data/processed/chatml.jsonl"
)
```

### 3. 运行数据质量检查

```bash
python scripts/quality_check.py \
  --input data/processed/chatml.jsonl \
  --report reports/quality-report.json
```

### 4. 行业数据处理模板

```bash
python scripts/prepare_dataset.py \
  --input data/raw/customer_service.xlsx \
  --output data/processed/customer_service_sft.jsonl \
  --format sft \
  --template customer-support
```

## 最佳实践

- 在微调前统一字段命名与数据格式，降低训练脚本适配成本
- 对敏感数据执行脱敏与合规审查，确保数据安全
- 为每批数据保留版本号、来源与审核记录，便于追踪
- 在导出训练集前进行重复率、长度分布、标签一致性检查
