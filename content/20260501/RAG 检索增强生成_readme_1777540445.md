SoloRAG is a high-performance, modular Retrieval-Augmented Generation (RAG) framework engineered by SoloAI to seamlessly bridge enterprise knowledge bases with large language models. It delivers production-ready, context-aware AI responses through optimized vector retrieval, intelligent chunking, and deterministic orchestration pipelines.

## 🚀 Key Features
- **Hybrid Retrieval Engine**: Combines dense vector embeddings, sparse BM25, and cross-encoder reranking for precision recall across structured and unstructured data.
- **Adaptive Semantic Chunking**: Context-aware document splitting with metadata preservation, token-boundary alignment, and recursive summarization.
- **Production-Grade Orchestration**: Async streaming, circuit breaking, fallback routing, and built-in observability (OpenTelemetry, Prometheus metrics).
- **Multi-Provider Agnostic**: Plug-and-play connectors for OpenAI, Anthropic, vLLM, Ollama, Qdrant, Weaviate, Milvus, and Pinecone.
- **Evaluation & Guardrails**: Integrated RAGAS scoring, hallucination detection, PII redaction, and deterministic output formatting.

## ⚡ Quick Start
Initialize a complete RAG pipeline in under 10 lines:
```python
from solorag import RAGPipeline, VectorStore, LLMConnector

# Configure core components
retriever = VectorStore(provider="qdrant", collection="enterprise_docs")
generator = LLMConnector(provider="openai", model="gpt-4o")
rag = RAGPipeline(retriever=retriever, generator=generator)

# Ingest & query
rag.ingest(["./data/policies.pdf", "./data/technical_specs.md"])
response = rag.query("What are the compliance requirements for data retention?")
print(response.answer)
```

## 🛠️ Usage Examples

### Hybrid Search with Custom Reranking
```python
from solorag.config import RetrievalConfig

config = RetrievalConfig(
    strategy="hybrid",
    top_k=8,
    rerank_model="BAAI/bge-reranker-v2-m3",
    chunk_size=1024,
    chunk_overlap=128,
    enable_metadata_filtering=True
)

rag.configure(config)
result = rag.query("How to configure TLS 1.3 for internal API gateways?")
print(f"Sources: {result.source_documents}")
```

### Streaming & Automated Evaluation
```python
# Real-time token streaming
for token in rag.stream_query("Generate a deployment checklist for Kubernetes"):
    print(token.delta, end="", flush=True)

# Run RAGAS evaluation pipeline
from solorag.eval import RAGASRunner
scores = RAGASRunner.evaluate(
    dataset="test_queries.jsonl",
    metrics=["faithfulness", "answer_relevancy", "context_precision"]
)
scores.export_report("eval_results.md")
```
