A high-performance, enterprise-grade data pipeline for large language model (LLM) fine-tuning. SoloAI's open-source toolkit streamlines ingestion, semantic cleaning, format conversion, and quality validation to accelerate domain adaptation and instruction alignment.

---

## ✨ Key Features

- **Multi-Source Ingestion & Schema Validation**: Seamlessly parse JSONL, CSV, Parquet, and Hugging Face Datasets with strict type/schema enforcement.
- **LLM-Aware Cleaning & Deduplication**: MinHash + semantic embedding filtering to remove near-duplicates, boilerplate, and low-signal text.
- **Instruction Format Converter**: One-click transformation to Alpaca, ShareGPT, OpenAI ChatML, and custom Jinja2 templates.
- **Automated Quality Scoring**: Built-in toxicity, bias, and coherence filters powered by lightweight reference models.
- **Distributed Processing Engine**: Native Ray/Spark backend for billion-token dataset preparation with auto-scaling and fault tolerance.
- **Training Pipeline Ready**: Direct export to `datasets`, `trl`, `unsloth`, and `vLLM` with optimized tokenization and packing.

## 📦 Installation

```bash
# Install via PyPI
pip install soloai-ft-data

# Or install from source
git clone https://github.com/SoloAI/fine-tuning-data-service.git
cd fine-tuning-data-service
pip install -e ".[distributed]"
```

> 💡 Requires Python 3.9+. For GPU-accelerated semantic deduplication, ensure `torch` and `sentence-transformers` are installed.

## 🚀 Quick Start

```python
from soloai_ft_data import DataPipeline

# Initialize pipeline with source and target format
pipeline = DataPipeline(
    source="raw_domain_data.jsonl",
    target_format="chatml",
    dedup_method="semantic",
    quality_threshold=0.85
)

# Process and push to Hugging Face Hub
dataset = pipeline.process()
dataset.push_to_hub("your-org/ft-ready-dataset")
print(f"✅ Processed {len(dataset)} high-quality samples.")
```

## 📖 Usage Examples

### 🔹 Custom Quality Filters & Distributed Execution
```python
from soloai_ft_data import DataPipeline, QualityFilter, ExportConfig

pipeline = DataPipeline(
    sources=["docs/", "logs.csv", "hf://datasets/example/raw"],
    schema="instruction_tuning"
)

# Chain domain-specific filters
pipeline.add_filter(QualityFilter.toxicity_threshold(0.1))
pipeline.add_filter(QualityFilter.semantic_dedup(threshold=0.95, model="all-MiniLM-L6-v2"))
pipeline.add_filter(QualityFilter.min_token_length(128))

# Run with Ray backend for large-scale processing
processed = pipeline.run(workers="auto", engine="ray")

# Export in multiple formats simultaneously
processed.export(
    config=ExportConfig(
        parquet="output/ft_dataset.parquet",
        jsonl="output/ft_dataset.jsonl",
        hf_hub="your-org/ft-ready-dataset"
    )
)
```

### 🔹 CLI Workflow
```bash
soloai-ft process \
  --input ./raw_data/ \
  --output ./cleaned/ \
  --format alpaca \
  --dedup minhash \
  --workers auto \
  --quality-threshold 0.8
```
