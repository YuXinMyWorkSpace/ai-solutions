## 📖 Overview
A production-grade, scalable data pipeline engineered specifically for Large Language Model (LLM) fine-tuning workflows. SoloAI automates raw corpus ingestion, semantic cleaning, instruction alignment, and quality evaluation to deliver high-fidelity training datasets for SFT, DPO, and RLHF pipelines.

## ✨ Core Features
- **Automated Data Hygiene**: MinHash/SimHash deduplication, PII redaction, and toxicity filtering at scale.
- **Multi-Format Conversion**: Seamless transformation between CSV, JSON, Parquet, JSONL, and Hugging Face `datasets` formats.
- **Instruction-Tuning Alignment**: Built-in template engine for zero-shot/few-shot SFT, preference pairs (DPO), and reward modeling.
- **Quality Scoring Engine**: Perplexity estimation, lexical diversity metrics, and LLM-as-a-Judge validation for dataset curation.
- **Distributed Processing**: Native Ray & Dask integration for horizontal scaling across multi-node clusters.
- **Domain-Specific Presets**: Pre-configured pipelines for Finance, Healthcare, Legal, and Code generation with compliance-ready filters.

## ⚡ Quick Start
```python
from soloai.ft_data import Pipeline, DatasetLoader, Exporter

# 1. Initialize pipeline with SFT configuration
pipeline = Pipeline(config="sft_base.yaml")

# 2. Load raw corpus (supports local paths, S3, or HF Hub)
raw_data = DatasetLoader.from_jsonl("raw_corpus.jsonl")

# 3. Run cleaning, formatting & quality scoring
processed = pipeline.run(raw_data)

# 4. Export to training-ready format
Exporter.to_hf_dataset(processed, repo="soloai/sft-dataset-v1", push_to_hub=True)
```

## 📚 Usage Examples

### 1. DPO Preference Pair Generation
```python
from soloai.ft_data import DPOProcessor

dpo = DPOProcessor(
    reward_model="reward-v2",
    threshold=0.82,
    strategy="contrastive_sampling"
)
preference_pairs = dpo.generate(raw_data, max_pairs=50000)
preference_pairs.save_parquet("dpo_train.parquet")
```

### 2. Domain-Specific Template Injection
```python
from soloai.ft_data import TemplateEngine

engine = TemplateEngine(domain="finance", compliance_level="strict")
formatted = engine.apply(
    raw_data,
    template="instruction-response",
    inject_context=True,
    validate_schema=True
)
formatted.to_jsonl("finance_sft_ready.jsonl")
```

### 3. Integration with Custom Quality Filters
```python
from soloai.ft_data import QualityFilter

filter_chain = QualityFilter([
    "length_range(200, 4000)",
    "language_detect(zh, en, threshold=0.95)",
    "llm_judge_score(min=4.5, model='qwen2.5-7b')"
])
high_quality = filter_chain.run(processed)
```
