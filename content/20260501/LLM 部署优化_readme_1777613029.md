## 📦 Installation

**Prerequisites:** Python 3.9+, CUDA 12.1+, NVIDIA GPU with compute capability ≥ 8.0

```bash
# Install via PyPI
pip install soloai-deploy-optimizer

# Or install from source (recommended for latest features)
git clone https://github.com/SoloAI/llm-deploy-optimizer.git
cd llm-deploy-optimizer
pip install -e ".[all]"
```

**Docker (Production Ready):**
```bash
docker pull ghcr.io/soloai/llm-deploy-optimizer:latest
docker run --gpus all -p 8000:8000 soloai/llm-deploy-optimizer serve --model meta-llama/Llama-3-8B-Instruct
```

## 🚀 Quick Start

Deploy a quantized model with optimized batching in under 10 lines of code:

```python
from soloai_deploy import LLMEngine, EngineConfig

config = EngineConfig(
    model_id="meta-llama/Llama-3-8B-Instruct",
    backend="vllm",
    quantization="fp8",
    tensor_parallel_size=2,
    max_model_len=8192,
    gpu_memory_utilization=0.85
)

engine = LLMEngine(config)
engine.start()

# Synchronous generation
response = engine.generate("Explain the trade-offs between KV-cache paging and static allocation.")
print(response.outputs[0].text)

# Graceful shutdown
engine.stop()
```

## 📖 Usage Examples

### 📐 Declarative Configuration (`deploy.yaml`)
```yaml
deployment:
  backend: tensorrt-llm
  model: mistralai/Mistral-7B-v0.1
  tensor_parallel_size: 4
  max_batch_size: 256
  kv_cache_dtype: auto

scaling:
  min_replicas: 1
  max_replicas: 12
  target_qps: 75
  cooldown_seconds: 30

monitoring:
  enable_tracing: true
  metrics_port: 9090
  log_level: INFO
  export_format: otel
```

### ⚡ CLI Profiling & Benchmarking
```bash
# Run automated throughput/latency profiling
soloai-optimize profile --config deploy.yaml --warmup 50 --iterations 500

# Export optimization report (PDF + JSON)
soloai-optimize report --format json --output ./benchmark_results.json

# Launch optimized serving instance
soloai-optimize serve --config deploy.yaml --port 8080
```

### 🔄 Programmatic Auto-Scaling Hook
```python
from soloai_deploy.scaling import AutoScaler

scaler = AutoScaler(
    engine=engine,
    min_nodes=1,
    max_nodes=6,
    qps_threshold=120,
    scale_up_delay=15,
    scale_down_delay=120
)

scaler.monitor()  # Runs in background thread, adjusts replicas dynamically
```
