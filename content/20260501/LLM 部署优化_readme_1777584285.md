## ✨ Features
- **Hardware-Agnostic Inference**: Unified runtime for NVIDIA CUDA, AMD ROCm, Intel CPU, and custom NPU backends.
- **Advanced Quantization**: Native support for AWQ, GPTQ, and FP8 with zero-copy weight loading and calibration pipelines.
- **High-Throughput Serving**: Continuous batching, speculative decoding, and paged attention for sub-100ms latency at scale.
- **KV-Cache Optimization**: Automatic fragmentation prevention, sliding window caching, and dynamic offloading to CPU/NVMe.
- **Production-Ready Deployment**: One-click Docker, Helm charts, and Kubernetes autoscaling templates with OpenAI-compatible API endpoints.
- **Built-in Observability**: Prometheus metrics, Grafana dashboards, and distributed tracing for latency, throughput, and memory profiling.

## 📦 Installation

### Prerequisites
- Python 3.9+
- CUDA 11.8+ or ROCm 5.4+ (for GPU acceleration)

### Install via PyPI
```bash
pip install solollm-deploy
```

### Install from Source
```bash
git clone https://github.com/SoloAI/solollm-deploy.git
cd solollm-deploy
pip install -e ".[gpu]"
```

### Docker Quick Deploy
```bash
docker run --gpus all -p 8000:8000 \
  -e MODEL_ID="meta-llama/Llama-3-8B-Instruct" \
  soloai/solollm-deploy:latest
```

## 🚀 Quick Start

Initialize a production-ready LLM server in under 10 lines of Python:

```python
from solollm_deploy import LLMEngine, ServingConfig

config = ServingConfig(
    model_id="meta-llama/Llama-3-8B-Instruct",
    quantization="awq_int4",
    max_batch_size=64,
    kv_cache_ratio=0.8,
    device="auto"
)

engine = LLMEngine(config)
engine.serve(host="0.0.0.0", port=8000)
```

Test the endpoint:
```bash
curl http://localhost:8000/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{"model": "llama-3-8b", "messages": [{"role": "user", "content": "Explain LLM deployment optimization in 3 bullet points."}]}'
```

## 🛠️ Usage Examples

### 1. Custom Quantization Pipeline
```python
from solollm_deploy.quant import Quantizer

quantizer = Quantizer(model_path="./llama-3-8b", method="gptq_4bit")
quantizer.calibrate(dataset="wikitext-2", batch_size=8)
quantizer.export(output_dir="./quantized_model")
```

### 2. Dynamic Batching with Priority Routing
```python
from solollm_deploy.scheduler import PriorityBatcher

batcher = PriorityBatcher(
    max_tokens_per_step=4096,
    priority_rules={"premium": 0.9, "standard": 0.5}
)
batcher.enqueue(prompt="High-priority request...", tier="premium")
batcher.enqueue(prompt="Standard request...", tier="standard")
```

### 3. Kubernetes Autoscaling Integration
```yaml
# k8s/deployment.yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: solollm-deploy
spec:
  template:
    spec:
      containers:
      - name: engine
        image: soloai/solollm-deploy:latest
        env:
        - name: AUTO_SCALE_METRIC
          value: "request_latency_ms"
        - name: TARGET_LATENCY
          value: "150"
```
