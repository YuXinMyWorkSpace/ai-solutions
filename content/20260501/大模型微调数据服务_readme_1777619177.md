# 🌐 SoloAI LLM Fine-Tuning Data Service (大模型微调数据服务)
[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/llm-finetune-data-service/ci.yml?branch=main&label=build&logo=github)](https://github.com/SoloAI/llm-finetune-data-service/actions) [![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](LICENSE)

> **SoloAI LLM Fine-Tuning Data Service** is an enterprise-grade, open-source toolkit for generating, curating, and formatting high-quality instruction datasets tailored for large language model fine-tuning. It automates data pipeline orchestration, quality assurance, and multi-framework export to accelerate domain-specific model alignment.

---

## ✨ Key Features
- **🔄 Automated Data Synthesis & Augmentation**: LLM-driven prompt generation with rule-based filtering, back-translation, and synthetic instruction expansion.
- **📊 Multi-Format Export Engine**: Native support for `JSONL`, `Parquet`, `Alpaca`, `ShareGPT`, and Hugging Face `datasets` formats with zero-config conversion.
- **🛡️ Quality Assurance Pipeline**: Built-in deduplication, toxicity/PII filtering, instruction alignment scoring, and length normalization.
- **🎯 Domain-Specific Template Engine**: Pre-configured schemas for Finance, Healthcare, Legal, Code, and Customer Support with customizable prompt structures.
- **⚡ Scalable Distributed Processing**: Ray-backed parallel execution for billion-token workloads with memory-efficient chunking and checkpointing.
- **🔌 Seamless Framework Integration**: Drop-in compatibility with Hugging Face `trl`, Axolotl, Unsloth, vLLM, and PyTorch training loops.

## 🚀 Quick Start
Initialize a pipeline, load raw text, apply quality filters, and export to Hugging Face Hub in under 10 lines of code.

```python
from soloai.data import FineTunePipeline

# 1. Configure pipeline
pipeline = FineTunePipeline(
    input_path="raw_corpus.jsonl",
    output_format="sharegpt",
    quality_threshold=0.85,
    language="zh"
)

# 2. Execute curation & export
dataset = pipeline.run()
dataset.save_to_hub("SoloAI/industry-finetune-v1")
```

## 📖 Usage Examples

### 1. Domain-Specific Instruction Generation
```python
from soloai.templates import DomainTemplateEngine

engine = DomainTemplateEngine(domain="finance", language="zh")
prompts = engine.generate(
    seed_data="quarterly_reports.csv",
    template_type="qa_pair",
    n_samples=5000
)
```

### 2. Advanced Quality Filtering & Deduplication
```python
from soloai.qa import QualityFilter

filter = QualityFilter(
    remove_duplicates=True,
    dedup_threshold=0.92,
    toxicity_model="unitary/toxic-bert",
    min_tokens=64,
    max_tokens=4096,
    alignment_scorer="bert-score"
)
cleaned_ds = filter.apply(prompts)
```

### 3. Hugging Face `trl` Integration
```python
from datasets import load_dataset
from trl import SFTTrainer

ds = load_dataset("SoloAI/industry-finetune-v1", split="train")

trainer = SFTTrainer(
    model="meta-llama/Llama-3-8b",
    train_dataset=ds,
    dataset_text_field="text",
    max_seq_length=4096
)
trainer.train()
```
