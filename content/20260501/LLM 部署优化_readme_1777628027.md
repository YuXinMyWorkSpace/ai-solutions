## 📦 Installation

Install via PyPI or pull the pre-built container for isolated deployment:

```bash
# Python Package
pip install soloai-llm-opt

# Docker (Recommended for Production)
docker pull ghcr.io/soloai/llm-optimize:latest
```

**System Requirements:**
- Python 3.9+
- CUDA 11.8+ / ROCm 5.7+ / x86_64 (CPU fallback)
- 16GB+ VRAM (for 7B-13B models), 32GB+ recommended for 30B+

## ⚡ Quick Start

Deploy and optimize a Hugging Face model in under 10 lines of code:

```python
from soloai_llm_opt import LLMEngine, QuantizationConfig

# Initialize engine with auto-hardware detection
engine = LLMEngine(
    model_id="meta-llama/Llama-3-8B-Instruct",
    quantization=QuantizationConfig(precision="int4", method="awq"),
    max_batch_size=128,
    device="auto"
)

# Run optimization pipeline (quantizes, compiles kernels, warms up KV cache)
engine.optimize()

# Inference
response = engine.generate(
    prompt="Explain the difference between continuous batching and static batching.",
    max_new_tokens=256,
    temperature=0.7
)
print(response)
```

## 📖 Usage Examples

### 1️⃣ Launch Production API Server
```bash
soloai-serve \
  --model mistralai/Mistral-7B-Instruct-v0.3 \
  --port 8000 \
  --tensor-parallel-size 2 \
  --enable-metrics
```
Access via standard OpenAI SDK:
```python
import openai
client = openai.OpenAI(base_url="http://localhost:8000/v1", api_key="dummy")
resp = client.chat.completions.create(
    model="mistral-7b",
    messages=[{"role": "user", "content": "Optimize my LLM pipeline."}]
)
```

### 2️⃣ Custom Benchmark & Profiling
```python
from soloai_llm_opt import Profiler

profiler = Profiler(engine)
metrics = profiler.run_benchmark(
    prompt_file="dataset.jsonl",
    concurrency_levels=[16, 32, 64],
    duration_sec=30
)
metrics.export_csv("benchmark_results.csv")
```

### 3️⃣ CPU-Only Deployment (AVX-512 Optimized)
```python
engine = LLMEngine(
    model_id="Qwen/Qwen2.5-7B-Instruct",
    device="cpu",
    backend="llama.cpp",
    threads=16
)
engine.optimize()  # Auto-compiles GGUF & enables vectorized kernels
```
