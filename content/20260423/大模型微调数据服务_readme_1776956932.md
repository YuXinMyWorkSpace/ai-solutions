A production-grade, open-source data service platform engineered to automate the ingestion, cleaning, alignment, and formatting of instruction-tuning datasets for Large Language Models (LLMs). Designed for seamless integration into modern SFT, DPO, and RLHF training pipelines with enterprise-grade scalability and traceability.

---

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| **Automated Curation & Deduplication** | MinHash/SimHash + semantic embedding clustering to eliminate redundant or near-duplicate samples. |
| **Multi-Format Export Engine** | Native support for JSONL, Parquet, Alpaca, ShareGPT, and Hugging Face `datasets` format. |
| **Quality Filtering & Safety Alignment** | Rule-based heuristics + LLM-as-a-Judge scoring for toxicity, hallucination, and instruction-following compliance. |
| **Domain-Specific Template System** | Pre-built schema injectors for Medical, Finance, Legal, Code, and Multilingual instruction tuning. |
| **Distributed Pipeline Execution** | Ray & Dask backend support for processing 100M+ tokens across multi-node clusters. |
| **Dataset Lineage & Versioning** | Built-in DVC integration with SHA-256 hashing, metadata tracking, and reproducible data snapshots. |

## 🚀 Quick Start

```python
from soloai.data import Pipeline, Filter, Formatter, QualityScorer

# 1. Initialize data pipeline
pipeline = Pipeline(source="raw_corpus.jsonl")

# 2. Apply deduplication & length filtering
pipeline.add_step(Filter.deduplicate(method="minhash", threshold=0.85))
pipeline.add_step(Filter.length(min_tokens=120, max_tokens=2048))

# 3. Format to standard SFT schema
pipeline.add_step(Formatter.to_alpaca(
    prompt_key="instruction",
    context_key="input",
    response_key="output"
))

# 4. Execute & export
dataset = pipeline.run()
dataset.export("sft_train_v1.jsonl", format="jsonl")
print(f"✅ Processed {len(dataset)} high-quality instruction pairs.")
```

## 🛠️ Usage Examples

### 1. LLM-as-a-Judge Quality Scoring
```python
from soloai.quality import JudgeScorer

scorer = JudgeScorer(model="meta-llama/Meta-Llama-3-8B-Instruct")
dataset = dataset.map(lambda row: {
    **row,
    "quality_score": scorer.evaluate(row["instruction"], row["output"])
})
dataset = dataset.filter(lambda row: row["quality_score"] >= 0.75)
```

### 2. Domain-Specific Template Injection
```python
from soloai.templates import MedicalTemplate, CodeTemplate

# Inject domain-specific system prompts & formatting rules
pipeline.add_step(MedicalTemplate.inject(
    system_prompt="You are a board-certified clinical assistant...",
    format="qa_pair"
))
```

### 3. Push to Hugging Face Hub
```python
from soloai.data import Dataset

dataset = Dataset.load("sft_train_v1.jsonl")
dataset.push_to_hub(
    repo_id="SoloAI/medical-sft-v1",
    private=True,
    commit_message="Initial SFT dataset release with quality filtering"
)
```
