# SoloAI RAG 检索增强生成
[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/rag-retrieval-augmented-generation/ci.yml?branch=main)](https://github.com/SoloAI/rag-retrieval-augmented-generation/actions)
[![License](https://img.shields.io/github/license/SoloAI/rag-retrieval-augmented-generation)](./LICENSE)

面向生产环境的 **RAG（Retrieval-Augmented Generation，检索增强生成）** 开源项目，帮助开发者快速构建结合私有知识库、向量检索与大语言模型推理的智能问答系统。  
SoloAI RAG 专注于高性能检索、模块化管道设计与易于扩展的 AI 应用集成，适用于企业知识问答、文档助手与垂直领域 Copilot 场景。

---

## Features

- **模块化 RAG Pipeline**：支持文档加载、切分、嵌入、索引、召回、重排与生成的完整链路
- **多数据源接入**：支持 Markdown、PDF、TXT、HTML 与自定义知识库导入
- **高性能向量检索**：兼容主流向量数据库与本地索引方案，便于快速部署
- **可扩展 LLM 集成**：支持 OpenAI 兼容接口及自定义大模型服务
- **检索优化能力**：支持 Top-K 检索、混合检索、上下文拼接与提示词模板配置
- **面向生产部署**：提供 CLI、Python SDK 与 API 集成方式，适合本地开发与云端部署

## Installation

### Requirements

- Python 3.10+
- `pip` or `poetry`
- 可选：向量数据库（如 FAISS、Milvus、Chroma）
- 可选：OpenAI 兼容大模型 API Key

### Install from source

```bash
git clone https://github.com/SoloAI/rag-retrieval-augmented-generation.git
cd rag-retrieval-augmented-generation
pip install -e .
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Environment variables

创建 `.env` 文件：

```bash
cp .env.example .env
```

示例配置：

```env

OPENAI_BASE_URL=https://api.openai.com/v1
EMBEDDING_MODEL=text-embedding-3-large
CHAT_MODEL=gpt-4o-mini
VECTOR_STORE=faiss
```

## Quick Start

下面示例展示如何将本地文档构建为知识库，并基于 RAG 执行问答。

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline(
    embedding_model="text-embedding-3-large",
    chat_model="gpt-4o-mini",
    vector_store="faiss"
)

# 1. 索引文档
pipeline.index(
    source_path="./docs",
    file_types=[".md", ".pdf", ".txt"]
)

# 2. 发起问答
response = pipeline.query("RAG 检索增强生成的核心流程是什么？")

print(response.answer)
print(response.sources)
```

运行：

```bash
python examples/quickstart.py
```

## Usage

### 1. 使用 CLI 构建索引

```bash
soloai-rag index ./knowledge-base \
  --file-types .md,.pdf,.txt \
  --chunk-size 512 \
  --chunk-overlap 64 \
  --vector-store faiss
```

### 2. 使用 CLI 执行问答

```bash
soloai-rag ask "如何提升检索增强生成的召回质量？" \
  --top-k 5 \
  --rerank true
```

### 3. Python API：自定义检索参数

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline(
    embedding_model="bge-large-zh",
    chat_model="gpt-4o-mini",
    vector_store="chroma"
)

pipeline.index(source_path="./data")

result = pipeline.query(
    question="企业内部知识库问答如何减少幻觉？",
    top_k=5,
    rerank=True,
    return_sources=True
)

print("Answer:")
print(result.answer)

print("\nSources:")
for source in result.sources:
    print(f"- {source['title']} ({source['score']:.4f})")
```

### 4. 自定义 Prompt Template

```python
from soloai_rag import RAGPipeline

template = """
你是一个企业知识库助手。
请基于以下上下文回答问题。
如果上下文中没有答案，请明确说明“不确定”。

上下文:
{context}

问题:
{question}
"""

pipeline = RAGPipeline(prompt_template=template)
response = pipeline.query("系统支持哪些向量数据库？")
print(response.answer)
```

### 5. API 服务模式

```bash
soloai-rag serve --host 0.0.0.0 --port 8080
```

示例请求：

```bash
curl -X POST http://localhost:8080/query \
  -H "Content-Type: application/json" \
  -d '{
    "question": "什么是 RAG 检索增强生成？",
    "top_k": 3
  }'
```

## Why SoloAI RAG?

SoloAI RAG 旨在为开发者和企业团队提供一个高可扩展、高可维护的 **检索增强生成框架**。  
通过将 **文档理解、语义检索、上下文构建与大模型生成** 有机结合，本项目可帮助你快速实现：

- 企业知识库问答
- 文档搜索与摘要
- FAQ 自动化助手
- 多文档上下文生成
- AI Copilot 与智能客服系统

对于关注 **RAG architecture、vector search、LLM grounding、retrieval pipeline、hallucination reduction** 的开发者与研究者，本项目提供了清晰的工程化实践基础。
