# SoloAI RAG 检索增强生成
[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/rag-project/ci.yml?branch=main)](https://github.com/SoloAI/rag-project/actions)
[![License](https://img.shields.io/github/license/SoloAI/rag-project)](./LICENSE)

一个面向生产环境的 **RAG（Retrieval-Augmented Generation，检索增强生成）** 开源项目，用于构建高质量、可扩展、可观测的知识问答与语义检索系统。  
本项目帮助开发者将 **向量检索、文档索引、上下文增强、LLM 推理** 整合为统一工作流，加速企业级 AI 应用落地。

---

## 功能特性

- **高性能 RAG 管线**：支持文档切分、向量化、召回、重排与生成式回答的完整链路
- **多数据源接入**：支持 Markdown、PDF、TXT、HTML 等常见知识库文档格式
- **可插拔向量数据库**：便于接入 FAISS、Milvus、Weaviate、Chroma 等向量存储
- **多模型兼容**：支持 OpenAI、开源 Embedding 模型及主流大语言模型
- **可观测与可调优**：支持召回结果分析、Prompt 调优、Chunk 策略优化
- **适用于企业知识问答**：可用于内部文档检索、客服助手、研发知识库、智能问答

---

## 安装

### 环境要求

- Python 3.10+
- pip 或 poetry
- 可选：Docker
- 可选：向量数据库服务（如 Milvus / Weaviate / Chroma）

### 使用 pip 安装

```bash
git clone https://github.com/SoloAI/rag-project.git
cd rag-project
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -U pip
pip install -r requirements.txt
```

### 使用 Poetry 安装

```bash
git clone https://github.com/SoloAI/rag-project.git
cd rag-project
poetry install
poetry shell
```

### 环境变量配置

创建 `.env` 文件：

```env
OPENAI_API_KEY=your_openai_api_key
EMBEDDING_MODEL=text-embedding-3-large
LLM_MODEL=gpt-4o-mini
VECTOR_STORE=chroma
DATA_DIR=./data
```

## 快速开始

以下示例展示如何加载本地知识文档、构建索引并执行一次 RAG 问答。

```python
from soloai_rag import RAGPipeline

rag = RAGPipeline(
    data_dir="./data",
    embedding_model="text-embedding-3-large",
    llm_model="gpt-4o-mini",
    vector_store="chroma"
)

rag.index()

response = rag.query("RAG 检索增强生成的核心流程是什么？")
print(response.answer)
print(response.sources)
```

运行示例：

```bash
python examples/quickstart.py
```

## 使用示例

### 1. 构建知识库索引

将文档放入 `./data` 目录后执行：

```python
from soloai_rag import DocumentIndexer

indexer = DocumentIndexer(
    data_dir="./data",
    chunk_size=512,
    chunk_overlap=64,
    embedding_model="text-embedding-3-large"
)

indexer.build()
```

### 2. 执行语义检索

```python
from soloai_rag import Retriever

retriever = Retriever(vector_store="chroma")
results = retriever.search("如何优化 RAG 的召回质量？", top_k=5)

for item in results:
    print(item.score, item.text)
```

### 3. 基于上下文生成回答

```python
from soloai_rag import RAGPipeline

rag = RAGPipeline(
    vector_store="chroma",
    llm_model="gpt-4o-mini"
)

response = rag.query(
    "请总结知识库中关于向量检索优化的最佳实践",
    top_k=5
)

print("Answer:\n", response.answer)
print("\nSources:")
for source in response.sources:
    print("-", source)
```

### 4. 使用命令行启动

```bash
python -m soloai_rag.cli index --data ./data
python -m soloai_rag.cli query "什么是检索增强生成？"
```
