# SoloAI RAG 检索增强生成
[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/rag-retrieval-generation/ci.yml?branch=main)](https://github.com/SoloAI/rag-retrieval-generation/actions)
[![License](https://img.shields.io/github/license/SoloAI/rag-retrieval-generation)](./LICENSE)

**SoloAI RAG 检索增强生成** 是一个面向生产环境的开源项目，用于构建高质量、可扩展、可观测的 **Retrieval-Augmented Generation (RAG)** 系统。  
它帮助开发者将文档检索、向量索引、重排序和大语言模型生成能力整合为可落地的 AI 知识问答与企业知识库解决方案。

---

## 简介

RAG（检索增强生成）通过将外部知识检索与大语言模型生成结合，显著提升回答的准确性、可解释性和时效性。  
本项目专注于 **文档摄取、向量检索、语义召回、上下文增强生成、评估与部署**，适用于企业知识库、智能客服、内部搜索、问答助手和行业 Copilot 等场景。

---

## 功能特性

- **多源文档接入**：支持 Markdown、PDF、TXT、HTML、JSON 等常见知识文档格式。
- **向量化与索引构建**：集成 Embedding 流程与向量数据库，支持高效语义检索。
- **混合检索与重排序**：支持关键词检索、向量检索及 reranker 重排序，提升召回质量。
- **上下文增强生成**：将相关上下文注入 LLM Prompt，减少幻觉并提高答案可靠性。
- **可扩展架构**：模块化设计，可快速替换 LLM、Embedding 模型、向量数据库与检索策略。
- **生产级集成能力**：适配 API 服务、日志监控、评估流程和 CI/CD 自动化部署。

## 快速开始

以下示例展示如何导入文档、构建索引并执行 RAG 问答。

```python
from soloai_rag.loader import DocumentLoader
from soloai_rag.indexer import VectorIndexer
from soloai_rag.retriever import Retriever
from soloai_rag.generator import RagGenerator

# 1. 加载文档
docs = DocumentLoader("./data").load()

# 2. 构建向量索引
indexer = VectorIndexer(
    embedding_model="text-embedding-3-large",
    vector_db="faiss",
    index_dir="./index"
)
indexer.build(docs)

# 3. 创建检索器
retriever = Retriever(index_dir="./index", top_k=5)

# 4. 创建生成器
generator = RagGenerator(chat_model="gpt-4o-mini")

# 5. 执行问答
query = "RAG 检索增强生成的核心流程是什么？"
contexts = retriever.search(query)
answer = generator.generate(query=query, contexts=contexts)

print(answer)
```

运行示例：

```bash
python examples/quickstart.py
```

## 使用示例

### 1. 构建企业知识库问答系统

```python
from soloai_rag.pipeline import RagPipeline

pipeline = RagPipeline(
    embedding_model="text-embedding-3-large",
    chat_model="gpt-4o-mini",
    vector_db="faiss",
    top_k=5
)

pipeline.ingest("./knowledge_base")
response = pipeline.ask("如何部署内部知识库问答机器人？")

print(response.answer)
print(response.sources)
```

### 2. 启用混合检索

```python
from soloai_rag.retriever import HybridRetriever

retriever = HybridRetriever(
    index_dir="./index",
    top_k=8,
    use_bm25=True,
    use_vector=True,
    rerank=True
)

results = retriever.search("如何提升 RAG 检索召回率？")
for item in results:
    print(item.score, item.text[:120])
```

### 3. 启动 API 服务

```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

示例请求：

```bash
curl -X POST "http://localhost:8000/api/v1/query" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "RAG 系统如何减少大模型幻觉？",
    "top_k": 5
  }'
```

示例响应：

```json
{
  "query": "RAG 系统如何减少大模型幻觉？",
  "answer": "RAG 通过先检索相关知识片段，再将可信上下文注入到模型生成环节，从而降低纯参数生成导致的幻觉风险。",
  "sources": [
    {
      "id": "doc_001",
      "title": "rag-overview.md",
      "score": 0.92
    }
  ]
}
```

### 4. 自定义 Prompt 模板

```python
from soloai_rag.generator import RagGenerator

generator = RagGenerator(
    chat_model="gpt-4o-mini",
    prompt_template="""
你是一个企业知识助手。
请严格依据提供的上下文回答问题。
如果上下文中没有答案，请明确说明“未在知识库中找到相关信息”。

问题:
{query}

上下文:
{contexts}

回答:
"""
)
```
