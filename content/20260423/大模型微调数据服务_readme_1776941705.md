# SoloAI 大模型微调数据服务

[![Build Status](https://img.shields.io/badge/build-passing-brightgreen)](#)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](./LICENSE)

**SoloAI 大模型微调数据服务** is an open-source solution focused on **LLM fine-tuning data preparation, cleaning, structuring, and delivery**. It helps teams standardize high-quality training datasets for supervised fine-tuning, instruction tuning, domain adaptation, and enterprise AI deployment.

## Overview

面向企业与开发团队，SoloAI 提供一套可扩展的 **大模型微调数据服务框架**，用于构建、清洗、标注、转换与验证高质量训练数据。项目旨在提升数据生产效率、保证数据一致性，并优化面向大语言模型的训练数据交付流程。

## Features

- **多源数据接入**：支持文本、对话、知识库、FAQ、工单、文档等多种数据来源
- **数据清洗与标准化**：自动去重、脱敏、格式归一、异常过滤
- **指令微调数据构建**：支持 SFT / Instruction Tuning / QA / Chat 格式转换
- **数据质量校验**：提供 schema 校验、字段完整性检查、样本一致性检测
- **可扩展数据管线**：模块化设计，便于接入企业内部系统与自定义规则
- **面向交付的导出能力**：支持 JSON / JSONL / CSV 等主流训练数据格式

## Quick Start

以下示例展示如何将原始问答数据转换为适合大模型监督微调的标准 JSONL 格式。

### 1. Prepare input data

`data/raw/sample.json`:

```json
[
  {
    "question": "什么是向量数据库？",
    "answer": "向量数据库用于存储和检索高维向量数据，常用于语义搜索与 RAG 场景。"
  },
  {
    "question": "微调和提示工程有什么区别？",
    "answer": "微调通过训练修改模型参数，提示工程则通过设计输入来提升输出效果。"
  }
]
```

### 2. Run data transformation

```bash
python scripts/prepare_sft.py \
  --input data/raw/sample.json \
  --output data/processed/train.jsonl \
  --format sft
```

### 3. Example output

`data/processed/train.jsonl`

```json
{"instruction":"回答用户问题","input":"什么是向量数据库？","output":"向量数据库用于存储和检索高维向量数据，常用于语义搜索与 RAG 场景。"}
{"instruction":"回答用户问题","input":"微调和提示工程有什么区别？","output":"微调通过训练修改模型参数，提示工程则通过设计输入来提升输出效果。"}
```

## Usage Examples

### Data cleaning

对原始语料执行去重、脱敏和空值清理：

```bash
python scripts/clean_data.py \
  --input data/raw/customer_service.jsonl \
  --output data/clean/cleaned.jsonl \
  --dedup \
  --mask-pii \
  --drop-empty
```

### Convert to chat fine-tuning format

将问答数据转换为对话式训练格式：

```bash
python scripts/convert_chat_format.py \
  --input data/clean/cleaned.jsonl \
  --output data/processed/chat_train.jsonl
```

示例输出：

```json
{
  "messages": [
    {"role": "system", "content": "你是一个专业的行业助手。"},
    {"role": "user", "content": "如何建立企业知识库问答系统？"},
    {"role": "assistant", "content": "可以从文档清洗、分块、向量化、召回和生成五个环节搭建。"}
  ]
}
```

### Validate dataset quality

执行数据字段与结构校验：

```bash
python scripts/validate_dataset.py \
  --input data/processed/chat_train.jsonl \
  --schema schemas/chat.schema.json
```

### Python API example

```python
from soloai.pipeline import DataPipeline
from soloai.transforms import CleanText, ConvertToSFT, ValidateSchema

pipeline = DataPipeline([
    CleanText(dedup=True, mask_pii=True),
    ConvertToSFT(instruction="请根据输入内容生成准确回答"),
    ValidateSchema("schemas/sft.schema.json")
])

pipeline.run(
    input_path="data/raw/source.json",
    output_path="data/processed/train.jsonl"
)
```
