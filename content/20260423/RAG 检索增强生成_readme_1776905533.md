# SoloAI RAG 检索增强生成  
[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/rag-retrieval-generation/ci.yml?branch=main)](https://github.com/SoloAI/rag-retrieval-generation/actions)

> SoloAI RAG 检索增强生成帮助开发者快速接入文档检索、向量索引、上下文注入与大模型推理流程，提升回答准确率与可解释性。

---

## 概述

**SoloAI RAG 检索增强生成** 是一个专注于 **RAG 检索增强生成系统开发** 的开源项目，适用于企业知识库问答、智能客服、内部文档搜索、Agent 工具链集成和私有化大模型应用。  
项目提供从 **文档摄取（ingestion）**、**文本切分（chunking）**、**向量化（embedding）**、**向量检索（vector retrieval）** 到 **大模型生成（LLM generation）** 的完整工作流。

---

## 特性

- **模块化 RAG Pipeline**：支持文档导入、切分、索引、召回、重排、生成的完整检索增强生成链路。
- **多后端支持**：可集成主流向量数据库、Embedding 模型与 LLM 提供商，便于灵活部署。
- **高质量检索能力**：支持 Top-K 检索、语义搜索、混合检索与上下文增强，提升生成答案准确性。
- **面向生产环境**：内置配置化管理、日志监控、可扩展接口，适合企业级落地。
- **开发者友好**：提供清晰 API、CLI 示例与快速启动方式，降低 RAG 系统开发门槛。
- **易于二次开发**：支持自定义检索器、Prompt 模板、重排器与生成策略。

## 快速开始

下面示例展示如何构建一个基础的 **RAG 检索增强生成** 流程：导入文档、建立索引并基于问题生成答案。

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline(
    embedding_model="text-embedding-3-large",
    llm_model="gpt-4o-mini",
    vector_store="qdrant"
)

# 1. 导入知识文档
pipeline.ingest([
    "docs/architecture.md",
    "docs/api_reference.md",
    "docs/faq.md"
])

# 2. 执行检索增强问答
response = pipeline.query(
    question="RAG 系统中如何提升检索召回率？",
    top_k=5
)

print(response.answer)
print(response.sources)
```

## 使用示例

### 1. 文档摄取与索引构建

```python
from soloai_rag import DocumentIngestor

ingestor = DocumentIngestor(
    chunk_size=500,
    chunk_overlap=50,
    embedding_model="text-embedding-3-large"
)

ingestor.add_file("data/knowledge_base.pdf")
ingestor.add_file("data/product_docs.md")
ingestor.build_index(vector_store="faiss")
```

### 2. 语义检索

```python
from soloai_rag import Retriever

retriever = Retriever(vector_store="faiss")
results = retriever.search("如何设计企业内部知识库问答系统？", top_k=3)

for doc in results:
    print(doc.score, doc.text[:200])
```

### 3. 检索增强生成问答

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline.from_config("configs/rag.yaml")

result = pipeline.query(
    question="什么是检索增强生成，适合哪些业务场景？",
    top_k=4,
    return_sources=True
)

print("Answer:\n", result.answer)
print("Sources:\n", result.sources)
```

### 4. CLI 使用方式

```bash
python -m soloai_rag ingest --input ./docs --store qdrant
python -m soloai_rag query "RAG 如何减少大模型幻觉？"
```
