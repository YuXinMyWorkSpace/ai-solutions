## ✨ Key Features
- 🔹 **Dynamic KV-Cache & PagedAttention**: Eliminates memory fragmentation and enables 3–5× higher concurrent request handling.
- 🔹 **Multi-Tier Quantization**: Seamless INT4/INT8/FP8 calibration with automated sensitivity analysis and fallback routing.
- 🔹 **Auto-Scaling Parallelism**: Intelligent tensor/pipeline parallelism distribution with topology-aware GPU scheduling.
- 🔹 **Hardware-Agnostic Runtime**: Unified execution layer supporting NVIDIA CUDA, AMD ROCm, Intel Gaudi, and Apple Metal.
- 🔹 **Real-Time Telemetry & Auto-Tuning**: Prometheus/Grafana integration with adaptive batch sizing and speculative decoding.
- 🔹 **OpenAI-Compatible API Gateway**: Drop-in replacement for existing LLM services with built-in rate limiting and request routing.

## 📦 Installation

### Prerequisites
- Python `3.9+`
- CUDA `12.1+` (or ROCm `5.7+` for AMD GPUs)
- `pip` or `conda` package manager

### Install via PyPI
```bash
pip install soloai-llm-deploy-opt
```

### Install from Source (Recommended for Custom Builds)
```bash
git clone https://github.com/SoloAI/llm-deploy-optimizer.git
cd llm-deploy-optimizer
pip install -e ".[gpu,monitoring]"
```

## 🚀 Quick Start

Initialize the engine with automatic hardware profiling and deploy an optimized inference server in under 10 lines of code:

```python
from soloai.deploy import LLMEngine, QuantizationConfig

# Auto-detect hardware & apply optimal precision
config = QuantizationConfig(
    precision="auto",      # Dynamically selects FP8/INT8/INT4
    enable_kv_cache=True,
    speculative_decoding=True
)

engine = LLMEngine(
    model="meta-llama/Llama-3-8B-Instruct",
    config=config
)

# Launch production-ready OpenAI-compatible endpoint
engine.serve(host="0.0.0.0", port=8080, workers=4)
```

## 📖 Usage Examples

### Distributed Inference with Custom Routing
```python
from soloai.deploy import DistributedConfig, RequestRouter

dist_cfg = DistributedConfig(
    tensor_parallel_size=4,
    pipeline_parallel_size=2,
    gpu_ids=[0, 1, 2, 3],
    interconnect="nccl"  # or "mpi" for multi-node
)

router = RequestRouter(engine, dist_cfg)
router.optimize_routing(strategy="dynamic_batching", max_batch_size=256)

# Execute optimized generation
response = router.generate(
    prompt="Explain the transformer architecture in 3 sentences.",
    max_tokens=128,
    temperature=0.7,
    top_p=0.9
)
print(response.text)
```

### CLI Monitoring & Auto-Tuning
```bash
# Start real-time telemetry dashboard
soloai-monitor --port 9090 --metrics gpu_util,token_throughput,kv_cache_hit_rate

# Run automated hyperparameter sweep for your hardware
soloai-optimize --model ./custom-model --target-latency 50ms --profile
```
