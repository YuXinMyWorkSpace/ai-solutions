# SoloAI 大模型微调数据服务

[![Build Status](https://img.shields.io/badge/build-passing-brightgreen.svg)](./)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](./LICENSE)

面向大模型微调（LLM Fine-tuning）的开源数据服务平台，提供从数据接入、清洗、标注、转换到训练集交付的一站式能力。  
专为指令微调、SFT、偏好数据构建、RAG 数据准备等场景设计，帮助团队高效管理高质量训练数据资产。

---

## 简介

**SoloAI 大模型微调数据服务** 是一个聚焦于 **LLM fine-tuning data pipeline** 的开源项目，用于构建标准化、可追溯、可扩展的数据处理流程。  
它支持多源数据接入、样本清洗与去重、指令数据格式化、数据集版本管理，以及面向主流训练框架的数据导出。

---

## 核心特性

- **多源数据接入**  
  支持本地文件、对象存储、数据库、API 等多种数据源，适配 JSON / JSONL / CSV / Parquet 等常见格式。

- **面向微调的数据处理流水线**  
  提供清洗、去重、分片、过滤、脱敏、质量检测等标准化处理能力，适用于 SFT、DPO、RLHF、RAG 等任务。

- **标准化数据模式**  
  内置指令微调、对话、多轮问答、偏好对（chosen / rejected）等数据 schema，便于对接训练框架。

- **数据集版本管理与可追溯性**  
  支持数据版本、转换记录、审计日志和元数据跟踪，方便复现实验与模型迭代。

- **高性能导出能力**  
  可将数据导出为 Hugging Face Datasets、JSONL、Parquet 等格式，适配 LoRA、SFTTrainer、DeepSpeed 等训练流程。

- **开放式集成**  
  提供 CLI、REST API 和 Python SDK，便于集成到数据平台、MLOps、AIOps 或企业内部 AI 工作流。

## 快速开始

下面示例演示如何将原始问答数据转换为适用于大模型指令微调的 JSONL 数据集。

### 1. 准备原始数据

```json
[
  {
    "question": "什么是大模型微调？",
    "answer": "大模型微调是指在预训练模型基础上，使用特定领域数据继续训练，以提升模型在特定任务上的表现。"
  }
]
```

### 2. 使用 CLI 生成微调数据集

```bash
soloai-data build \
  --input ./data/raw_qa.json \
  --output ./data/train.jsonl \
  --task sft \
  --schema instruction
```

### 3. 生成结果示例

```json
{"instruction":"什么是大模型微调？","input":"","output":"大模型微调是指在预训练模型基础上，使用特定领域数据继续训练，以提升模型在特定任务上的表现。"}
```

## 使用示例

### 使用 Python SDK 处理 SFT 数据

```python
from soloai_data import DatasetBuilder

builder = DatasetBuilder(task="sft", schema="instruction")

builder.load("./data/raw_qa.json")
builder.clean(
    remove_duplicates=True,
    min_length=5,
    pii_mask=True
)
builder.transform(
    input_key="question",
    output_key="answer"
)
builder.export("./data/train.jsonl", format="jsonl")
```

### 构建偏好学习数据集（DPO / RLHF）

```python
from soloai_data import PreferenceDatasetBuilder

builder = PreferenceDatasetBuilder(schema="preference_pair")

builder.load("./data/preferences.jsonl")
builder.validate(required_fields=["prompt", "chosen", "rejected"])
builder.export("./data/dpo_train.parquet", format="parquet")
```

### 通过 REST API 创建数据任务

```bash
curl -X POST http://localhost:8080/api/v1/jobs \
  -H "Content-Type: application/json" \
  -d '{
    "task": "sft",
    "schema": "conversation",
    "input_uri": "s3://llm-data/raw/chat-data.jsonl",
    "output_uri": "s3://llm-data/processed/sft-chat-data.jsonl",
    "options": {
      "deduplicate": true,
      "pii_mask": true,
      "language": "zh"
    }
  }'
```

### 与 Hugging Face Datasets 集成

```python
from datasets import load_dataset

dataset = load_dataset("json", data_files="./data/train.jsonl")
print(dataset["train"][0])
```

### 导出适配对话格式数据

```bash
soloai-data convert \
  --input ./data/chat_raw.jsonl \
  --output ./data/chat_sft.jsonl \
  --schema conversation \
  --format jsonl
```
