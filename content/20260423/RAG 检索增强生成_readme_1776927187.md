# SoloAI RAG 检索增强生成
[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/rag-retrieval-augmented-generation/ci.yml?branch=main)](https://github.com/SoloAI/rag-retrieval-augmented-generation/actions)
[![License](https://img.shields.io/github/license/SoloAI/rag-retrieval-augmented-generation)](./LICENSE)

一个面向生产环境的 **RAG（Retrieval-Augmented Generation，检索增强生成）** 开源项目，帮助开发者快速构建基于知识库的智能问答、企业搜索与 AI 助手系统。  
项目聚焦于高质量检索、上下文增强生成与可扩展架构，适用于 LLM 应用、知识管理和智能客服场景。

---

## 概述

**SoloAI RAG 检索增强生成** 提供从文档解析、向量索引、语义检索到大语言模型生成的一体化能力，帮助团队高效搭建可信、可追溯、可扩展的 AI 知识应用。  
该项目适合用于企业知识库问答、文档检索、内部助手、FAQ 自动化和多源数据增强生成等场景。

## 特性

- **端到端 RAG Pipeline**：覆盖数据导入、切分、Embedding、向量检索、重排序与生成全流程
- **多数据源接入**：支持 Markdown、PDF、TXT、HTML、数据库及自定义知识源
- **高性能语义检索**：支持向量数据库集成与混合检索策略，提升召回率与准确性
- **可插拔 LLM / Embedding Provider**：兼容 OpenAI、本地模型、Hugging Face 及私有化部署方案
- **上下文增强与引用追踪**：生成结果可附带来源片段，提高结果可信度与可解释性
- **面向生产环境设计**：支持配置化部署、API 集成、日志监控与扩展开发

## 安装

### 环境要求

- Python 3.10+
- pip 或 poetry
- 可选：Docker / Docker Compose
- 可选：向量数据库（如 FAISS、Chroma、Milvus、Weaviate）

### 使用 pip 安装

```bash
git clone https://github.com/SoloAI/rag-retrieval-augmented-generation.git
cd rag-retrieval-augmented-generation
pip install -r requirements.txt
```

### 使用 poetry 安装

```bash
git clone https://github.com/SoloAI/rag-retrieval-augmented-generation.git
cd rag-retrieval-augmented-generation
poetry install
```

### 环境变量配置

创建 `.env` 文件：

```bash
cp .env.example .env
```

示例配置：

```env
OPENAI_API_KEY=your_api_key
EMBEDDING_MODEL=text-embedding-3-large
LLM_MODEL=gpt-4o-mini
VECTOR_STORE=chroma
DATA_DIR=./data
```

## 快速开始

下面的示例展示了如何导入文档、建立索引并执行一次 RAG 问答。

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline(
    data_dir="./data",
    embedding_model="text-embedding-3-large",
    llm_model="gpt-4o-mini",
    vector_store="chroma"
)

# 1. 构建知识库索引
pipeline.ingest()

# 2. 提问
response = pipeline.query("RAG 检索增强生成的核心流程是什么？")

print(response.answer)
print(response.sources)
```

运行：

```bash
python examples/quickstart.py
```

## 使用示例

### 1. 从本地文档构建知识库

```python
from soloai_rag import DocumentLoader, IndexBuilder

loader = DocumentLoader("./documents")
docs = loader.load()

index_builder = IndexBuilder(
    embedding_model="text-embedding-3-large",
    vector_store="faiss"
)

index_builder.build(docs)
```

### 2. 执行语义检索

```python
from soloai_rag import Retriever

retriever = Retriever(top_k=5)
results = retriever.search("如何提升 RAG 系统召回率？")

for item in results:
    print(item.score, item.content)
```

### 3. 使用检索结果增强生成

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline()
response = pipeline.query(
    question="企业知识库问答系统如何实现可追溯回答？",
    return_sources=True
)

print("Answer:", response.answer)
print("Sources:")
for source in response.sources:
    print("-", source)
```

### 4. 启动 API 服务

```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

示例请求：

```bash
curl -X POST "http://localhost:8000/query" \
  -H "Content-Type: application/json" \
  -d '{
    "question": "什么是 RAG？",
    "top_k": 3
  }'
```

示例响应：

```json
{
  "answer": "RAG（Retrieval-Augmented Generation）是一种将外部知识检索与大语言模型生成结合的方法，用于提升回答准确性与时效性。",
  "sources": [
    {
      "id": "doc_001",
      "title": "RAG Introduction",
      "score": 0.92
    }
  ]
}
```
