# 🤖 SoloAI LLM Fine-Tuning Data Service
![Build Status](https://img.shields.io/badge/build-passing-brightgreen?logo=github&style=flat-square) ![License](https://img.shields.io/badge/License-Apache%202.0-blue?style=flat-square) ![Python](https://img.shields.io/badge/Python-3.9%2B-3776AB?style=flat-square&logo=python) ![PyPI](https://img.shields.io/pypi/v/soloai-llm-data?color=007EC6&style=flat-square)

A production-grade, open-source data pipeline and service framework engineered specifically for **Large Language Model (LLM) fine-tuning**. It automates end-to-end dataset curation, quality evaluation, instruction alignment, and version-controlled export to accelerate domain-specific model adaptation and RLHF preparation.

---

## ✨ Key Features
- **Multi-Source Ingestion & Parsing**: Seamlessly ingest raw text, PDFs, JSON, CSV, and web-scraped corpora with automatic schema normalization.
- **Intelligent Cleaning & Deduplication**: Fuzzy matching, MinHash LSH, and semantic similarity filtering to eliminate redundancy and low-quality noise.
- **Instruction Tuning Generation**: Rule-based + LLM-assisted prompt templating for SFT, DPO, and preference dataset formatting.
- **Automated Quality Evaluation**: Built-in metrics (coherence, instruction compliance, toxicity, factuality) with configurable scoring thresholds.
- **Reproducible Dataset Versioning**: Immutable snapshots, lineage tracking, and one-click export to JSONL, Parquet, or Hugging Face Datasets.
- **Distributed Processing Engine**: Native Ray/Dask integration for horizontal scaling across CPU/GPU clusters.

## 🚀 Quick Start

Initialize a fine-tuning data pipeline, process raw inputs, and export a production-ready dataset in under 10 lines of code:

```python
from soloai.data import FineTunePipeline, DataConfig

# 1. Configure pipeline
config = DataConfig(
    input_dir="./raw_documents/",
    output_dir="./datasets/sft_v1/",
    task_type="instruction_tuning",
    enable_dedup=True,
    quality_threshold=0.82
)

# 2. Execute pipeline
pipeline = FineTunePipeline(config)
dataset = pipeline.run()

print(f"✅ Successfully curated {len(dataset)} high-quality samples.")
```

## 📖 Usage Examples

### 🔹 Custom Filtering & Semantic Routing
```python
from soloai.data.filters import RegexFilter, SemanticClusterFilter

# Keep only compliance & finance-related content
pipeline.add_filter(RegexFilter(patterns=["regulation", "audit", "compliance"]))

# Route samples into domain clusters using embedding similarity
pipeline.add_filter(SemanticClusterFilter(
    model="BAAI/bge-large-en-v1.5",
    n_clusters=5,
    min_cluster_size=50
))
```

### 🔹 LLM-Based Quality Scoring & Alignment
```python
from soloai.data.evaluators import LLMQualityEvaluator

evaluator = LLMQualityEvaluator(
    model="Qwen/Qwen2.5-7B-Instruct",
    criteria=["instruction_following", "factual_accuracy", "safety"]
)

# Score and filter low-quality samples
scored_dataset = evaluator.evaluate(dataset)
clean_dataset = scored_dataset.filter(lambda x: x["quality_score"] >= 0.85)
```

### 🔹 Export to Hugging Face Hub
```python
from soloai.data.exporters import HFExporter

exporter = HFExporter(dataset)
exporter.push_to_hub(
    repo_id="SoloAI/enterprise-finance-sft-v1",
    private=True,
    commit_message="Initial SFT dataset release"
)
```
