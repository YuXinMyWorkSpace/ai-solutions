# SoloAI-RAG: Enterprise Retrieval-Augmented Generation Framework
[![Build Status](https://img.shields.io/github/actions/workflow/status/soloai/soloai-rag/ci.yml?branch=main&label=Build&logo=github)](https://github.com/soloai/soloai-rag/actions)
[![License](https://img.shields.io/github/license/soloai/soloai-rag?color=blue)](LICENSE)

**SoloAI-RAG** is a production-grade, modular framework for Retrieval-Augmented Generation (RAG) that unifies high-precision vector retrieval, context-aware prompt engineering, and LLM orchestration. Designed for enterprise scalability, it enables developers to build accurate, domain-specific AI applications with minimal boilerplate and deterministic evaluation pipelines.

---

## 🚀 Key Features
- **Hybrid Retrieval Engine**: Combines dense vector search (e.g., FAISS, Qdrant) with sparse BM25 lexical matching for optimal recall across structured and unstructured corpora.
- **Adaptive Chunking & Metadata Filtering**: Dynamic document segmentation with semantic boundary detection and strict metadata scoping to reduce hallucination and context bloat.
- **Multi-LLM Routing & Fallback**: Deterministic model selection based on query complexity, latency SLAs, and cost constraints, with automatic fallback chains.
- **Built-in RAG Evaluation**: Native integration with RAGAS and LangSmith-compatible tracing for measuring context precision, answer faithfulness, and retrieval latency.
- **Async & Streaming API**: Fully asynchronous pipeline execution with SSE/WS streaming support for real-time token delivery and enterprise-grade concurrency.

## ⚡ Quick Start

Initialize a RAG pipeline, ingest documents, and query in under 10 lines:

```python
from soloai_rag import RAGPipeline, VectorRetriever, LLMRouter

# 1. Configure pipeline
pipeline = RAGPipeline(
    retriever=VectorRetriever(
        embedding_model="text-embedding-3-large",
        vector_db="qdrant",
        top_k=5
    ),
    generator=LLMRouter(primary="gpt-4o", fallback="claude-3.5-sonnet"),
    chunk_size=512,
    chunk_overlap=64
)

# 2. Ingest local/remote documents
pipeline.ingest(["./docs/", "https://api.example.com/specs.yaml"])

# 3. Query with streaming
for token in pipeline.stream_query("Explain the OAuth2 PKCE flow with Python examples."):
    print(token, end="", flush=True)
```

## 📖 Usage Examples

### Advanced Metadata Filtering & Hybrid Search
```python
from soloai_rag import HybridRetriever, QueryRewriter

retriever = HybridRetriever(
    dense_weight=0.7,
    sparse_weight=0.3,
    metadata_filters={"department": "engineering", "version": ">=2.1"}
)

# Query rewriting improves retrieval precision for ambiguous prompts
rewritten_query = QueryRewriter().expand("How do we handle rate limits?")
results = retriever.search(rewritten_query, k=4)
```

### Evaluation & Tracing Pipeline
```python
from soloai_rag.eval import RAGEvaluator, DatasetLoader

evaluator = RAGEvaluator(metrics=["faithfulness", "context_precision", "answer_relevancy"])
dataset = DatasetLoader.from_csv("test_queries.csv")

results = evaluator.run(pipeline, dataset)
results.to_json("eval_report.json")
print(f"Average Faithfulness: {results.mean('faithfulness'):.3f}")
```
