# 🤖 SoloAI LLM Fine-Tuning Data Service
[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/llm-finetune-data-service/ci.yml?branch=main&label=build)](https://github.com/SoloAI/llm-finetune-data-service/actions)
[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

A production-grade, open-source data engineering pipeline built for large language model (LLM) fine-tuning. SoloAI's service automates dataset curation, quality filtering, instruction formatting, and synthetic data generation to accelerate Supervised Fine-Tuning (SFT), Direct Preference Optimization (DPO), and domain-specific model adaptation.

---

## ✨ Key Features
- **Automated Data Curation & Deduplication**: MinHash/SimHash-based near-duplicate removal with configurable similarity thresholds.
- **Multi-Format Instruction Engine**: Native support for Alpaca, ShareGPT, ChatML, and custom JSONL/Parquet templates.
- **Quality Scoring & Filtering**: Hybrid heuristic + LLM-as-a-judge pipeline for toxicity, factual consistency, and instruction-following alignment.
- **Synthetic Data Generation**: Controllable prompt templates, temperature scheduling, and self-correction loops for high-diversity corpus expansion.
- **Seamless Framework Integration**: Drop-in compatibility with Hugging Face `datasets`, PyTorch `DataLoader`, and vLLM/TRL training stacks.

## 🚀 Quick Start
Initialize a standard SFT data pipeline and process raw corpora into model-ready instruction pairs:

```python
from soloai.data import FineTunePipeline

# Configure pipeline for instruction tuning
pipeline = FineTunePipeline(
    task_type="sft",
    max_seq_length=4096,
    quality_threshold=0.85,
    dedup_strategy="minhash"
)

# Execute end-to-end processing
dataset = pipeline.run(
    input_path="./raw_data/",
    output_format="sharegpt",
    save_path="./processed_data/train.jsonl"
)

print(f"✅ Processed {len(dataset)} high-quality instruction pairs.")
```

## 📖 Usage Examples

### 1. Custom Template & Field Mapping
Map heterogeneous JSON/CSV fields to standardized instruction-tuning formats:

```python
pipeline.configure_template(
    system_prompt="You are an expert financial analyst.",
    user_template="{context}\n\nQuestion: {question}",
    assistant_template="{answer}",
    field_mapping={"context": "doc_text", "question": "query", "answer": "response"}
)
```

### 2. LLM-as-a-Judge Quality Filtering
Apply automated scoring to filter low-quality or hallucinated generations:

```python
from soloai.quality import JudgeFilter

judge = JudgeFilter(
    model_name="Qwen/Qwen2.5-7B-Instruct",
    criteria=["factual_accuracy", "instruction_following", "coherence"],
    threshold=4.0  # out of 5
)

cleaned_ds = judge.apply(dataset)
print(f"📉 Filtered out {(1 - len(cleaned_ds)/len(dataset))*100:.1f}% of low-quality samples.")
```

### 3. Export to Hugging Face & PyTorch
```python
# Push to HF Hub
dataset.push_to_hub("your-org/finance-sft-v1")

# PyTorch DataLoader integration
from torch.utils.data import DataLoader
loader = DataLoader(dataset, batch_size=16, collate_fn=pipeline.collator)
```
