**SoloAI RAG Framework** is a production-grade, modular Retrieval-Augmented Generation engine engineered to seamlessly connect enterprise knowledge bases with state-of-the-art LLMs. It delivers low-latency, context-grounded responses through advanced document chunking, hybrid vector search, and automated hallucination guardrails.

---

## ✨ Key Features
- 🔍 **Hybrid Retrieval Pipeline**: Combines dense embeddings, sparse BM25, and metadata filtering for precision recall across structured and unstructured data.
- 📐 **Adaptive Chunking & Enrichment**: Semantic-aware splitting with automatic metadata extraction (tables, citations, code blocks) and cross-document relationship mapping.
- 🛡️ **Self-Correction & Guardrails**: Built-in factuality scoring, citation tracing, and LLM-as-a-judge evaluation to mitigate hallucinations in production environments.
- 🔌 **Plug-and-Play Architecture**: Fully compatible with LangChain, LlamaIndex, and custom vector stores (Qdrant, Milvus, Weaviate, Pinecone).
- 📊 **Enterprise Observability**: Native tracing, latency profiling, and RAGAS/DeepEval integration for continuous pipeline optimization and A/B testing.

## 🚀 Quick Start

Initialize a production-ready RAG pipeline in under 10 lines:

```python
from soloai_rag import RAGPipeline, EmbeddingModel, VectorStore
from soloai_rag.retrievers import HybridRetriever

# 1. Configure components
pipeline = RAGPipeline(
    embedding=EmbeddingModel("text-embedding-3-large"),
    retriever=HybridRetriever(top_k=5, alpha=0.7),
    vector_store=VectorStore("qdrant", collection="enterprise_kb")
)

# 2. Ingest documents (PDFs, Markdown, CSV, HTML)
pipeline.ingest(["./docs/product_manual.pdf", "./docs/api_reference.md"])

# 3. Query with grounded generation
response = pipeline.query("What are the recommended retry policies for webhook failures?")
print(response.answer)
print("📎 Sources:", [doc.metadata["source"] for doc in response.sources])
```

## 🛠️ Advanced Usage Examples

### Custom Chunking & Metadata Routing
```python
from soloai_rag.processors import SemanticChunker, MetadataRouter

chunker = SemanticChunker(
    max_tokens=512,
    overlap_ratio=0.15,
    preserve_tables=True
)

router = MetadataRouter(rules={
    "legal": {"min_similarity": 0.85, "fallback": False},
    "technical": {"max_tokens": 1024, "rerank": True}
})

pipeline.set_chunker(chunker).set_router(router)
```

### Streaming & Evaluation Integration
```python
from soloai_rag.eval import RAGASMetrics

# Stream response tokens
for token in pipeline.query_stream("Explain the OAuth2 authorization code flow"):
    print(token, end="", flush=True)

# Run automated evaluation suite
metrics = RAGASMetrics.evaluate(
    dataset="test_queries.jsonl",
    pipeline=pipeline,
    metrics=["faithfulness", "answer_relevancy", "context_precision"]
)
print(metrics.summary())
```
