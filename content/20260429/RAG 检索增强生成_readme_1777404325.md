## 📖 Overview
SoloRAG is a high-performance, modular Retrieval-Augmented Generation (RAG) framework engineered by SoloAI to bridge the gap between static enterprise knowledge bases and dynamic LLM inference. It delivers context-aware, citation-backed responses with enterprise-grade scalability, low latency, and built-in hallucination mitigation.

## ✨ Key Features
- **Hybrid Retrieval Engine**: Combines dense vector embeddings with sparse BM25 keyword search for optimal recall & precision.
- **Adaptive Context Windowing**: Dynamic chunking with semantic boundary detection and overlap optimization to prevent context truncation.
- **Multi-Format Ingestion Pipeline**: Native support for PDFs, Markdown, HTML, SQL/NoSQL databases, and API endpoints with metadata extraction.
- **Real-Time Evaluation & Guardrails**: Integrated RAGAS scoring, citation verification, and configurable hallucination filters.
- **Production-Ready Caching**: Semantic cache layer with TTL management to reduce LLM API costs and accelerate repeated queries.
- **Zero-Trust Security**: Role-based access control (RBAC), PII redaction, and encrypted vector storage out-of-the-box.

## ⚡ Quick Start
```python
from solorag import RAGPipeline, VectorStore, LLMConnector

# 1. Initialize core components
store = VectorStore(provider="chromadb", collection="enterprise_docs")
llm = LLMConnector(model="gpt-4o", temperature=0.1, api_key="YOUR_KEY")

# 2. Build pipeline
pipeline = RAGPipeline(retriever=store, generator=llm, top_k=5)

# 3. Ingest & index documents
pipeline.ingest(["./data/architecture.pdf", "./data/api_specs.md"])

# 4. Query with citations
response = pipeline.query("How does the fallback routing mechanism handle latency spikes?")
print(f"Answer: {response.text}")
print(f"Sources: {[doc.metadata['source'] for doc in response.citations]}")
```

## 🛠️ Usage Examples

### 🔍 Advanced: Hybrid Search & Streaming
```python
from solorag.retrieval import HybridRetriever
from solorag.chunking import SemanticChunker

# Configure chunking & retrieval weights
chunker = SemanticChunker(max_tokens=512, overlap_ratio=0.15)
retriever = HybridRetriever(dense_weight=0.65, sparse_weight=0.35)

pipeline = RAGPipeline(chunker=chunker, retriever=retriever, generator=llm)

# Stream response with real-time token generation
for token in pipeline.stream_query("Explain the consensus protocol used in cluster sync."):
    print(token, end="", flush=True)
```

### 📊 Evaluation & Quality Assurance
```python
from solorag.eval import RAGEvaluator, HallucinationDetector

evaluator = RAGEvaluator(dataset="./benchmarks/qa_pairs.jsonl")
metrics = evaluator.run(pipeline)

print(f"Faithfulness: {metrics.faithfulness:.3f}")
print(f"Answer Relevance: {metrics.answer_relevance:.3f}")
print(f"Context Recall: {metrics.context_recall:.3f}")
```
