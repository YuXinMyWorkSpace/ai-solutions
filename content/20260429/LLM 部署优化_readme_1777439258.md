# 🚀 SoloAI / LLM-Deploy-Optimizer

[![Build Status](https://img.shields.io/badge/build-passing-brightgreen)](https://github.com/SoloAI/LLM-Deploy-Optimizer/actions)
[![License](https://img.shields.io/badge/License-Apache_2.0-blue.svg)](LICENSE)

A high-performance inference engine and deployment toolkit engineered to accelerate Large Language Model (LLM) serving across heterogeneous hardware. SoloAI’s framework reduces latency by up to 60% while optimizing GPU memory utilization, throughput, and operational costs.

## ✨ Key Features
- **Dynamic KV-Cache & PagedAttention**: Zero-waste memory allocation with automatic cache eviction, page-level management, and fragmentation prevention.
- **Hardware-Agnostic Compilation**: Seamless execution on CUDA, ROCm, Intel Gaudi, and CPU with auto-fallback routing and kernel fusion.
- **Advanced Quantization Pipeline**: Native support for AWQ, GPTQ, FP8, and INT4 with calibration-free deployment and accuracy preservation.
- **Speculative Decoding & Adaptive Batching**: Real-time request scheduling with draft-model acceleration, dynamic micro-batching, and priority queuing.
- **Distributed Inference Scaling**: Tensor/Pipeline parallelism with automatic sharding, NCCL/RDMA-optimized communication, and fault-tolerant recovery.
- **Production-Ready Telemetry**: OpenTelemetry integration, Prometheus metrics, Grafana dashboards, and Kubernetes HPA hooks for auto-scaling.

## 📦 Installation

### Prerequisites
- Python 3.9+
- CUDA 12.1+ or ROCm 5.7+ (GPU deployment)
- `pip` or `conda`

### Quick Install
```bash
pip install soloai-llm-deploy
```

### Build from Source
```bash
git clone https://github.com/SoloAI/LLM-Deploy-Optimizer.git
cd LLM-Deploy-Optimizer
pip install -e ".[dev]"
```

## ⚡ Quick Start
```python
from soloai.deploy import OptimizedPipeline

# Auto-optimizes compute graph, memory layout, and batching strategy
pipeline = OptimizedPipeline(
    model_id="meta-llama/Llama-3-8B-Instruct",
    device="cuda:0",
    quantization="awq",
    max_batch_size=128,
    enable_speculative_decoding=True
)

response = pipeline.generate("Explain transformer architecture concisely.")
print(response)
```

## 🛠️ Usage Examples

### Multi-GPU Distributed Serving
```python
from soloai.deploy import DistributedEngine

engine = DistributedEngine(
    model_id="Qwen/Qwen2-72B-Instruct",
    tp_size=4,          # Tensor Parallelism
    pp_size=2,          # Pipeline Parallelism
    backend="nccl",
    enable_offload=True
)
engine.serve(host="0.0.0.0", port=8000, workers=8)
```

### Custom Quantization & Memory Optimization
```python
from soloai.quant import Quantizer
from soloai.memory import KVCacheManager

quantizer = Quantizer(bits=4, algorithm="awq", group_size=128)
cache_mgr = KVCacheManager(max_tokens=32768, eviction_policy="lru")

# Compile model with quantization + cache strategy
optimized_model = quantizer.compile(model, cache_manager=cache_mgr)
optimized_model.save_pretrained("./llm-optimized-v1")
```

### FastAPI Production Integration
```python
from fastapi import FastAPI
from soloai.serve import AsyncInferenceServer

app = FastAPI()
server = AsyncInferenceServer(model_path="./llm-optimized-v1", max_concurrency=256)

@app.post("/v1/generate")
async def generate(request: dict):
    return await server.stream_generate(prompt=request["prompt"])
```
