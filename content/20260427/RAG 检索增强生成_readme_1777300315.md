# SoloAI RAG 检索增强生成
[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/rag-retrieval-augmented-generation/ci.yml?branch=main)](https://github.com/SoloAI/rag-retrieval-augmented-generation/actions)

面向企业与开发者的 **RAG（Retrieval-Augmented Generation，检索增强生成）** 开源项目，帮助大模型基于私有知识库实现更准确、可追溯、可扩展的问答与内容生成。  
该项目聚焦中文场景与工程化落地，适用于知识库问答、智能客服、企业搜索、文档助手等 AI 应用。

---

## 概述

**SoloAI RAG 检索增强生成** 提供从文档解析、向量化、索引构建、召回排序到大模型生成的完整工作流。  
项目设计目标是让开发者快速搭建高质量、低幻觉、支持多数据源接入的 RAG 系统。

---

## 核心特性

- **多源文档接入**：支持 Markdown、PDF、TXT、HTML、JSON 等常见知识库格式
- **向量检索与混合检索**：支持语义检索、关键词检索及 Hybrid Search 提升召回质量
- **可插拔式 LLM / Embedding 架构**：兼容主流大模型与向量模型，便于替换和扩展
- **分块、重排与上下文优化**：内置 Chunking、Reranking、Prompt Assembly 等关键链路
- **可追溯答案生成**：返回引用来源与上下文片段，增强答案可信度
- **工程化部署友好**：适合本地开发、容器化部署、API 服务化与企业私有化落地

---

## 技术架构

典型 RAG 流程如下：

1. **数据导入**：加载企业知识文档
2. **文本清洗与切分**：按语义或规则进行 chunk 切分
3. **Embedding 向量化**：将文本块转换为向量表示
4. **索引存储**：写入向量数据库或检索引擎
5. **Query 检索**：根据用户问题召回相关上下文
6. **Rerank 重排**：提升最终上下文质量
7. **LLM 生成**：结合检索结果生成答案
8. **引用返回**：输出答案及来源文档信息

## 快速开始

下面示例展示如何基于本地文档目录构建一个最小可运行的 RAG 问答系统。

```python
from soloai_rag.loader import DocumentLoader
from soloai_rag.index import VectorIndexer
from soloai_rag.retriever import Retriever
from soloai_rag.generator import RAGPipeline

# 1. 加载文档
docs = DocumentLoader(path="./data").load()

# 2. 构建索引
indexer = VectorIndexer(
    embedding_model="text-embedding-3-large",
    vector_store="faiss"
)
indexer.build(docs)

# 3. 创建检索器
retriever = Retriever(index=indexer, top_k=5)

# 4. 构建 RAG Pipeline
pipeline = RAGPipeline(
    retriever=retriever,
    llm_model="gpt-4o-mini"
)

# 5. 提问
response = pipeline.ask("RAG 检索增强生成的核心优势是什么？")

print("Answer:", response.answer)
print("Sources:", response.sources)
```

运行：

```bash
python examples/quickstart.py
```

## 使用示例

### 1. 构建知识库索引

```python
from soloai_rag.loader import DocumentLoader
from soloai_rag.index import VectorIndexer

docs = DocumentLoader(path="./knowledge_base").load()

indexer = VectorIndexer(
    embedding_model="bge-large-zh",
    vector_store="faiss"
)

indexer.build(docs)
indexer.save("./storage/index")
```

### 2. 加载已存在索引并检索

```python
from soloai_rag.index import VectorIndexer
from soloai_rag.retriever import Retriever

index = VectorIndexer.load("./storage/index")
retriever = Retriever(index=index, top_k=3)

results = retriever.search("如何降低大模型幻觉？")

for item in results:
    print(item.text)
    print(item.metadata)
```

### 3. 使用重排模型优化召回

```python
from soloai_rag.retriever import Retriever
from soloai_rag.rerank import Reranker

retriever = Retriever(index=index, top_k=10)
reranker = Reranker(model="bge-reranker-large")

query = "企业知识库问答如何提升准确率？"
candidates = retriever.search(query)
ranked = reranker.rerank(query, candidates)

for doc in ranked[:5]:
    print(doc.text[:200])
```

### 4. 启动 API 服务

```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

示例请求：

  -H "Content-Type: application/json" \

    "top_k": 5
  }'
```

  "sources": [
    {
      "title": "rag_architecture.md",
      "score": 0.93
    }
  ]
}
```
