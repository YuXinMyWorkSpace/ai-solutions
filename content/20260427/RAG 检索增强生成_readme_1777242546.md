# SoloAI RAG 检索增强生成
[![Build Status](https://img.shields.io/badge/build-passing-brightgreen)](#)

SoloAI RAG 检索增强生成 is an open-source framework for building high-performance **Retrieval-Augmented Generation (RAG)** applications. It helps developers connect large language models with private or domain-specific knowledge bases to deliver more accurate, grounded, and explainable AI responses.

## Features

- **Modular RAG Pipeline**: Ingestion, chunking, embedding, indexing, retrieval, reranking, and generation.
- **Multi-Source Knowledge Support**: Load data from Markdown, PDF, TXT, JSON, databases, and APIs.
- **Vector Search Ready**: Works with popular vector databases and embedding providers.
- **Configurable Retrieval Strategies**: Top-k retrieval, hybrid search, metadata filtering, and reranking.
- **Production-Oriented Design**: Environment-based configuration, API-first integration, and scalable architecture.
- **Developer Friendly**: Simple setup, clear interfaces, and extensible components for custom workflows.

## Installation

### Prerequisites

- Python 3.10+
- `pip` or `poetry`
- Access to an LLM API and embedding model
- Optional: vector database such as FAISS, Milvus, Weaviate, or Pinecone

### Install from source

```bash
git clone https://github.com/SoloAI/rag-jisuo-enhanced-generation.git
cd rag-jisuo-enhanced-generation
pip install -r requirements.txt
```

### Optional: Create a virtual environment

```bash
python -m venv .venv
source .venv/bin/activate    # macOS / Linux
.venv\Scripts\activate       # Windows
pip install -r requirements.txt
```

### Environment configuration

Create a `.env` file in the project root:

```env
OPENAI_API_KEY=your_api_key
EMBEDDING_MODEL=text-embedding-3-large
CHAT_MODEL=gpt-4o-mini
VECTOR_STORE=faiss
DATA_DIR=./data
```

## Quick Start

Below is a minimal example showing how to ingest documents, build an index, and run a RAG query.

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline.from_config({
    "embedding_model": "text-embedding-3-large",
    "chat_model": "gpt-4o-mini",
    "vector_store": "faiss",
    "chunk_size": 512,
    "chunk_overlap": 64
})

pipeline.ingest("./data")
pipeline.build_index()

response = pipeline.query("什么是 RAG 检索增强生成？")
print(response.answer)
print(response.sources)
```

## Usage Examples

### 1. Document ingestion

```python
from soloai_rag import DocumentLoader

loader = DocumentLoader()
docs = loader.load("./knowledge_base", file_types=[".md", ".pdf", ".txt"])
print(f"Loaded {len(docs)} documents")
```

### 2. Build vector index

```python
from soloai_rag import IndexBuilder

builder = IndexBuilder(
    embedding_model="text-embedding-3-large",
    vector_store="faiss"
)

builder.create_index("./knowledge_base")
```

### 3. Run retrieval + generation

```python
from soloai_rag import Retriever, Generator

retriever = Retriever(top_k=5, search_type="hybrid")
generator = Generator(chat_model="gpt-4o-mini")

context = retriever.search("如何提高企业知识库问答准确率？")
answer = generator.generate(
    question="如何提高企业知识库问答准确率？",
    context=context
)

print(answer)
```

### 4. API service example

```python
from fastapi import FastAPI
from soloai_rag import RAGPipeline

app = FastAPI()
pipeline = RAGPipeline.load("./artifacts")

@app.get("/query")
def query(q: str):
    result = pipeline.query(q)
    return {
        "question": q,
        "answer": result.answer,
        "sources": result.sources
    }
```

Run the API:

```bash
uvicorn app:app --reload --host 0.0.0.0 --port 8000
```
