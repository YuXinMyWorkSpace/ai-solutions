## Installation

### Prerequisites

- Python 3.10+
- `pip` or `poetry`
- NVIDIA GPU recommended for production inference
- CUDA toolkit compatible with your serving stack
- Linux environment recommended for performance workloads

### Clone the repository

```bash
git clone https://github.com/SoloAI/llm-deployment-optimization.git
cd llm-deployment-optimization
```

### Install with `pip`

```bash
python -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```

### Optional: install development dependencies

```bash
pip install -r requirements-dev.txt
```

### Optional: install with Poetry

```bash
poetry install
poetry shell
```

## Quick Start

Below is a simple example showing how to launch an optimized inference service.

### 1. Start the service

```bash
python -m src.server \
  --model /models/your-llm \
  --host 0.0.0.0 \
  --port 8080 \
  --dtype float16 \
  --max-batch-size 16
```

### 2. Send a test request

```bash
curl -X POST http://localhost:8080/generate \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Explain the benefits of quantization for LLM deployment.",
    "max_tokens": 128,
    "temperature": 0.7
  }'
```

### Example response

```json
{
  "text": "Quantization reduces model memory usage and can improve inference speed, enabling more efficient deployment on limited hardware while lowering serving cost."
}
```

## Usage Examples

## 1. Launch with optimized config

Use a YAML configuration file for reproducible deployment.

```bash
python -m src.server --config configs/production.yaml
```

Example `configs/production.yaml`:

```yaml
model_path: /models/llm
host: 0.0.0.0
port: 8080
dtype: float16
tensor_parallel: 2
max_batch_size: 32
streaming: true
quantization: int8
metrics: true
```

---

## 2. Benchmark inference performance

Measure latency and throughput under different concurrency levels.

```bash
python scripts/benchmark.py \
  --endpoint http://localhost:8080/generate \
  --concurrency 32 \
  --requests 500
```

Example output:

```text
Average latency: 142ms
P95 latency: 228ms
Throughput: 78.4 req/s
GPU memory usage: 14.2GB
```

---

## 3. Python client example

```python
import requests

payload = {
    "prompt": "Summarize best practices for LLM serving in production.",
    "max_tokens": 100,
    "temperature": 0.5
}

response = requests.post("http://localhost:8080/generate", json=payload, timeout=30)
response.raise_for_status()

result = response.json()
print(result["text"])
```

## 4. Streaming inference example

curl -N -X POST http://localhost:8080/generate/stream \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Generate a deployment checklist for a production LLM API.",
    "max_tokens": 150
  }'
