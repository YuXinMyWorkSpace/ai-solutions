## Overview

**SoloAI 大模型微调数据服务** 提供一套标准化、可扩展的数据工程方案，用于支持大语言模型（LLM）微调、SFT、指令数据构建、偏好数据整理、数据质量评估与行业数据交付。  
该项目适用于需要进行**模型训练数据处理、指令微调数据生产、行业语料治理、数据集标准化输出**的团队与企业。

---

## Features

- **多类型微调数据支持**  
  支持指令微调（SFT）、问答对、对话数据、分类数据、摘要数据、RAG 知识片段、偏好对（Preference Data）等多种数据格式。

- **标准化数据清洗与治理**  
  提供去重、脱敏、格式修复、异常过滤、内容归一化、噪声识别等数据预处理能力，提升训练数据可用性。

- **高质量标注与质检流程**  
  支持人工标注、规则校验、多轮审核、抽样验收与一致性评估，保障大模型微调数据质量。

- **灵活的数据导出格式**  
  可输出 JSON / JSONL / CSV 等常见格式，便于接入 Hugging Face、OpenAI 格式、LLaMA Factory、DeepSpeed 等训练流程。

- **行业场景定制化交付**  
  面向金融、医疗、教育、电商、客服、法务等行业提供定制化语料构建与数据集交付方案。

- **适配 AI 训练流水线**  
  可无缝接入数据处理、训练、评测与迭代闭环，支持从原始数据到微调数据集的完整工程链路。

## Quick Start

下面示例展示了一个简单的数据清洗与格式转换流程，将原始对话数据转换为适用于大模型微调的 JSONL 格式。

### 1. 准备原始数据

```json
[
  {
    "instruction": "请总结以下内容",
    "input": "SoloAI 提供大模型微调数据服务，覆盖数据清洗、标注和质检。",
    "output": "SoloAI 提供覆盖清洗、标注和质检的大模型微调数据服务。"
  }
]
```

### 2. 运行数据处理脚本

```bash
python scripts/prepare_sft_data.py \
  --input data/raw/sample.json \
  --output data/processed/train.jsonl
```

### 3. 输出示例

```jsonl
{"instruction":"请总结以下内容","input":"SoloAI 提供大模型微调数据服务，覆盖数据清洗、标注和质检。","output":"SoloAI 提供覆盖清洗、标注和质检的大模型微调数据服务。"}
```

## Usage Examples

以下示例展示项目在实际大模型训练数据生产中的常见使用方式。

### 1. 数据清洗

```bash
python scripts/clean_data.py \
  --input data/raw/corpus.jsonl \
  --output data/clean/cleaned.jsonl \
  --deduplicate \
  --normalize \
  --remove_sensitive
```

适用场景：
- 原始语料去重
- 非结构化文本标准化
- 敏感信息脱敏
- 脏数据过滤

---

### 2. 构建指令微调数据（SFT）

```bash
python scripts/build_sft_dataset.py \
  --input data/clean/cleaned.jsonl \
  --schema alpaca \
  --output data/sft/train.jsonl
```

输出格式示例：

```json
{
  "instruction": "解释什么是监督微调",
  "input": "",
  "output": "监督微调（SFT）是基于高质量标注样本对预训练模型进行任务适配的过程。"
}
```

---

### 3. 偏好数据构建（DPO / RLHF）

```bash
python scripts/build_preference_data.py \
  --input data/raw/preferences.jsonl \
  --output data/preference/dpo_train.jsonl
```

输出示例：

```json
{
  "prompt": "请写一段产品介绍",
  "chosen": "这是一款面向企业的数据服务平台，可提升模型训练效率。",
  "rejected": "这是个东西，还行。"
}
```

---

### 4. 数据质量评估

```bash
python scripts/evaluate_dataset.py \
  --input data/sft/train.jsonl \
  --report reports/data_quality_report.json
```

典型评估指标：
- 重复率
- 平均长度
- 标签一致性
- 无效样本占比
- 格式合法率
- 行业术语覆盖率
