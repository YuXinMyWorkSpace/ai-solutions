# SoloAI RAG 检索增强生成
[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/rag-search/main.yml?branch=main)](https://github.com/SoloAI/rag-search/actions)
[![License](https://img.shields.io/github/license/SoloAI/rag-search)](./LICENSE)

一个面向生产环境的 **RAG（Retrieval-Augmented Generation，检索增强生成）** 开源项目，帮助开发者快速构建基于私有知识库的 AI 问答、企业知识助手与智能检索系统。  
本项目聚焦 **文档解析、向量检索、重排序、上下文增强生成** 等核心能力，适用于企业知识管理、客服自动化、搜索问答与垂直领域 Copilot 场景。

---

## Features

- **端到端 RAG Pipeline**：覆盖文档导入、切分、Embedding、向量索引、召回、重排序与 LLM 生成。
- **多数据源支持**：支持 Markdown、PDF、TXT、HTML 及自定义数据接入。
- **可插拔检索架构**：兼容主流向量数据库与 Embedding/LLM 服务，便于灵活扩展。
- **高质量召回与重排序**：支持 Top-K 检索、语义召回与 Re-ranking，提高回答准确率。
- **面向生产部署**：支持 API 服务化、配置化运行、日志监控与容器化部署。
- **适合企业知识库场景**：可用于内部文档问答、SOP 检索、技术支持知识助手与行业智能客服。

## Installation

### Requirements

- Python 3.10+
- pip / poetry
- 可选：Docker
- 可选：向量数据库（如 FAISS、Milvus、Qdrant、pgvector）
- 可选：LLM / Embedding API Key

### Install from source

```bash
git clone https://github.com/SoloAI/rag-search.git
cd rag-search
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -U pip
pip install -r requirements.txt
```

### Environment configuration

创建 `.env` 文件：

```bash
cp .env.example .env
```

示例配置：

```env
OPENAI_API_KEY=your_openai_api_key
EMBEDDING_MODEL=text-embedding-3-large
CHAT_MODEL=gpt-4o-mini

VECTOR_STORE=faiss
CHUNK_SIZE=500
CHUNK_OVERLAP=50
TOP_K=5
```

### Optional: Docker

```bash
docker build -t soloai-rag .
docker run --env-file .env -p 8000:8000 soloai-rag
```

## Quick Start

以下示例展示如何导入本地文档并执行一次基础的 RAG 问答。

```python
from soloai_rag import RAGPipeline

rag = RAGPipeline(
    embedding_model="text-embedding-3-large",
    chat_model="gpt-4o-mini",
    vector_store="faiss"
)

# 1. 索引文档
rag.ingest("./docs")

# 2. 发起查询
response = rag.query("RAG 检索增强生成的核心流程是什么？")

print(response.answer)
print(response.sources)
```

启动 API 服务：

```bash
python -m app.server
```

默认接口示例：

```bash
curl -X POST "http://localhost:8000/query" \
  -H "Content-Type: application/json" \
  -d '{
    "question": "如何提升企业知识库问答准确率？"
  }'
```

## Usage Examples

### 1. Ingest documents into the knowledge base

```python
from soloai_rag import DocumentIngestor

ingestor = DocumentIngestor(
    chunk_size=500,
    chunk_overlap=50
)

ingestor.load("./knowledge")
ingestor.index()
```

### 2. Query with retrieval + generation

```python
from soloai_rag import RAGPipeline

rag = RAGPipeline()
result = rag.query(
    question="请总结产品部署文档中的关键步骤",
    top_k=5,
    rerank=True
)

print("Answer:", result.answer)
for source in result.sources:
    print(source)
```

### 3. Advanced configuration with custom retriever

```python
from soloai_rag import RAGPipeline, FaissRetriever

retriever = FaissRetriever(index_path="./data/faiss.index")

rag = RAGPipeline(
    retriever=retriever,
    top_k=8,
    enable_rerank=True
)

result = rag.query("如何设计高质量的 RAG 检索策略？")
print(result.answer)
```

### 4. REST API integration

```python
import requests

payload = {
    "question": "什么是检索增强生成？",
    "top_k": 5
}

resp = requests.post("http://localhost:8000/query", json=payload)
print(resp.json())
```
