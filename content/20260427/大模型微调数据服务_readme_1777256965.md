# SoloAI 大模型微调数据服务

[![Build Status](https://img.shields.io/badge/build-passing-brightgreen)](#)  
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

**SoloAI 大模型微调数据服务** is an open-source toolkit and service framework for preparing, validating, transforming, and managing high-quality datasets for large language model fine-tuning. It is designed to help teams standardize data pipelines, improve dataset quality, and accelerate supervised fine-tuning, instruction tuning, and domain adaptation workflows.

---

## Overview

Fine-tuning data quality directly impacts model performance, safety, consistency, and downstream usability. SoloAI provides a practical foundation for **LLM fine-tuning data engineering**, including dataset cleaning, schema normalization, multi-format conversion, quality inspection, and scalable workflow integration.

This project is suitable for teams working on:

- Instruction tuning datasets
- SFT / supervised fine-tuning pipelines
- Domain-specific LLM adaptation
- Chat dataset transformation
- Multi-turn conversation data processing
- Data quality governance for model training

---

## Features

- **Multi-format dataset support**: Process JSON, JSONL, CSV, and common chat fine-tuning formats.
- **Data cleaning and normalization**: Remove noise, deduplicate samples, normalize fields, and standardize schema.
- **Quality validation pipeline**: Detect missing fields, malformed messages, invalid labels, and formatting inconsistencies.
- **Conversation dataset transformation**: Convert raw business data into instruction-response or multi-turn chat training formats.
- **Scalable workflow integration**: Run locally or integrate into CI/CD, data platforms, or model training pipelines.
- **Extensible architecture**: Add custom validators, converters, and preprocessing rules for industry-specific datasets.

## Quick Start

Prepare and validate a fine-tuning dataset in a few commands.

### Example input

```json
{
  "instruction": "用一句话解释什么是微调。",
  "input": "",
  "output": "微调是基于预训练模型使用特定任务数据进行进一步训练，以提升模型在目标场景中的表现。"
}
```

### Validate and transform dataset

```bash
python -m soloai_data_service validate \
  --input data/train.jsonl \
  --schema sft \
  --report reports/validation.json

python -m soloai_data_service transform \
  --input data/train.jsonl \
  --output data/train.normalized.jsonl \
  --target-format chatml
```

### Python API example

```python
from soloai_data_service import DatasetValidator, DatasetTransformer

validator = DatasetValidator(schema="sft")
report = validator.validate_file("data/train.jsonl")

print(report.summary())

transformer = DatasetTransformer(target_format="chatml")
transformer.transform_file(
    input_path="data/train.jsonl",
    output_path="data/train.chatml.jsonl"
)
```

## Usage Examples

## 1. Validate supervised fine-tuning data

Use this command to check whether your dataset matches the expected SFT schema.

```bash
python -m soloai_data_service validate \
  --input data/sft.jsonl \
  --schema sft \
  --strict
```

Expected schema example:

```json
{
  "instruction": "总结以下段落的核心观点",
  "input": "大模型训练依赖高质量数据治理流程。",
  "output": "高质量数据治理是大模型训练效果的关键基础。"
}
```

---

## 2. Convert business records into chat training format

Transform structured records into a multi-turn chat dataset for conversational model fine-tuning.

```bash
python -m soloai_data_service transform \
  --input data/business_records.jsonl \
  --output data/chat_dataset.jsonl \
  --target-format conversations
```

Output example:

```json
{
  "messages": [
    { "role": "system", "content": "你是一个专业的行业助手。" },
    { "role": "user", "content": "请解释什么是数据清洗。" },
    { "role": "assistant", "content": "数据清洗是对原始数据进行纠错、去重、标准化和质量修复的过程。" }
  ]
}
```

---

## 3. Deduplicate and clean noisy records

```bash
python -m soloai_data_service clean \
  --input data/raw.jsonl \
  --output data/clean.jsonl \
  --deduplicate \
  --trim-whitespace \
  --drop-empty-output
```

---

## 4. Run via Python in a pipeline

```python
from soloai_data_service import DataCleaner, DatasetValidator

cleaner = DataCleaner(
    deduplicate=True,
    trim_whitespace=True,
    drop_empty_output=True
)

cleaner.clean_file("data/raw.jsonl", "data/clean.jsonl")

validator = DatasetValidator(schema="chat")
result = validator.validate_file("data/clean.jsonl")

if not result.has_errors:
    print("Dataset is ready for fine-tuning.")
else:
    print(result.errors[:5])
```
