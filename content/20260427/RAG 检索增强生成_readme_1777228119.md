# SoloAI RAG 检索增强生成
[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/rag-retrieval-generation/ci.yml?branch=main)](https://github.com/SoloAI/rag-retrieval-generation/actions)

## 项目简介

**SoloAI RAG 检索增强生成** 是一个专注于高性能知识检索与大模型生成融合的开源框架。它支持多种文档格式接入、向量数据库集成、语义检索、混合检索、Prompt 编排与可扩展 API，帮助团队快速搭建可靠、可控、可落地的 RAG 系统。

---

## 核心特性

- **多源文档接入**：支持 PDF、Markdown、TXT、HTML、Word 等常见知识文档格式
- **检索增强生成链路**：内置文档切分、Embedding、向量索引、召回、重排序与生成流程
- **灵活的向量数据库集成**：可对接 FAISS、Milvus、Weaviate、Chroma 等主流向量存储
- **混合检索能力**：支持关键词检索、语义检索与 Hybrid Search，提升召回准确率
- **模块化与可扩展架构**：便于自定义 LLM、Embedding 模型、Prompt 模板和检索策略
- **面向生产环境**：支持 API 服务化部署、日志监控、配置管理与企业知识库场景落地

## 快速开始

以下示例展示如何导入本地知识库，构建向量索引，并完成一次 RAG 问答。

```python
from soloai_rag.loader import DirectoryLoader
from soloai_rag.pipeline import RAGPipeline

# 1. 加载知识库文档
docs = DirectoryLoader("./knowledge_base").load()

# 2. 初始化 RAG 流水线
rag = RAGPipeline(
    embedding_model="text-embedding-3-large",
    llm_model="gpt-4o-mini",
    vector_store="faiss"
)

# 3. 建立索引
rag.index_documents(docs)

# 4. 发起问答
response = rag.query("请总结该知识库中关于 RAG 架构设计的最佳实践。")

print(response.answer)
print(response.sources)
```

启动 API 服务：

```bash
python -m soloai_rag.api
```

默认服务地址：

```bash
http://127.0.0.1:8000
```

## 使用示例

### 1. 文档索引构建

```python
from soloai_rag.loader import FileLoader
from soloai_rag.pipeline import RAGPipeline

doc = FileLoader("docs/architecture.pdf").load()

rag = RAGPipeline(
    embedding_model="text-embedding-3-large",
    llm_model="gpt-4o-mini",
    vector_store="chroma"
)

rag.index_documents(doc)
```

### 2. 检索 + 生成

```python
result = rag.query(
    question="企业知识库问答系统中，如何提升召回质量？",
    top_k=5,
    rerank=True
)

print("Answer:", result.answer)
print("Sources:", result.sources)
```

### 3. 仅检索，不生成

```python
hits = rag.retrieve(
    query="混合检索与向量检索的区别",
    top_k=3
)

for idx, hit in enumerate(hits, 1):
    print(f"{idx}. score={hit.score}")
    print(hit.content)
    print("-" * 80)
```

### 4. FastAPI 接口调用示例

请求：

```bash
curl -X POST "http://127.0.0.1:8000/query" \
  -H "Content-Type: application/json" \
  -d '{
    "question": "什么是 RAG 检索增强生成？",
    "top_k": 5,
    "rerank": true
  }'
```

返回示例：

```json
{
  "answer": "RAG 检索增强生成是一种将外部知识检索与大语言模型生成能力结合的方法...",
  "sources": [
    {
      "id": "doc_001",
      "title": "rag-introduction.md",
      "score": 0.92
    }
  ]
}
```
