# SoloAI RAG 检索增强生成
[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/rag-retrieval-augmented-generation/ci.yml?branch=main)](https://github.com/SoloAI/rag-retrieval-augmented-generation/actions)

面向生产环境的 **RAG（Retrieval-Augmented Generation，检索增强生成）** 开源项目，帮助开发者快速构建基于私有知识库、向量检索与大语言模型推理的智能问答系统。  
本项目专注于 **高质量检索、可扩展知识接入、低延迟生成与可观测性**，适用于企业知识助手、客服自动化、文档问答与行业 AI 应用。

---

## Overview

**SoloAI RAG 检索增强生成** 提供一套模块化、可扩展的 RAG 技术栈，覆盖文档解析、文本切分、Embedding、向量索引、召回排序、Prompt 编排以及 LLM 推理。  
项目目标是让团队能够以更低成本、更高准确率落地企业级 AI 检索增强生成系统。

**Keywords:** RAG, 检索增强生成, Retrieval-Augmented Generation, 向量数据库, Embedding, LLM, 企业知识库, 文档问答, AI Search, Semantic Search

---

## Features

- **模块化 RAG Pipeline**：覆盖数据导入、预处理、索引构建、语义检索、重排序与生成全流程
- **多数据源支持**：支持 Markdown、PDF、TXT、HTML、知识库文档等多种内容接入
- **向量检索与混合检索**：支持 Embedding 检索、关键词检索及 Hybrid Search 组合策略
- **可插拔模型与存储后端**：便于对接主流 LLM、Embedding 模型与向量数据库
- **面向生产环境优化**：支持缓存、批处理、可观测性、日志追踪与性能调优
- **适用于企业场景**：适配内部知识助手、FAQ 问答、智能客服、研究与分析类应用

## Installation

### Requirements

- Python 3.10+
- pip / virtualenv
- 可选：Docker
- 可选：向量数据库（如 FAISS / Milvus / pgvector / Weaviate）

### Install from source

```bash
git clone https://github.com/SoloAI/rag-retrieval-augmented-generation.git
cd rag-retrieval-augmented-generation

python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate

pip install -U pip
pip install -r requirements.txt
```

### Environment variables

创建 `.env` 文件：

```bash
OPENAI_API_KEY=your_api_key
EMBEDDING_MODEL=text-embedding-3-large
CHAT_MODEL=gpt-4o-mini
VECTOR_DB=faiss
```

### Optional: Docker

```bash
docker build -t soloai-rag .
docker run --env-file .env -p 8000:8000 soloai-rag
```

## Quick Start

下面示例展示如何加载本地文档、构建向量索引并执行一次 RAG 问答：

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline(
    embedding_model="text-embedding-3-large",
    chat_model="gpt-4o-mini",
    vector_store="faiss"
)

pipeline.ingest("./data/knowledge_base")
pipeline.build_index()

response = pipeline.query("RAG 检索增强生成的核心优势是什么？")
print(response.answer)
print(response.sources)
```

示例输出：

```text
answer:
RAG 的核心优势在于将外部知识检索与大语言模型生成结合起来，从而提升回答的准确性、时效性与可解释性。

sources:
- ./data/knowledge_base/rag_intro.md
- ./data/knowledge_base/architecture.pdf
```

## Usage Examples

### 1. Ingest documents

```python
from soloai_rag import DocumentIngestor

ingestor = DocumentIngestor(chunk_size=800, chunk_overlap=100)
docs = ingestor.load("./docs")
chunks = ingestor.chunk(docs)

print(f"Loaded {len(docs)} documents")
print(f"Generated {len(chunks)} chunks")
```

### 2. Build vector index

```python
from soloai_rag import VectorIndexer

indexer = VectorIndexer(
    embedding_model="text-embedding-3-large",
    vector_store="faiss"
)

indexer.add_documents(chunks)
indexer.save("./storage/index")
```

### 3. Run semantic retrieval

```python
results = indexer.search(
    query="企业内部知识库问答如何提升准确率？",
    top_k=5
)

for item in results:
    print(item.score, item.metadata["source"])
```

### 4. Generate answer with retrieved context

```python
from soloai_rag import AnswerGenerator

generator = AnswerGenerator(chat_model="gpt-4o-mini")

answer = generator.generate(
    query="企业内部知识库问答如何提升准确率？",
    contexts=[r.text for r in results]
)

print(answer)
```

### 5. Start API service

```bash
python -m soloai_rag.api --host 0.0.0.0 --port 8000
```

示例请求：

```bash
curl -X POST http://localhost:8000/query \
  -H "Content-Type: application/json" \
  -d '{
    "query": "什么是检索增强生成？",
    "top_k": 5
  }'
```

示例响应：

```json
{
  "answer": "检索增强生成（RAG）是一种将信息检索与大语言模型生成结合的架构，用于提升答案的准确性与可追溯性。",
  "sources": [
    "docs/rag_basics.md",
    "docs/vector_search.md"
  ]
}
```
