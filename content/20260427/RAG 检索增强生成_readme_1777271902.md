# SoloAI RAG 检索增强生成
[![Build Status](https://img.shields.io/badge/build-passing-brightgreen)](#)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

**SoloAI RAG 检索增强生成** is an open-source framework for building production-ready **Retrieval-Augmented Generation (RAG)** systems. It helps developers combine document retrieval, vector search, knowledge grounding, and LLM generation to deliver more accurate, traceable, and domain-aware AI applications.

---

## Overview

RAG（检索增强生成）通过将大语言模型与外部知识库连接，显著提升回答的准确性、时效性与可解释性。 This project provides a modular and extensible foundation for building enterprise-grade RAG pipelines for knowledge assistants, document Q&A, internal search, and domain-specific AI agents.

## Features

- **Modular RAG pipeline**: ingestion, chunking, embedding, retrieval, reranking, and generation
- **Multi-source knowledge ingestion**: supports documents, markdown, PDFs, and structured text
- **Vector database integration**: designed for common vector stores and semantic search workflows
- **Retrieval quality optimization**: hybrid retrieval, top-k search, and reranking support
- **Traceable answers**: citation-ready responses with source grounding
- **Production-friendly architecture**: configurable components, API-first design, and extensibility

## Quick Start

Below is a minimal example to ingest documents and run a RAG query.

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline(
    embedding_model="text-embedding-3-large",
    llm_model="gpt-4o-mini",
    vector_store="qdrant"
)

pipeline.ingest("./data")

response = pipeline.query(
    "什么是 RAG 检索增强生成？请结合知识库内容回答。",
    top_k=5,
    return_sources=True
)

print(response.answer)
print(response.sources)
```

### Run from CLI

```bash
python -m soloai_rag ingest --input ./data
python -m soloai_rag query --question "RAG 的核心架构是什么？"
```

## Usage Examples

## 1. Document Ingestion

Load and index a local knowledge base.

```python
from soloai_rag import DocumentIndexer

indexer = DocumentIndexer(
    embedding_model="text-embedding-3-large",
    vector_store="qdrant"
)

indexer.add_documents("./knowledge_base")
indexer.build_index()
```

## 2. Retrieval and Generation

Query indexed content and generate grounded answers.

```python
from soloai_rag import Retriever, Generator

retriever = Retriever(vector_store="qdrant")
generator = Generator(llm_model="gpt-4o-mini")

docs = retriever.search("如何优化 RAG 的召回率？", top_k=3)
answer = generator.generate(
    question="如何优化 RAG 的召回率？",
    context=docs
)

print(answer)
```

## 3. Hybrid Search

Combine semantic retrieval with keyword filtering.

```python
results = pipeline.query(
    "企业知识库问答系统的最佳实践",
    top_k=5,
    search_type="hybrid",
    filters={"department": "engineering"}
)
```

## 4. API Service Example

```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

Example request:

```bash
curl -X POST "http://localhost:8000/query" \
  -H "Content-Type: application/json" \
  -d '{
    "question": "请总结文档中的核心技术方案",
    "top_k": 5,
    "return_sources": true
  }'
```
