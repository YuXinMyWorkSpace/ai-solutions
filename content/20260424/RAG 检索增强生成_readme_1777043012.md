## 📖 Overview
SoloRAG is a production-ready, modular framework for **RAG 检索增强生成 (Retrieval-Augmented Generation)** that optimizes semantic search, context routing, and LLM orchestration. Engineered for low-latency inference and enterprise scalability, it enables developers to deploy accurate, knowledge-grounded AI applications with minimal boilerplate.

## ✨ Key Features
- **Hybrid Retrieval Engine**: Combines dense vector search, BM25 sparse retrieval, and metadata filtering for high-precision context matching
- **Adaptive Chunking & Reranking**: Semantic-aware document splitting with cross-encoder reranking to minimize hallucination and maximize context relevance
- **Multi-Format Ingestion Pipeline**: Native support for PDFs, Markdown, JSON, CSV, and structured databases with automatic schema extraction
- **Built-in Evaluation Suite**: Integrated metrics for faithfulness, context recall, answer relevance, and latency benchmarking
- **Async Streaming Architecture**: Non-blocking LLM generation with token-level streaming, retry logic, and circuit-breaker patterns
- **Observability & Caching**: OpenTelemetry tracing, Redis-backed semantic cache, and prompt/version tracking for production deployments

## 🚀 Quick Start
Initialize a minimal RAG pipeline in under 10 lines of code:
```python
from solorag import RAGPipeline, VectorStore, LLMClient

# 1. Load & index documents
store = VectorStore.from_directory("docs/knowledge_base/")

# 2. Configure generator
llm = LLMClient(model="openai/gpt-4o-mini", temperature=0.2)

# 3. Build & query pipeline
pipeline = RAGPipeline(retriever=store, generator=llm)
response = pipeline.query("What are the key compliance requirements for GDPR?")

print("✅ Answer:", response.text)
print("📚 Sources:", [doc.metadata["source"] for doc in response.sources])
```

## 🛠 Usage Examples

### 🔍 Advanced Hybrid Retrieval & Custom Chunking
```python
from solorag.retrievers import HybridRetriever
from solorag.processors import SemanticChunker
from solorag.config import RetrievalConfig

chunker = SemanticChunker(
    strategy="recursive", 
    max_tokens=512, 
    overlap=64
)

config = RetrievalConfig(
    dense_top_k=5,
    sparse_top_k=3,
    reranker_model="cross-encoder/ms-marco-MiniLM-L-6-v2",
    enable_metadata_filter=True
)

retriever = HybridRetriever(chunker=chunker, config=config)
retriever.add_documents("data/technical_manuals/")
```

### 📊 Evaluation & Benchmarking
```python
from solorag.eval import RAGEvaluator, FaithfulnessMetric, ContextRecallMetric

evaluator = RAGEvaluator(
    dataset="test_qa_pairs.jsonl",
    metrics=[FaithfulnessMetric(), ContextRecallMetric()]
)

results = evaluator.run(pipeline)
results.export_report("eval_report.md")
```
