## 项目简介

**SoloAI RAG 检索增强生成** 是一个用于构建 AI 检索问答系统的开源框架，支持从文档解析、向量化索引、混合检索到大模型生成的完整工作流。  
它能够将企业私有数据与大语言模型结合，显著提升回答的准确性、可解释性与可控性。

### 适用场景

- 企业知识库问答
- PDF / Word / Markdown 文档智能检索
- 客服机器人与内部助手
- 搜索增强生成（Search-Augmented Generation）
- 行业垂直 AI Copilot
- 多数据源知识整合与问答

---

## 核心特性

- **多源文档接入**：支持 PDF、Markdown、TXT、HTML、JSON 等常见数据格式解析与入库。
- **向量检索与混合检索**：支持 Embedding 向量搜索、关键词检索以及 Hybrid Search，提升召回效果。
- **上下文增强生成**：基于检索结果动态拼接上下文，降低大模型幻觉，提升回答准确率。
- **可扩展模型与存储层**：可对接主流 LLM、Embedding 模型、向量数据库与对象存储。
- **模块化架构**：文档切分、索引构建、检索、重排、提示词编排等能力解耦，便于二次开发。
- **工程化部署友好**：支持本地开发、容器化部署、API 服务化以及企业知识库集成。

## 安装说明

### 环境要求

- Python 3.10+
- pip 或 poetry
- 可选：Docker / Docker Compose
- 可选：向量数据库（如 FAISS、Milvus、Qdrant、Chroma）

### 使用 pip 安装

```bash
git clone https://github.com/SoloAI/rag-retrieval-augmented-generation.git
cd rag-retrieval-augmented-generation
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -U pip
pip install -r requirements.txt
```

### 配置环境变量

创建 `.env` 文件：

```bash
cp .env.example .env
```

示例配置：

```env
OPENAI_API_KEY=your_api_key
EMBEDDING_MODEL=text-embedding-3-large
LLM_MODEL=gpt-4o-mini
VECTOR_STORE=faiss
DATA_DIR=./data
```

### Docker 安装

```bash
docker build -t soloai-rag .
docker run --env-file .env -p 8000:8000 soloai-rag
```

## 快速开始

以下示例展示如何导入文档、构建索引，并执行一次基础的 RAG 问答。

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline(
    llm_model="gpt-4o-mini",
    embedding_model="text-embedding-3-large",
    vector_store="faiss"
)

# 1. 加载并索引文档
pipeline.index_documents("./data")

# 2. 执行检索增强生成
response = pipeline.query("请总结该知识库中关于 RAG 检索增强生成 的核心优势。")

print(response.answer)
print(response.sources)
```

启动本地 API 服务：

```bash
python -m app.server
```

默认服务地址：

```text
http://localhost:8000
```

## 使用示例

### 1. 构建本地知识库

```python
from soloai_rag import DocumentIndexer

indexer = DocumentIndexer(
    chunk_size=800,
    chunk_overlap=120,
    embedding_model="text-embedding-3-large",
    vector_store="faiss"
)

indexer.add_directory("./docs")
indexer.build()
```

### 2. 执行检索问答

```python
from soloai_rag import Retriever, Generator

retriever = Retriever(top_k=5, search_type="hybrid")
generator = Generator(model="gpt-4o-mini")

query = "RAG 系统如何降低大模型幻觉？"
docs = retriever.search(query)
answer = generator.generate(query=query, context_docs=docs)

print(answer)
```

### 3. API 调用示例

```bash
curl -X POST http://localhost:8000/api/v1/query \
  -H "Content-Type: application/json" \
  -d '{
    "query": "请解释检索增强生成的工作原理",
    "top_k": 5,
    "return_sources": true
  }'
```

示例响应：

```json
{
  "answer": "RAG（检索增强生成）通过先检索外部知识，再将相关上下文提供给大语言模型生成回答，从而提高准确性与可追溯性。",
  "sources": [
    {
      "title": "rag-overview.md",
      "score": 0.93
    }
  ]
}
```

### 4. 自定义检索参数

```python
response = pipeline.query(
    "企业为什么需要私有化 RAG 知识库？",
    top_k=8,
    rerank=True,
    temperature=0.2,
    return_sources=True
)

print(response)
```

### 5. 命令行方式导入与查询

```bash
python cli.py ingest --input ./docs --index faiss
python cli.py query --question "什么是 RAG 检索增强生成？"
```
