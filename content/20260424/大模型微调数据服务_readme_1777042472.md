A high-performance, open-source data pipeline engineered for **大模型微调数据服务** (LLM Fine-Tuning Data Service). SoloAI automates dataset curation, instruction-response generation, quality filtering, and format conversion to accelerate model alignment, domain adaptation, and preference optimization.

---

## ✨ Key Features

- **Automated SFT & Preference Pair Generation**: Convert raw corpora into instruction-tuning and DPO/RLHF-ready datasets using configurable prompt templates and LLM-assisted synthesis.
- **Multi-Stage Quality Filtering**: Built-in deduplication, toxicity detection, semantic consistency scoring, and length normalization to ensure high signal-to-noise ratios.
- **Format-Agnostic Export**: Seamless conversion to `JSONL`, `Parquet`, `Arrow`, and Hugging Face `Dataset` objects with strict schema validation.
- **Domain-Specific Template Engine**: Jinja2-based templating system supporting few-shot prompting, chain-of-thought structuring, and role-based system prompts.
- **Scalable & GPU-Accelerated**: Async batch processing, distributed chunking, and vLLM/TGI backend integration for high-throughput data generation.
- **Transparent Analytics**: CLI-driven dataset profiling (length distribution, token counts, duplication rates, reward margins) for reproducible data audits.

## 🚀 Quick Start

Generate a clean SFT dataset from raw text in under 10 lines of code:

```python
from soloai import DataPipeline, QualityFilter, FormatConverter

# 1. Initialize pipeline with a reference model for generation
pipeline = DataPipeline(model="meta-llama/Llama-3-8B-Instruct")

# 2. Load raw corpus & generate instruction-response pairs
raw_data = pipeline.load_corpus("raw_domain_corpus.txt")
sft_pairs = pipeline.generate_pairs(raw_data, template="qa_v2.json")

# 3. Filter low-quality samples & export
clean_data = QualityFilter(sft_pairs).run(threshold=0.85)
FormatConverter(clean_data).to_jsonl("output/sft_dataset.jsonl")
```

## 🛠️ Usage Examples

### 📊 Domain-Specific SFT Dataset Generation
```python
from soloai import DomainAdapter

adapter = DomainAdapter(
    domain="finance",
    prompt_template="templates/financial_sft.jinja",
    max_tokens=2048,
    batch_size=64
)
dataset = adapter.process("raw_financial_reports.csv")
dataset.push_to_hub("SoloAI/finance-sft-v1", private=True)
```

### ⚖️ DPO Preference Data Construction
```python
from soloai import DPOBuilder

dpo = DPOBuilder(
    chosen_threshold=0.9,
    rejected_threshold=0.6,
    strategy="reward_diff",
    model="Qwen/Qwen2.5-7B-Instruct"
)
preference_pairs = dpo.build(dataset)
preference_pairs.save("output/dpo_preference.jsonl")
```

### 🔍 Data Quality & Distribution Analysis
```bash
soloai analyze --input output/sft_dataset.jsonl \
               --metrics length,duplication,toxicity,semantic_diversity \
               --report output/quality_report.html
```
