## Overview

**SoloAI RAG 检索增强生成** is an open-source project that combines document indexing, semantic retrieval, and large language model generation to deliver accurate, context-aware responses. It is suitable for enterprise knowledge bases, customer support copilots, internal search systems, and domain-specific AI assistants.

## Features

- **Hybrid Retrieval Pipeline**: Support for dense vector retrieval, keyword search, and reranking.
- **Document Ingestion & Chunking**: Process PDFs, Markdown, TXT, HTML, and structured data sources.
- **LLM Grounding**: Improve response accuracy by injecting relevant context into prompts.
- **Pluggable Vector Store Support**: Easily integrate with FAISS, Milvus, Weaviate, Pinecone, or Chroma.
- **Production-Friendly API**: Simple CLI and Python API for indexing, querying, and evaluation.
- **Scalable & Extensible Architecture**: Modular components for embeddings, retrievers, rerankers, and generators.

## Quick Start

Index local documents and run a query in minutes.

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline(
    embedding_model="text-embedding-3-large",
    llm_model="gpt-4o-mini",
    vector_store="faiss"
)

pipeline.index_documents("./data")

response = pipeline.query("什么是 RAG 检索增强生成？它的典型应用场景有哪些？")
print(response.answer)
print(response.sources)
```

## Usage

### 1. Index Documents

```python
from soloai_rag import DocumentIndexer

indexer = DocumentIndexer(
    embedding_model="text-embedding-3-large",
    vector_store="faiss"
)

indexer.add_path("./docs")
indexer.build()
indexer.save("./storage/index")
```

### 2. Load an Existing Index

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline.load(
    index_path="./storage/index",
    llm_model="gpt-4o-mini"
)

result = pipeline.query("请总结系统架构设计原则")
print(result.answer)
```

### 3. CLI Example

```bash
python -m soloai_rag index --input ./docs --output ./storage/index
python -m soloai_rag query --index ./storage/index --question "如何部署该系统？"
```

### 4. Advanced Retrieval Configuration

```python
from soloai_rag import RAGPipeline, RetrievalConfig

config = RetrievalConfig(
    top_k=5,
    rerank=True,
    hybrid_search=True,
    score_threshold=0.25
)

pipeline = RAGPipeline(
    embedding_model="text-embedding-3-large",
    llm_model="gpt-4o-mini",
    vector_store="faiss",
    retrieval_config=config
)

result = pipeline.query("请基于知识库解释该产品的核心功能")
print(result.answer)
for source in result.sources:
    print(source)
```
