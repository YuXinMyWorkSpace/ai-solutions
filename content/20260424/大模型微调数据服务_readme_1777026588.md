在大模型落地过程中，**高质量数据** 往往比模型本身更决定最终效果。

## 快速开始

以下示例展示如何将原始问答数据转换为适合大模型监督微调（SFT）的 JSONL 格式。

### 输入示例

```json
[
  {
    "instruction": "解释什么是向量数据库",
    "input": "",
    "output": "向量数据库是一种专门用于存储、索引和检索向量嵌入的数据系统。"
  }
]
```

### Python 处理示例

```python
import json

raw_data = [
    {
        "instruction": "解释什么是向量数据库",
        "input": "",
        "output": "向量数据库是一种专门用于存储、索引和检索向量嵌入的数据系统。"
    }
]

with open("train.jsonl", "w", encoding="utf-8") as f:
    for item in raw_data:
        record = {
            "messages": [
                {"role": "user", "content": item["instruction"]},
                {"role": "assistant", "content": item["output"]}
            ]
        }
        f.write(json.dumps(record, ensure_ascii=False) + "\n")

print("SFT 数据已生成: train.jsonl")
```

### 输出示例

```json
{"messages": [{"role": "user", "content": "解释什么是向量数据库"}, {"role": "assistant", "content": "向量数据库是一种专门用于存储、索引和检索向量嵌入的数据系统。"}]}
```

## 使用示例

### 1. 数据清洗

适合对抓取数据、业务对话日志或历史知识库进行标准化处理。

```python
def clean_record(text: str) -> str:
    text = text.strip()
    text = text.replace("\u3000", " ")
    text = " ".join(text.split())
    return text

sample = "   大模型   微调 数据服务    "
print(clean_record(sample))
```

输出：

```text
大模型 微调 数据服务
```

---

### 2. 指令微调数据格式转换

将业务文本转为常见的 instruction-following 数据格式。

```python
dataset = [
    {
        "question": "什么是 SFT？",
        "answer": "SFT 是监督微调，用于让模型学习目标任务的输入输出映射。"
    }
]

converted = []
for item in dataset:
    converted.append({
        "instruction": item["question"],
        "input": "",
        "output": item["answer"]
    })

print(converted)
```
