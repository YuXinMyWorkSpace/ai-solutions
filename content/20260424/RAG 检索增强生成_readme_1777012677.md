## 项目简介

**SoloAI RAG 检索增强生成** 是一个专注于构建高质量知识增强问答系统的开源框架，支持文档切分、向量化索引、语义检索、上下文增强生成以及可扩展的模型与向量数据库接入。  
该项目旨在帮助团队快速搭建 AI 知识库、企业内部问答系统、智能客服和多源文档理解平台。

---

## 核心特性

- **多源文档接入**：支持 Markdown、TXT、PDF、HTML 等常见文档格式的导入与解析
- **语义检索增强**：基于向量检索与相似度召回，提升大模型回答的准确性与上下文相关性
- **模块化架构**：支持替换 Embedding 模型、向量数据库、重排序器与大语言模型
- **高质量生成链路**：支持检索、重排、上下文拼接与生成的一体化 RAG Pipeline
- **面向生产部署**：便于接入 API 服务、私有知识库、企业内部系统与自动化流程
- **易扩展与可集成**：适合集成到 AI Agent、FAQ Bot、企业 Copilot、搜索增强生成应用中

## 快速开始

下面示例展示如何加载本地文档、建立向量索引，并进行一次 RAG 问答。

```python
from soloai_rag import RAGPipeline, DocumentLoader, VectorStore

# 1. 加载文档
loader = DocumentLoader("./data")
documents = loader.load()

# 2. 构建向量索引
vector_store = VectorStore.from_documents(
    documents=documents,
    embedding_model="text-embedding-3-large"
)

# 3. 初始化 RAG Pipeline
rag = RAGPipeline(
    vector_store=vector_store,
    llm_model="gpt-4o-mini"
)

# 4. 发起检索增强问答
response = rag.query("请总结知识库中关于 RAG 检索增强生成的核心价值。")

print(response.answer)
print(response.sources)
```

## 使用示例

### 1. 构建本地知识库

```python
from soloai_rag import KnowledgeBase

kb = KnowledgeBase(
    data_path="./docs",
    embedding_model="text-embedding-3-large",
    vector_db="faiss"
)

kb.build_index()
print("知识库索引构建完成")
```

### 2. 基于检索结果生成答案

```python
query = "企业为什么需要 RAG 系统？"
result = kb.ask(query)

print("Answer:")
print(result.answer)

print("\nRetrieved Chunks:")
for chunk in result.chunks:
    print("-", chunk.text[:120])
```

### 3. 自定义检索参数

```python
result = kb.ask(
    "如何优化检索召回质量？",
    top_k=5,
    rerank=True,
    temperature=0.2
)

print(result.answer)
```

### 4. 启动 API 服务

```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

示例接口调用：

```bash
curl -X POST "http://127.0.0.1:8000/query" \
  -H "Content-Type: application/json" \
  -d '{
    "question": "什么是检索增强生成？",
    "top_k": 3
  }'
```

示例响应：

```json
{
  "answer": "检索增强生成（RAG）是一种将外部知识检索与大语言模型生成相结合的方法，用于提升回答的准确性、时效性与可解释性。",
  "sources": [
    {
      "file": "intro.md",
      "score": 0.93
    }
  ]
}
```
