[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/llm-finetune-data-service/ci.yml?branch=main)](https://github.com/SoloAI/llm-finetune-data-service/actions)

帮助团队更高效地为 LLM / SFT / 指令微调 / 偏好优化等场景准备可用、可控、可扩展的数据资产。

---

## 简介

**SoloAI 大模型微调数据服务**旨在为企业级 AI 项目提供端到端的数据处理能力，包括原始语料治理、指令数据构造、数据去重清洗、质量评估、格式转换与交付。项目适用于通用大模型、垂直行业模型、智能客服、知识助手、内容生成、代码助手等场景。

---

## 核心特性

- **多源数据接入**：支持文本、表格、FAQ、知识库、客服记录、工单、网页内容等多种数据来源
- **微调数据清洗与去重**：支持敏感信息过滤、重复样本识别、低质量样本剔除与规范化处理
- **指令数据构建**：支持 SFT、QA、对话、多轮指令、分类、摘要、抽取等训练格式生成
- **质量控制与验收**：支持抽检、规则校验、标签一致性检查、字段完整性验证
- **多框架格式兼容**：可导出为 JSON、JSONL、CSV 等格式，适配 Hugging Face、OpenAI、LLaMA Factory 等训练流程
- **面向行业场景交付**：适配金融、教育、电商、政企、医疗、制造、客服等行业数据服务需求

## 安装

### 环境要求

- Python 3.9+
- pip 22+
- 推荐使用虚拟环境

### 1. 克隆项目

```bash
git clone https://github.com/SoloAI/llm-finetune-data-service.git
cd llm-finetune-data-service
```

### 2. 创建虚拟环境

```bash
python -m venv .venv
source .venv/bin/activate
```

Windows:

```bash
.venv\Scripts\activate
```

### 3. 安装依赖

```bash
pip install -r requirements.txt
```

### 4. 可选：开发模式安装

```bash
pip install -e .
```

## 快速开始

下面示例演示如何将原始问答数据转换为适用于大模型监督微调（SFT）的 JSONL 格式。

### 准备输入数据

`data/raw/sample.json`:

```json
[
  {
    "question": "什么是大模型微调？",
    "answer": "大模型微调是指在预训练模型基础上，使用特定任务数据继续训练，以提升模型在目标场景中的表现。"
  },
  {
    "question": "如何提升训练数据质量？",
    "answer": "可以通过去重、清洗、纠错、标注规范化和人工质检等方式提升训练数据质量。"
  }
]
```

### 执行转换

```bash
python scripts/prepare_sft.py \
  --input data/raw/sample.json \
  --output data/processed/sft.jsonl \
  --format alpaca
```

### 输出示例

`data/processed/sft.jsonl`:

```json
{"instruction":"什么是大模型微调？","input":"","output":"大模型微调是指在预训练模型基础上，使用特定任务数据继续训练，以提升模型在目标场景中的表现。"}
{"instruction":"如何提升训练数据质量？","input":"","output":"可以通过去重、清洗、纠错、标注规范化和人工质检等方式提升训练数据质量。"}
```

## 使用示例

### 1. 数据清洗

```bash
python scripts/clean_data.py \
  --input data/raw/source.jsonl \
  --output data/clean/cleaned.jsonl \
  --remove-duplicates \
  --filter-sensitive \
  --min-length 10
```

功能说明：

- 删除重复样本
- 过滤过短文本
- 脱敏与敏感信息过滤
- 标准化字段结构

---

### 2. 构建多轮对话训练数据

```bash
python scripts/build_conversation.py \
  --input data/raw/chat_logs.jsonl \
  --output data/processed/conversations.jsonl \
  --role-map user:human assistant:gpt
```

输出格式示例：

```json
{
  "messages": [
    {"role": "system", "content": "你是一个专业的行业助手。"},
    {"role": "user", "content": "请介绍一下模型微调数据清洗流程。"},
    {"role": "assistant", "content": "模型微调数据清洗通常包括格式统一、去重、敏感信息过滤、质量抽检和样本重写等步骤。"}
  ]
}
```

---

### 3. 数据质量校验

```bash
python scripts/validate_dataset.py \
  --input data/processed/sft.jsonl \
  --schema configs/sft.schema.yaml
```

示例输出：

```bash
Validation Summary
------------------
Total Samples: 12000
Valid Samples: 11876
Invalid Samples: 124
Missing Fields: 58
Encoding Errors: 12
Duplicate Records: 54
```

---

### 4. Python API 调用示例

```python
from soloai_data.pipeline import DataPipeline

pipeline = DataPipeline(
    input_path="data/raw/domain_qa.jsonl",
    output_path="data/output/train.jsonl"
)

pipeline.clean(
    remove_duplicates=True,
    filter_sensitive=True,
    normalize_text=True
)

pipeline.transform(
    task="sft",
    template="alpaca"
)

pipeline.validate(
    schema="configs/sft.schema.yaml"
)

pipeline.export(format="jsonl")
```
