## Installation

### Requirements

- Python 3.9+
- pip or poetry
- Docker (recommended for containerized deployment)
- Optional: CUDA-compatible GPU for accelerated inference

### Install from source

```bash
git clone https://github.com/SoloAI/llm-deployment-optimization.git
cd llm-deployment-optimization
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -U pip
pip install -r requirements.txt
```

### Optional: install in editable mode

```bash
pip install -e .
```

### Docker setup

```bash
docker build -t soloai/llm-deployment-optimization:latest .
docker run --rm -p 8080:8080 soloai/llm-deployment-optimization:latest
```

## Quick Start

Start a local optimized inference server:

```bash
python -m soloai_llm_optimization.serve \
  --model ./models/your-llm \
  --host 0.0.0.0 \
  --port 8080 \
  --batch-size 8 \
  --dtype float16
```

Send a test request:

```bash
curl -X POST "http://localhost:8080/v1/chat/completions" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "your-llm",
    "messages": [
      {"role": "system", "content": "You are a precise assistant."},
      {"role": "user", "content": "Explain how inference batching improves throughput."}
    ],
    "temperature": 0.2,
    "max_tokens": 256
  }'
```

## Usage

### 1. Start an optimized API service

```bash
python -m soloai_llm_optimization.serve \
  --model meta-llama/Llama-3-8B-Instruct \
  --port 8080 \
  --tensor-parallel 2 \
  --max-concurrent-requests 128 \
  --enable-metrics
```

### 2. Enable quantized inference

```bash
python -m soloai_llm_optimization.serve \
  --model ./models/llm \
  --quantization int8 \
  --gpu-memory-utilization 0.9
```

### 3. Deploy with Docker Compose

```yaml
version: "3.9"

services:
  llm-server:
    image: soloai/llm-deployment-optimization:latest
    ports:
      - "8080:8080"
    environment:
      MODEL_PATH: /models/your-llm
      QUANTIZATION: int8
      ENABLE_METRICS: "true"
    volumes:
      - ./models:/models
```

Run:

```bash
docker compose up -d
```

### 4. Example Python client

```python
import requests

url = "http://localhost:8080/v1/chat/completions"
payload = {
    "model": "your-llm",
    "messages": [
        {"role": "user", "content": "Give me three strategies for reducing LLM inference cost."}
    ],
    "temperature": 0.3,
    "max_tokens": 180
}

response = requests.post(url, json=payload, timeout=60)
response.raise_for_status()
print(response.json())
```

### 5. Example optimization workflow

```bash
# Benchmark baseline
python -m soloai_llm_optimization.benchmark \
  --endpoint http://localhost:8080 \
  --requests 500 \
  --concurrency 32

# Re-run with quantization and batching enabled
python -m soloai_llm_optimization.benchmark \
  --endpoint http://localhost:8080 \
  --requests 500 \
  --concurrency 32 \
  --compare baseline.json
```
