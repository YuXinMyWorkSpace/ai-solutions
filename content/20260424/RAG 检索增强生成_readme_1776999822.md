# SoloAI RAG 检索增强生成

[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/rag-retrieval-augmented-generation/ci.yml?branch=main)](https://github.com/SoloAI/rag-retrieval-augmented-generation/actions)
[![License](https://img.shields.io/github/license/SoloAI/rag-retrieval-augmented-generation)](./LICENSE)

面向生产环境的 **RAG（Retrieval-Augmented Generation，检索增强生成）** 开源项目，帮助开发者快速构建基于私有知识库的问答、搜索、Copilot 与企业 AI 应用。该项目聚焦高质量检索、可扩展索引与大模型生成链路，适用于文档问答、知识助手和智能客服等场景。

---

## Overview

**SoloAI RAG 检索增强生成** 提供从文档解析、向量化、索引构建、语义检索到大模型回答生成的完整技术栈。项目设计强调模块化、可观测性和易集成性，适合用于企业知识库、SaaS 平台、内部助手以及垂直行业 AI 解决方案。

**Keywords:** RAG, 检索增强生成, Retrieval-Augmented Generation, Vector Search, Embedding, LLM, Knowledge Base, Semantic Search, AI Assistant, 企业知识库

---

## Features

- **多数据源接入**：支持 Markdown、PDF、TXT、HTML 及自定义数据管道
- **高质量向量检索**：支持 Embedding、Top-K 检索、相似度召回与上下文拼接
- **可插拔 LLM/Embedding 适配层**：便于接入 OpenAI、Azure OpenAI、本地模型或其他兼容接口
- **模块化 RAG Pipeline**：覆盖文档切分、索引构建、召回、重排、生成等核心链路
- **面向生产环境**：支持配置化部署、日志监控、缓存策略与 API 集成
- **开发者友好**：清晰的 Python API、CLI 示例与快速启动体验

---

## Architecture

```text
Documents -> Chunking -> Embeddings -> Vector Store -> Retrieval -> Reranking -> LLM Generation -> Final Answer
```

典型流程：

1. 导入业务文档或知识库数据
2. 执行分块与向量化
3. 写入向量数据库或本地索引
4. 用户发起问题
5. 检索相关上下文
6. 将上下文注入 Prompt
7. 调用大模型生成可信回答

## Quick Start

下面示例展示如何构建一个最小可用的 RAG 问答流程。

```python
from soloai_rag import RAGPipeline, DocumentLoader, VectorIndex

# 1. 加载文档
docs = DocumentLoader("./data").load()

# 2. 构建索引
index = VectorIndex.from_documents(
    docs,
    embedding_model="text-embedding-3-large"
)

# 3. 初始化 RAG Pipeline
rag = RAGPipeline(
    index=index,
    llm_model="gpt-4o-mini",
    top_k=5
)

# 4. 提问
response = rag.ask("请总结该知识库中关于 RAG 检索增强生成 的核心能力。")

print(response.answer)
print(response.sources)
```

运行：

```bash
python examples/quickstart.py
```

## Usage Examples

### 1. Index local documents

```python
from soloai_rag import DocumentLoader, VectorIndex

docs = DocumentLoader("./knowledge_base").load()
index = VectorIndex.from_documents(docs, embedding_model="text-embedding-3-large")
index.save("./storage/index")
```

### 2. Load an existing index

```python
from soloai_rag import VectorIndex

index = VectorIndex.load("./storage/index")
```

### 3. Ask with retrieved sources

```python
from soloai_rag import RAGPipeline, VectorIndex

index = VectorIndex.load("./storage/index")

rag = RAGPipeline(
    index=index,
    llm_model="gpt-4o-mini",
    top_k=3,
    return_sources=True
)

result = rag.ask("如何提升企业知识库问答的准确率？")

print("Answer:")
print(result.answer)

print("\nSources:")
for source in result.sources:
    print(f"- {source['title']} ({source['score']:.4f})")
```

### 4. CLI example

```bash
python -m soloai_rag.index --input ./knowledge_base --output ./storage/index
python -m soloai_rag.ask --index ./storage/index --query "RAG 检索增强生成 的典型企业应用有哪些？"
```

### 5. API service example

```python
from fastapi import FastAPI
from soloai_rag import RAGPipeline, VectorIndex

app = FastAPI()
index = VectorIndex.load("./storage/index")
rag = RAGPipeline(index=index, llm_model="gpt-4o-mini", top_k=5)

@app.get("/query")
def query(q: str):
    result = rag.ask(q)
    return {
        "query": q,
        "answer": result.answer,
        "sources": result.sources,
    }
```

启动服务：

```bash
uvicorn app:app --host 0.0.0.0 --port 8000
```
