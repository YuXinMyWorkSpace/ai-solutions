## 项目简介

**SoloAI 大模型微调数据服务** 是一个用于支持大模型训练与微调数据生产流程的开源项目，覆盖数据采集、数据治理、样本构造、质量评估与格式导出等核心环节。  
适用于通用大模型、行业垂类模型、RAG增强训练、SFT/Instruction Tuning、对话数据构建、偏好数据整理等应用场景。

---

## 核心特性

- **多源数据处理**：支持文本、问答、知识库、工单、文档、对话日志等多种数据源接入
- **数据清洗与标准化**：内置去重、脱敏、格式规范化、噪声过滤、分片与结构化处理流程
- **微调样本构造**：支持 SFT、Instruction、Chat、Prompt-Completion 等常见训练数据格式
- **质量评估机制**：支持规则校验、字段完整性检查、样本一致性评估与可追溯审计
- **灵活导出能力**：可导出 JSONL、CSV、Parquet 等训练友好格式，便于对接主流训练框架
- **企业级交付支持**：适用于行业数据服务、定制化数据集构建、模型训练前数据工程与验收交付

## 快速开始

以下示例展示如何将原始业务数据转换为适用于大模型监督微调（SFT）的训练样本。

### 1. 准备原始数据

`data/raw/customer_service.jsonl`

```json
{"question": "如何重置密码？", "answer": "进入账户设置，点击“重置密码”，根据提示完成验证。"}
{"question": "发票什么时候开具？", "answer": "订单完成后可在订单中心申请电子发票。"}
```

### 2. 执行数据清洗与样本构造

```bash
python scripts/build_dataset.py \
  --input data/raw/customer_service.jsonl \
  --output data/processed/sft_dataset.jsonl \
  --task sft \
  --language zh \
  --deduplicate \
  --anonymize
```

### 3. 输出示例

`data/processed/sft_dataset.jsonl`

```json
{"instruction": "请回答用户问题", "input": "如何重置密码？", "output": "进入账户设置，点击“重置密码”，根据提示完成验证。"}
{"instruction": "请回答用户问题", "input": "发票什么时候开具？", "output": "订单完成后可在订单中心申请电子发票。"}
```

## 使用示例

### 示例 1：构建对话式微调数据

```python
from soloai_data_service.pipeline import DatasetBuilder

builder = DatasetBuilder(
    task="chat",
    language="zh",
    deduplicate=True,
    anonymize=True
)

records = [
    {
        "messages": [
            {"role": "user", "content": "我想取消订单"},
            {"role": "assistant", "content": "请提供订单号，我来帮您查询取消条件。"}
        ]
    }
]

dataset = builder.build(records)
builder.export_jsonl(dataset, "data/output/chat_dataset.jsonl")
```

### 示例 2：导出 Prompt-Completion 格式

```python
from soloai_data_service.formatters import PromptCompletionFormatter

formatter = PromptCompletionFormatter()

sample = formatter.format({
    "prompt": "总结以下文本：人工智能正在改变企业服务模式。",
    "completion": "人工智能正在推动企业服务向自动化、智能化方向升级。"
})

print(sample)
```

输出：

```json
{
  "prompt": "总结以下文本：人工智能正在改变企业服务模式。",
  "completion": "人工智能正在推动企业服务向自动化、智能化方向升级。"
}
```

### 示例 3：数据质量校验

```bash
python scripts/validate_dataset.py \
  --input data/processed/sft_dataset.jsonl \
  --schema schemas/sft.schema.json \
  --report reports/validation_report.json
```

### 示例 4：导出用于训练框架的 JSONL 文件

```bash
python scripts/export_dataset.py \
  --input data/processed/intermediate.jsonl \
  --format jsonl \
  --output data/train/train.jsonl
```
