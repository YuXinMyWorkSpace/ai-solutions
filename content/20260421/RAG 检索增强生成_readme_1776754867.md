# SoloAI RAG 检索增强生成

[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/rag-retrieval-augmented-generation/ci.yml?branch=main)](https://github.com/SoloAI/rag-retrieval-augmented-generation/actions)
[![License](https://img.shields.io/github/license/SoloAI/rag-retrieval-augmented-generation)](LICENSE)

> 一个面向生产环境的 **RAG（Retrieval-Augmented Generation，检索增强生成）** 开源项目，帮助开发者快速构建基于私有知识库的智能问答、文档检索与生成式 AI 应用。  
> SoloAI RAG 提供从文档索引、向量检索、上下文增强到大模型生成的一体化工作流，适用于企业知识库、客服机器人和智能助手等场景。

## 功能特性

- **端到端 RAG 工作流**：支持文档解析、切分、Embedding、向量存储、检索与 LLM 生成。
- **多数据源接入**：可扩展支持 PDF、Markdown、TXT、HTML、数据库及企业内部知识文档。
- **高性能语义检索**：基于向量数据库与相似度搜索，实现高召回、低延迟的上下文检索。
- **可插拔大模型与 Embedding 模型**：兼容 OpenAI、本地模型、开源推理服务及自定义 API。
- **面向生产环境设计**：支持配置化部署、环境变量管理、日志监控与模块化扩展。
- **适合 AI 搜索与企业问答**：优化检索增强生成链路，提升答案准确性、可解释性与知识可控性。

## 安装

### 环境要求

- Python 3.10+
- pip 或 poetry
- 可选：Docker / Docker Compose
- 可选：向量数据库（如 FAISS、Chroma、Milvus、Weaviate）

### 方式一：使用 pip 安装

```bash
git clone https://github.com/SoloAI/rag-retrieval-augmented-generation.git
cd rag-retrieval-augmented-generation
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -U pip
pip install -r requirements.txt
```

### 方式二：使用 Poetry

```bash
git clone https://github.com/SoloAI/rag-retrieval-augmented-generation.git
cd rag-retrieval-augmented-generation
poetry install
```

### 配置环境变量

创建 `.env` 文件：

```env
OPENAI_API_KEY=your_api_key
EMBEDDING_MODEL=text-embedding-3-large
LLM_MODEL=gpt-4o-mini
VECTOR_STORE=chroma
DATA_DIR=./data
INDEX_DIR=./storage/index
```

## 快速开始

以下示例展示如何将本地文档建立索引，并通过 RAG 检索增强生成回答问题。

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline(
    embedding_model="text-embedding-3-large",
    llm_model="gpt-4o-mini",
    vector_store="chroma",
)

# 1. 构建索引
pipeline.index_documents("./data")

# 2. 提问
response = pipeline.query("RAG 检索增强生成的核心流程是什么？")

print(response.answer)
print(response.sources)
```

运行：

```bash
python examples/quickstart.py
```

## 使用示例

### 1. 索引本地知识库

```python
from soloai_rag import DocumentIndexer

indexer = DocumentIndexer(
    data_dir="./knowledge_base",
    chunk_size=512,
    chunk_overlap=64,
    embedding_model="text-embedding-3-large",
    vector_store="faiss",
)

indexer.build()
```

### 2. 执行语义检索

```python
from soloai_rag import Retriever

retriever = Retriever(index_path="./storage/index", top_k=5)

results = retriever.search("如何提升企业内部知识问答的准确率？")

for doc in results:
    print(doc.score, doc.metadata["source"])
```

### 3. 基于检索结果生成答案

```python
from soloai_rag import RAGPipeline

pipeline = RAGPipeline(
    vector_store="faiss",
    llm_model="gpt-4o-mini",
)

response = pipeline.query(
    question="请总结该系统的优势",
    top_k=4,
    return_sources=True,
)

print("Answer:", response.answer)
print("Sources:", response.sources)
```

### 4. 启动 API 服务

```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

示例请求：

```bash
curl -X POST "http://localhost:8000/query" \
  -H "Content-Type: application/json" \
  -d '{
    "question": "什么是 RAG 检索增强生成？",
    "top_k": 3
  }'
```

示例响应：

```json
{
  "answer": "RAG（Retrieval-Augmented Generation，检索增强生成）是一种将信息检索与大语言模型生成结合的架构，通过先检索相关知识，再将上下文注入模型生成更准确、可追溯的回答。",
  "sources": [
    {
      "source": "docs/intro.md",
      "score": 0.92
    }
  ]
}
```
