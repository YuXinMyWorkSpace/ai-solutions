## 项目简介

**SoloAI 大模型微调数据服务** 是一个为 **LLM SFT、Instruction Tuning、RLHF 数据准备** 而设计的开源数据服务系统。  
它支持结构化与非结构化数据处理、对话样本构建、数据质量校验、标准格式导出，以及面向训练平台的自动化数据工作流集成。

适用于：

- 通用对话模型微调
- 行业知识问答数据构建
- 指令跟随数据集生成
- 多轮对话数据整理
- 人工标注与数据审核流程管理
- OpenAI / ShareGPT / Alpaca / JSONL 等训练格式转换

---

## 核心特性

- **统一数据接入**：支持从 CSV、JSON、JSONL、数据库、对象存储等多种来源导入训练样本。
- **数据清洗与标准化**：内置去重、脱敏、字段映射、文本规范化和脏数据过滤能力。
- **多轮对话与指令数据构建**：支持 SFT、ChatML、ShareGPT、Alpaca 等主流微调数据格式。
- **数据质量校验**：提供 schema 校验、字段完整性检查、长度限制、标签约束和采样预览。
- **任务化数据导出**：支持按项目、版本、标签、审核状态导出训练集，便于持续迭代。
- **开放式 API 与自动化集成**：可无缝接入训练平台、标注平台与 MLOps/LLMOps 工作流。

## 安装

### 环境要求

- Python 3.10+
- pip 23+
- PostgreSQL 14+（可选）
- Redis 6+（可选）
- Docker / Docker Compose（推荐）

### 方式一：本地安装

```bash
git clone https://github.com/SoloAI/llm-finetune-data-service.git
cd llm-finetune-data-service

python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate

pip install -U pip
pip install -r requirements.txt
```

### 方式二：Docker 部署

```bash
git clone https://github.com/SoloAI/llm-finetune-data-service.git
cd llm-finetune-data-service

docker compose up -d --build
```

### 环境变量配置

复制并编辑环境变量文件：

```bash
cp .env.example .env
```

示例配置：

```env
APP_ENV=development
APP_HOST=0.0.0.0
APP_PORT=8080

DATABASE_URL=postgresql://postgres:postgres@localhost:5432/soloai_data
REDIS_URL=redis://localhost:6379/0

STORAGE_BACKEND=local
STORAGE_PATH=./data

API_KEY=replace-with-your-secret-key
```

## 快速开始

启动服务：

```bash
uvicorn app.main:app --host 0.0.0.0 --port 8080 --reload
```

服务启动后，可以通过以下方式验证：

```bash
curl http://localhost:8080/health
```

返回示例：

```json
{
  "status": "ok",
  "service": "llm-finetune-data-service"
}
```

### 创建一个微调数据集

```bash
curl -X POST http://localhost:8080/api/v1/datasets \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "name": "customer-support-sft",
    "description": "客服问答微调数据集",
    "task_type": "sft",
    "format": "sharegpt"
  }'
```

### 导入一条训练样本

```bash
curl -X POST http://localhost:8080/api/v1/records \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "dataset_id": "ds_123456",
    "messages": [
      {"role": "system", "content": "你是专业客服助手。"},
      {"role": "user", "content": "我的订单什么时候发货？"},
      {"role": "assistant", "content": "请提供订单号，我可以帮您查询物流状态。"}
    ],
    "metadata": {
      "source": "manual-labeling",
      "language": "zh-CN"
    }
  }'
```

## 使用示例

### 1. Python SDK / API 调用示例

```python
import requests

BASE_URL = "http://localhost:8080/api/v1"
API_KEY = "YOUR_API_KEY"

headers = {
    "Authorization": f"Bearer {API_KEY}",
    "Content-Type": "application/json",
}

payload = {
    "name": "medical-qa-sft",
    "description": "医疗问答微调数据",
    "task_type": "sft",
    "format": "jsonl"
}

resp = requests.post(f"{BASE_URL}/datasets", headers=headers, json=payload)
print(resp.status_code, resp.json())
```

### 2. 导出为 JSONL 训练数据

```bash
curl -X POST http://localhost:8080/api/v1/exports \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "dataset_id": "ds_123456",
    "target_format": "jsonl",
    "filters": {
      "review_status": "approved",
      "min_turns": 2,
      "language": "zh-CN"
    }
  }'
```

返回示例：

```json
{
  "export_id": "exp_7890",
  "status": "processing",
  "target_format": "jsonl"
}
```

### 3. 数据格式示例

#### ShareGPT 格式

```json
{
  "messages": [
    { "role": "system", "content": "你是一个有帮助的助手。" },
    { "role": "user", "content": "解释一下什么是微调？" },
    { "role": "assistant", "content": "微调是指在预训练模型基础上，使用特定任务数据继续训练，以提升模型在目标场景中的表现。" }
  ]
}
```

#### Alpaca 格式

```json
{
  "instruction": "解释什么是模型微调",
  "input": "",
  "output": "模型微调是指在预训练模型之上，结合任务相关数据进行进一步训练，以适配特定业务需求。"
}
```

### 4. 数据清洗任务示例

```bash
curl -X POST http://localhost:8080/api/v1/jobs/clean \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "dataset_id": "ds_123456",
    "rules": {
      "deduplicate": true,
      "mask_sensitive_info": true,
      "normalize_whitespace": true,
      "drop_empty_messages": true,
      "max_length": 4096
    }
  }'
```

### 5. 查询数据集统计信息

```bash
curl http://localhost:8080/api/v1/datasets/ds_123456/stats \
  -H "Authorization: Bearer YOUR_API_KEY"
```

示例返回：

```json
{
  "dataset_id": "ds_123456",
  "total_records": 120340,
  "approved_records": 118902,
  "rejected_records": 1438,
  "languages": ["zh-CN", "en-US"],
  "avg_turns": 3.7,
  "formats": ["sharegpt", "jsonl"]
}
```
