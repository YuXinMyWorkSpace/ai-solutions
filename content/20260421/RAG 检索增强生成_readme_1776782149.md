# SoloAI RAG 检索增强生成
[![Build Status](https://img.shields.io/github/actions/workflow/status/SoloAI/rag-retrieval-generation/ci.yml?branch=main)](https://github.com/SoloAI/rag-retrieval-generation/actions)

高性能、可扩展的 **RAG（Retrieval-Augmented Generation，检索增强生成）** 开源框架，专注于将向量检索、知识库构建与大语言模型生成能力进行高效集成。  
适用于企业知识问答、智能客服、文档检索、私有数据问答与 AI Agent 知识增强等场景。

---

## 概述

**SoloAI RAG 检索增强生成** 是一个面向生产环境设计的开源项目，帮助开发者快速构建基于私有知识库的 RAG 应用。项目支持文档解析、文本切分、向量化、索引构建、语义检索、上下文重排与大模型生成，提升回答准确率、可追溯性与上下文相关性。

---

## 核心特性

- **多源知识接入**：支持 Markdown、PDF、TXT、HTML 等多种文档格式的导入与解析
- **向量检索增强**：集成 Embedding 模型与向量数据库，实现高效语义检索
- **可扩展 RAG Pipeline**：支持文档切分、召回、重排、Prompt 拼接与生成链路自定义
- **生产级架构设计**：适合构建企业知识库、内部搜索、AI Copilot 与智能问答系统
- **兼容主流模型生态**：支持对接 OpenAI、开源 LLM、本地部署模型及多种 Embedding 服务
- **可观测与可追踪**：便于分析检索命中、上下文质量与生成效果，优化 RAG 性能

## 快速开始

以下示例展示如何基于本地知识库快速构建一个 RAG 问答流程。

```python
from soloai_rag import RAGPipeline, DocumentLoader, VectorIndexer

# 1. 加载文档
loader = DocumentLoader(data_dir="./data")
documents = loader.load()

# 2. 构建索引
indexer = VectorIndexer(
    embedding_model="text-embedding-3-large",
    vector_store="faiss",
    persist_dir="./storage/index"
)
indexer.build(documents)

# 3. 创建 RAG Pipeline
rag = RAGPipeline(
    llm_model="gpt-4o-mini",
    retriever=indexer.as_retriever(top_k=5)
)

# 4. 执行问答
response = rag.query("请总结该知识库中关于 RAG 架构设计的最佳实践。")
print(response.answer)
print(response.sources)
```

## 使用示例

### 1. 构建本地知识库索引

```python
from soloai_rag import DocumentLoader, TextSplitter, VectorIndexer

loader = DocumentLoader(data_dir="./docs")
docs = loader.load()

splitter = TextSplitter(chunk_size=500, chunk_overlap=100)
chunks = splitter.split_documents(docs)

indexer = VectorIndexer(
    embedding_model="bge-large-zh",
    vector_store="chroma",
    persist_dir="./storage/chroma"
)
indexer.build(chunks)
```

### 2. 执行语义检索

```python
retriever = indexer.as_retriever(top_k=3)
results = retriever.search("如何优化检索召回率与生成质量？")

for item in results:
    print(item.score, item.metadata["source"])
    print(item.page_content[:200])
```

### 3. 基于检索上下文生成答案

```python
from soloai_rag import RAGPipeline

rag = RAGPipeline(
    llm_model="gpt-4o",
    retriever=retriever,
    system_prompt=(
        "你是一个专业的 RAG 系统助手。"
        "请基于检索到的上下文回答问题，并尽量给出来源。"
    )
)

response = rag.query("RAG 中为什么需要重排（reranking）？")
print("Answer:", response.answer)
print("Citations:", response.sources)
```
